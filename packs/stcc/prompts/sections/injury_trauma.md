# injury_trauma 分诊知识片段

本片段整合了 STCC 指南中 **injury_trauma** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 热烧伤（STCC/Burns_Thermal.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因（对于化学烧伤 / 请参见化学品烧伤处理流程(112)） / 烧伤面积

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 大面积烧伤呈白色且无痛感
  - 剧烈疼痛且大面积烧伤区域红肿并有水泡
  - 呼吸困难
  - 意识状态改变
  - 胸痛或心跳快速不规则
  - 面部或鼻毛焦枯

**B级（立即急诊）**（立即前往急诊室）
  - 烧伤区域炭化
  - 水泡状或无痛白色烧伤面积大于手掌大小
  - 颈部或肢体环绕性烧伤
  - 吸入烟雾
  - 关节处的烧伤
  - >10%体表面积被烧伤

**C级（当天就医）**（当天内联系医生/诊所）
  - 近期有烧伤史，伴有红肿加剧、疼痛加重、肿胀、红线状红斑、浓稠分泌物、发热等症状
  - 糖尿病病史
  - 家庭治疗和非处方药后仍有中度至重度疼痛

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 未接种破伤风疫苗，疫苗状态未知或最后一次破伤风疫苗接种已超过5年
  - 疼痛持续>48小时
  - 多个开放性水泡

**D级居家护理**
  - 将冷却包敷在受伤区域，直到冷却包移除时疼痛缓解（可能需要几小时）。可以浸泡在凉水中。不要直接将冰块放在烧伤处。
  - 不要使用药膏、油脂、黄油或止痛软膏。
  - 可以涂抹氢氧化镁乳液、芦荟胶或酸奶以达到舒缓效果。向药师咨询其他非处方产品的建议。遵循标签上的指示。
  - 保持受伤区域清洁，并用干净、干燥的无粘连性敷料覆盖。

**回电/复诊标准**
  - 红肿加剧，疼痛加重，出现红线状红斑，脓液或浑浊分泌物，发热
  - 多个水泡破裂
  - 48小时后无改善
  - 最后一次破伤风疫苗接种已超过10年

## 化学烧伤（STCC/Burns_Chemical.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 化学物质名称 / 烧伤部位

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸困难
  - 意识状态改变
  - 胸痛或心跳加快不规则

**B级（立即急诊）**（立即前往急诊室）
  - 眼睛暴露于酸性物质如电池酸或腐蚀性物质（下水道清洁剂、苛性碱）
  - 剧烈疼痛且烧伤部位发红、起泡、变白或焦黑
  - 面部、耳朵、生殖器、颈部、手部、脚部或主要关节处的烧伤面积大于手掌大小
  - 烧伤围绕颈部或四肢
  - 暴露于制毒实验室化学品并出现症状

**C级（当天就医）**（当天内联系医生/诊所）
  - 遵循家庭护理措施20分钟后仍有持续疼痛
  - 眼睛持续红润、分泌物或泪水过多
  - 视力变化
  - 角膜颜色部分变白或浑浊
  - 感染迹象：发红加剧、疼痛、肿胀、分泌物或发热
  - 中等至重度疼痛在家中治疗后仍存在

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 破伤风疫苗接种时间超过5年
  - 多个开放性水泡

**D级居家护理**
  - 如果佩戴隐形眼镜，取下。在水中盆中或自来水下冲洗眼睛15到20分钟。将头部置于水龙头下方，使水流从内眼角流向外眼角，并用手指撑开眼睑。直到疼痛减轻为止。
  - 冲洗后，用敷料或干净布覆盖双眼并寻求医疗帮助。
  - 不要揉搓眼睛。
  - 对于辣椒素暴露（如辣椒、胡椒喷雾），使用全脂牛奶盒冲洗眼睛；其作用类似于清洁剂，可以去除刺激物。

**回电/复诊标准**
  - 感染迹象
  - 水泡破裂
  - 48小时内没有改善或病情加重
  - 最后一次破伤风疫苗接种时间超过10年

## 烧伤，电击（STCC/Burns_Electrical.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 伤口入口和出口位置 / 破伤风免疫状态

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 高电压电击
  - 意识丧失或无脉搏或呼吸
  - 心跳加速或不规则
  - 呼吸困难
  - 颈部以上烧伤
  - 明显的入口和出口伤口

**B级（立即急诊）**（立即前往急诊室）
  - 由电弧或闪击造成的烧伤
  - 头发被烧焦或缺失但皮肤没有烧伤
  - 口腔或唇部的烧伤，尤其是婴儿或幼儿
  - 环绕颈部或肢体的烧伤
  - 关节上的烧伤
  - 失忆或有任何意识丧失期

**C级（当天就医）**（当天内联系医生/诊所）
  - 低电压小范围烧伤
  - 怀孕超过20周

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 没有破伤风免疫接种，或免疫状态未知，或最后一次破伤风免疫接种超过5年前
  - 烧伤愈合不良
  - 感染迹象：红肿、条纹状红斑、肿胀或有分泌物
  - 没有其他症状，但个人或家长担忧

**D级居家护理**
  - 停止燃烧过程。脱下着火的衣物和首饰。
  - 用干净湿润的冷布冷却烧伤区域。
  - 不要涂抹油膏、油脂、黄油或止痛药膏。
  - 保持伤口清洁，用干净干燥的无粘性敷料覆盖。

## 创伤（STCC/Laceration.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 过敏史 / 疼痛评分

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 大面积撕裂伤
  - 部分或完全断肢
  - 头部、口腔后部或喉咙、胸部、颈部、生殖器或腹部的深伤口
  - 呼吸困难
  - 脉动性或喷射性出血
  - 创伤中可见骨头

**B级（立即急诊）**（立即前往急诊室）
  - 撕裂、分离、锯齿状或深伤口
  - 新近缝合、钉闭或粘合的伤口现在分裂开
  - 严重疼痛
  - 无法移动或受伤部位活动受限
  - 糖尿病史
  - 直接压迫10分钟后仍持续出血

**C级（当天就医）**（当天内联系医生/诊所）
  - 家庭护理措施无法缓解的持续性疼痛
  - 未接种破伤风疫苗，免疫状态不明，或最后一次破伤风疫苗接种超过5年
  - 关节上的伤口且难以保持伤口边缘闭合
  - 7至10天后伤口愈合不良

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 口腔内的小创伤
  - 伤口边缘可以自行闭合并控制住出血
  - 没有破伤风疫苗接种史
  - 头痛、肌肉疼痛、全身不适或发热
  - 父母或来电者对疤痕感到担忧

**D级居家护理**
  - 用干净的绷带或布直接压迫伤口以控制出血。
  - 用肥皂和水清洁伤口。
  - 将伤口边缘拉拢，并用蝶形或无菌条带固定。不要重叠皮肤边缘，让蝶形或无菌条带自然脱落前保持原位。
  - 可每天涂抹抗生素软膏2至3次。

**回电/复诊标准**
  - 疼痛、肿胀或出血加剧
  - 头痛、肌肉疼痛、全身不适或发热
  - 感染迹象，麻木感或刺痛感
  - 1周至10天后创伤愈合不良

## 踝部损伤（STCC/Ankle_Injury.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药情况 / 既往病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 骨头穿透皮肤
  - 明显畸形
  - 脚冷或发蓝
  - 难以控制出血

**B级（立即急诊）**（立即前往急诊室）
  - 剧烈疼痛
  - 无法承重
  - 受伤后立即无法行走

**C级（当天就医）**（当天内联系医生/诊所）
  - 24小时后，尽管使用冰敷、抬高、压迫和休息，肿胀、疼痛或瘀伤仍在加剧

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 受伤一段时间后发生的肿胀、不适、瘀伤或活动受限

**D级居家护理**
  - 在受伤后的前24到48小时内，每隔2小时将冰袋敷于受伤部位20至30分钟。不要直接将冰放在皮肤上。在冰袋和皮肤之间放置一层薄毛巾或袜子。未开封的冷冻蔬菜包或装在密封塑料袋中的碎冰也适用。
  - 尽可能经常地抬高踝部，持续前24到48小时。
  - 当人在活动时，应使用弹性绷带或气动夹板包裹踝部。如果脚趾开始肿胀、麻木或变得冷、痛，请解开绷带并重新松散地缠绕。
  - 在24小时后或根据耐受情况开始进行踝关节锻炼。

## 擦伤（STCC/Abrasions.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 其他伤害 / 药物

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 难以控制出血
  - 有血友病史
  - 大面积身体受影响

**B级（立即急诊）**（立即前往急诊室）
  - 无法清除伤口中的污垢或其他异物
  - 源头污染，且最后一次破伤风疫苗接种已超过5年

**C级（当天就医）**（当天内联系医生/诊所）
  - 有糖尿病史
  - 受影响部位活动困难
  - 伤口为24至48小时旧，并出现感染迹象：红肿、疼痛、触之发热、从创面延伸出的红线、渗液或脓液，或发热

**D级居家护理**
  - 用干净的绷带或布直接按压伤口以控制出血。
  - 每天用肥皂和水彻底清洗伤口。
  - 在接下来几天内，每日2至3次涂抹抗生素软膏。遵循标签上的指示。
  - 1至2天内使用干燥、清洁的敷料覆盖伤口。

**回电/复诊标准**
  - 感染迹象
  - 愈合延迟 >1周

## 背部/颈部损伤（STCC/BackNeck_Injury.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 受伤机制 / 用药情况 / 疼痛评分

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸困难
  - 严重的背部或颈部疼痛
  - 无法移动手指或脚趾
  - 手臂、腿部、手指或脚趾出现麻木、刺痛或无力
  - 大便或尿失禁
  - 难以止血

**B级（立即急诊）**（立即前往急诊室）
  - 过去一周内发生的创伤事件并伴随：
  - 疼痛加剧
  - 无力或失禁
  - 麻木或刺痛持续增加或存在
  - 尿潴留>4小时且膀胱感觉充盈
  - 肛门周围和腹股沟新出现的麻木感

**C级（当天就医）**（当天内联系医生/诊所）
  - 年龄>65岁
  - 有癌症或出血性疾病史
  - 疼痛干扰日常活动

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 持续的轻度到中度颈部或背部疼痛对休息、热敷、冷敷和止痛药无反应
  - 轻度到中度颈部或背部疼痛且未采取家庭护理措施

**D级居家护理**
  - 不要移动受伤者；保持患者温暖。
  - 如怀疑颈部受伤，应在颈部两侧放置沙袋、书籍、卷起的杂志、床单或毛巾。用胶带从一个支撑物穿过前额至对面的支撑物固定。
  - 如果受害者在水中，请将身体和颈部作为一个整体直到医疗帮助到达之前保持不动。救援者应位于受害者的头部位置，双臂伸展到受害者头、颈及肩膀下方，保持背部和颈部直线。不要让颈部前后或左右弯曲。
  - 不要移除穿透物体。留待医院评估。

## 严重出血（STCC/Bleeding_Severe.md）

**关键追问**：姓名 / 年龄 / 起病时间 / 原因 / 位置 / 药物

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 枪伤、刀伤或其他穿透性伤口且难以控制出血
  - 休克迹象包括：
  - 头晕，意识模糊
  - 无反应
  - 皮肤苍白、冰冷或潮湿
  - 脉搏快速

**B级（立即急诊）**（立即前往急诊室）
  - 直接压迫超过10分钟后仍持续出血
  - 裂开的流血伤口
  - 有出血障碍病史且难以控制出血
  - 正在服用抗凝药物
  - 无法移动受伤部位以上的肢体或手指

**D级居家护理**
  - 让患者躺下，并抬高受伤部分。
  - 直接对伤口施压至少10分钟。
  - 如果伤口喷血，用手掌和无菌或干净的绷带直接覆盖伤口施压。
  - 如果伤口深且大，用绷带或布料填充。避免使用纸巾或卫生纸。

**回电/复诊标准**
  - 出血或肿胀在出血得到控制后的24小时后发生
  - 感染迹象：疼痛加剧、分泌物、发热、肿胀、脓液、条纹或红斑

## Few-shot 示例

**用户问题**：我被开水烫到手背，出现了水泡，大约手掌大小。

**正确回答**：

【严重度判定】
**C级（当天内就医）**
手掌大小的热烧伤伴水泡，面积较大，需要当天就医处理避免感染。

【家庭护理 / 立刻行动】
- 立即用凉流水冲洗烫伤部位10-20分钟，降温
- 不要刺破水泡，不要涂抹牙膏、酱油等民间偏方
- 用干净的纱布或毛巾轻轻覆盖，前往医院

【回电 / 复诊标准】
就医后如出现伤口化脓、红肿扩大或发热，需尽快复诊。
