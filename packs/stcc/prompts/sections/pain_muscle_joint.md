# pain_muscle_joint 分诊知识片段

本片段整合了 STCC 指南中 **pain_muscle_joint** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 背痛（STCC/Back_Pain.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 疼痛部位 / 药物使用情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 双腿进行性无力
  - 新发突发的腿部或脚部麻木刺痛感，或膀胱/肠道控制丧失
  - 新发的会阴或肛门区域麻木
  - 60岁以上人群的新发迅速加重疼痛
  - 头晕、眩晕或腹部胀满感
  - 4小时以上无法排尿且感觉膀胱充盈

**B级（立即急诊）**（立即前往急诊室）
  - 伴有血尿的剧烈疼痛
  - 腿部、脚部或脚趾活动困难
  - 女性发热伴恶心呕吐
  - 长时间卧床或轮椅使用后突发疼痛，或年龄>60岁
  - <60岁人群的新发迅速加重疼痛
  - 疼痛放射至会阴或生殖器区域

**C级（当天就医）**（当天内联系医生/诊所）
  - 疼痛放射至臀部或四肢
  - 持续性剧烈疼痛
  - 排尿频繁或排尿时疼痛
  - 因不适行走困难
  - 非处方药无效
  - 男性发热伴恶心呕吐

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 慢性背痛、背部问题、背部手术、肾结石或肾脏疾病史
  - 无放射性或活动受限的轻微疼痛
  - 发热伴流感症状

**D级居家护理**
  - 限制轻度活动2至3天。
  - 使用硬床垫，或者在软床垫下放置木板。
  - 避免长时间坐着、举重或跳跃等活动直到疼痛缓解（不要卧床不起）。
  - 按照医嘱服用常规止痛药。儿童禁用阿司匹林。<20岁者避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病或胃部问题患者以及孕妇避免使用布洛芬。请遵循药物标签上的指示。

**回电/复诊标准**
  - 3天内无改善
  - 疼痛加剧
  - 疼痛放射至四肢、会阴或生殖器区域
  - 排尿困难、频繁排尿、发热或血尿

## 踝关节问题（STCC/Ankle_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 既往病史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 突然出现严重呼吸困难
  - 踝部或足部冷或发绀

**B级（立即急诊）**（立即前往急诊室）
  - 大腿/小腿肿胀和疼痛
  - 剧烈疼痛
  - 无法行走
  - 踝部、小腿或大腿的皮肤触摸时感觉热或发红
  - 单腿/踝部突然肿胀

**C级（当天就医）**（当天内联系医生/诊所）
  - 有心脏病、肝病、肾病、癌症、近期疾病、喉咙痛、皮肤感染或腿部手术的病史
  - 正在使用新的处方药
  - 双侧踝部肿胀
  - 关节或大脚趾基底部剧烈疼痛
  - 关节上方的皮肤发红且光亮
  - 怀孕期间体重突然增加

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 家庭护理后无改善
  - 近期体重增加超过10磅
  - 其他关节疼痛
  - 全身不适

**D级居家护理**
  - 将腿抬高，使脚高于心脏。不要在膝盖下垫任何东西。在小腿下方放置枕头。
  - 减少饮食中的盐分。
  - 休息。
  - 对患处加热。不要直接睡在电热毯上。使用电热毯时，务必在皮肤和电热毯之间放置布料隔层。

**回电/复诊标准**
  - 情况未改善或恶化
  - 疼痛加剧
  - 移动能力下降

## 胳膊或手部问题（STCC/Arm_or_Hand_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 位置 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 在或紧接用力后突然出现手臂疼痛
  - 胸闷、胸痛
  - 疼痛放射至颈部、下颌和肩部

**B级（立即急诊）**（立即前往急诊室）
  - 手臂、手或手指冰冷或发绀
  - 受影响的手臂无脉搏
  - 突然出现一侧手臂无力
  - 深度切割伤
  - 无法移动深度切割伤以上的部分手臂或手

**C级（当天就医）**（当天内联系医生/诊所）
  - 无法移动受伤区域上方或下方的关节
  - 剧烈疼痛
  - 无法使用受影响的身体部位
  - 可疑的原因，特别是在儿童中
  - 受影响区域红、热、痛或肿胀，但无已知损伤

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 关节疼痛或肿胀，且活动受限
  - 48小时后肿胀或瘀伤加剧
  - 受伤后30分钟内出现疼痛、肿胀或变色
  - 颈部僵硬伴手臂或手部疼痛、麻木或刺痛
  - 弯曲手臂或手时肘部、腕部或手指关节疼痛
  - 夜间尤其是手部的麻木或刺痛感

**D级居家护理**
  - 在受伤后的头24到48小时，每两小时将冰袋敷于受伤区域20至30分钟。不要直接将冰放在皮肤上，应在冰袋和皮肤之间放置一条薄毛巾或袜子。未开封的冷冻蔬菜包装或密封塑料袋中的碎冰也很合适。
  - 将受影响部位抬高至心脏水平以上以帮助减轻肿胀。
  - 与医生讨论截肢后的幻痛问题。有多种治疗选项，包括药物、针灸、脊髓刺激器、镜箱疗法、注射、植入装置、生物反馈和按摩疗法。
  - 让受影响的部位休息。

## 手指和脚趾问题（STCC/Finger_and_Toe_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 导致原因 / 过敏情况 / 用药历史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 截肢或接近截肢
  - 畸形伴有皮肤破裂或骨头穿透皮肤
  - 手指或脚趾比其他手指和脚趾冷或发蓝

**B级（立即急诊）**（立即前往急诊室）
  - 明显的畸形或脱位
  - 剧烈疼痛
  - 无法取下戒指，且手指开始变苍白、白色或蓝色
  - 发烧或寒战
  - 指甲松动或脱落
  - 高压射钉枪伤害

**C级（当天就医）**（当天内联系医生/诊所）
  - 肿胀且触痛的手指垫
  - 感染迹象：疼痛增加、肿胀、红斑、发热、红色条纹或分泌物
  - 受伤处最近的关节活动困难
  - 手指和脚趾比其他手指和脚趾麻木
  - 指甲下有血液，伴有疼痛加剧或压力感

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 无外伤的情况下出现疼痛或肿胀
  - 按照家庭护理指导后仍无法取下戒指
  - 中度肿胀
  - 糖尿病患者伤口愈合缓慢
  - 指甲变色和增厚

**D级居家护理**
  - 受伤后24至48小时内间歇性冰敷，并抬高该肢体。不要直接将冰块放在皮肤上，应在冰与皮肤之间放置一条湿布或其他织物屏障。
  - 立即在受伤后取下戒指，以防止肿胀发生。使用肥皂、润肤露、凡士林或其它润滑剂。
  - 通过将其绑在相邻的手指上来固定受伤的手指或脚趾。
  - 服用您平时的止痛药（阿司匹林、对乙酰氨基酚、布洛芬）。不要给儿童服用阿司匹林。20岁以下者避免使用类似阿司匹林的产品。肝病患者应避免使用对乙酰氨基酚。肾病、胃部问题或怀孕的情况下应避免使用布洛芬。遵循

**回电/复诊标准**
  - 感染迹象：疼痛增加、肿胀、红斑、发热、红色条纹、发烧或分泌物
  - 冰敷和抬高措施无法缓解的剧烈疼痛和肿胀
  - 无法取下戒指，且手指开始变苍白、白色或蓝色
  - 剧烈疼痛且无法执行释放指甲下血液的操作

## 足部问题（STCC/Foot_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 过敏史 / 糖尿病或血管疾病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 脚比另一只冷或发蓝

**B级（立即急诊）**（立即前往急诊室）
  - 无法行走或移动脚
  - 发烧且脚趾关节红肿痛
  - 糖尿病患者且有开放性伤口或家庭护理措施无效的感染

**C级（当天就医）**（当天内联系医生/诊所）
  - 超过1周未愈合的伤口
  - 持续超过1周的肿胀
  - 脚趾关节红、肿、痛且发热，没有已知的受伤史
  - 足底有红、肿、痛区域

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 行走或跑步后的疼痛
  - 脚趾周围有红、肿、热区
  - 受伤后脚趾肿胀且淤青
  - 长时间行走或站立后双脚疼痛
  - 持续瘙痒
  - 足间或足底皮肤软、红、脱皮

**D级居家护理**
  - 出现感染迹象（疼痛、肿胀、红斑、红线、分泌物或温暖），每天4至6次，每次20分钟，在受影响区域使用湿热敷包。
  - 受伤后，抬高脚并每天4至6次，每次20分钟，将冰袋放在该区域。在最初的24到48小时内用弹性绷带包裹以减少肿胀。不要直接将冰放在皮肤上，应在冰和皮肤之间放置一块洗布或其他织物屏障。
  - 用胶带将受伤的脚趾固定到相邻的脚趾7至10天，提供支撑并促进愈合。
  - 在疼痛硬结周围应用海绵环，以减少鞋子对区域的压力。

## 手/腕部问题（STCC/HandWrist_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药情况 / 破伤风免疫状态

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 无法通过加压止血或有喷射性出血
  - 骨头穿透皮肤
  - 意识改变
  - 受伤部位远端没有脉搏
  - 呼吸困难
  - 胸闷和疼痛放射至颈部、下颌或肩部

**B级（立即急诊）**（立即前往急诊室）
  - 受影响肢体的手指发冷、变蓝且麻木
  - 挤压伤
  - 穿刺伤：子弹、刀子、钉子等
  - 高压注射伤害
  - 明显的畸形，但远端脉搏存在
  - 体力劳动后突然出现手臂或手部疼痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 突然严重肿胀
  - 石膏或夹板固定，并且颜色、感觉或运动减弱
  - 深割伤，已止血
  - 突发剧烈疼痛
  - 包扎的肢体在手指活动时感到疼痛

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 中度疼痛
  - 无法从伤口清除泥土或其他异物
  - 活动时疼痛
  - 发烧、肿胀、伤口有分泌物或红纹
  - 不能握拳
  - 受伤区域上方或下方的关节无法移动

**D级居家护理**
  - 从受影响的肢体上取下所有戒指。
  - 将手抬高至心脏以上，以帮助减少肿胀。
  - 在受伤部位敷冰袋24小时内以减轻肿胀和疼痛。之后交替使用冷热敷。不要直接将冰放在皮肤上；应在冰与皮肤之间放置湿布或其他布料屏障。
  - 对腕部和手部施加压力绷带，并在最初24至48小时内固定，以减少肿胀。如果应用后出现麻木或刺痛，请松开绷带。

## 髋部疼痛/损伤（STCC/Hip_PainInjury.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药情况 / 破伤风免疫状态

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 骨骼穿出皮肤
  - 患肢较另一侧短
  - 明显畸形
  - 跌倒后无法负重站立
  - 足部外旋或内旋
  - 运动或感觉减退

**B级（立即急诊）**（立即前往急诊室）
  - 行走时剧烈疼痛
  - 严重瘀伤
  - 麻木或刺痛感
  - 损伤及有出血障碍病史

**C级（当天就医）**（当天内联系医生/诊所）
  - 近期损伤或跌倒后，行走或移动时中等程度疼痛

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 通常止痛措施无法缓解的慢性疼痛
  - 损伤、运动或跌倒后的间歇性疼痛

**D级居家护理**
  - 尽可能在48小时内让受伤部位休息。
  - 前24小时将腿抬高放在枕头上。
  - 在前24至48小时内，每3到4小时对受伤区域应用冰袋20到30分钟，之后交替使用热敷。不要直接将冰块放在皮肤上；在冰和皮肤之间放置一块湿布或其他织物屏障。
  - 按常规服用止痛药。不要给儿童服用阿司匹林。年龄<20岁者避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病、胃部问题或孕妇应避免使用布洛芬。遵循药品标签上的说明。

## Few-shot 示例

**用户问题**：我搬重物后腰痛，弯腰困难，但腿没有麻木感。

**正确回答**：

【严重度判定】
**D级（居家观察）**
急性腰扭伤，无神经症状（腿麻/放射痛），暂不需急诊，可先居家护理。

【家庭护理 / 立刻行动】
- 前48小时冰敷（每次20分钟，每天3-4次）
- 之后可改为热敷
- 尽量保持日常轻度活动（完全卧床反而不利恢复）
- 可按说明书服用布洛芬等非处方止痛药

【回电 / 复诊标准】
需就医的信号：
- 疼痛向臀部或腿部放射（坐骨神经痛可能）
- 出现腿部麻木或无力
- 大小便控制困难（紧急，须立即急诊）
- 症状3-5天无改善

【依据】
参考：STCC/Back_Pain.md
本建议仅供参考，最终请遵医嘱。如有疑虑请直接拨打120。
