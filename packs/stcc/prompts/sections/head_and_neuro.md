# head_and_neuro 分诊知识片段

本片段整合了 STCC 指南中 **head_and_neuro** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 头部损伤（STCC/Head_Injury.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 过敏史 / 受伤机制

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 手臂或腿部移动困难、无力、不协调或言语不清
  - 严重颈痛

**B级（立即急诊）**（立即前往急诊室）
  - 呼吸异常或呼吸困难
  - 意识改变、难以唤醒、混乱、激动
  - 无法控制的出血
  - 抽搐活动
  - 前语言期儿童不认识父母或看护人
  - 持续视觉障碍

**C级（当天就医）**（当天内联系医生/诊所）
  - 鼻孔或耳朵持续流出血液或液体
  - 张口、裂开、参差不齐或深伤口
  - 耳后或眼下的瘀斑
  - 持续出血>10分钟
  - 丧失意识的历史
  - 受伤后行为改变

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 1岁以下婴儿头部有软绵绵的肿胀区域持续>12小时
  - 额头或头皮上有肿块（鹅蛋状）
  - 间歇性头痛对止痛药无反应

**D级居家护理**
  - 每2小时使用冰袋或冷敷20分钟以减轻肿胀和不适。在皮肤与冰之间放置布障。
  - 允许儿童受伤后睡眠。24小时内每2小时唤醒一次，评估其警觉性和反应性。
  - 受伤后的头24小时内避免剧烈活动。在安静的环境中休息，头部略微抬高。
  - 使用对乙酰氨基酚缓解不适。不要给儿童服用阿司匹林。如果年龄<20岁，避免类似阿司匹林的产品。如果有肝病，请避免使用对乙酰氨基酚。如果有肾病、胃部问题或怀孕的情况下，避免使用布洛芬。请遵循标签上的说明。

**回电/复诊标准**
  - 状况未改善或恶化
  - 持续头痛
  - 冰敷后超过24小时的持续肿胀
  - 鼻孔或耳朵流出血液或清澈液体

## 头痛（STCC/Headache.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 过敏史 / 偏头痛或已知动脉瘤病史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 突然剧烈疼痛或“一生中最严重的头痛”
  - 突然出现的身体一侧无力、步态不稳、麻木和/或刺痛
  - 混乱、唤醒困难、行为异常
  - 突然说话困难或言语不清
  - 颈部僵硬伴发热（低头时疼痛）
  - 视力模糊或复视

**B级（立即急诊）**（立即前往急诊室）
  - 持续呕吐
  - 高血压病史和头晕
  - 体温>103°F (39.4°C)且对退烧措施无反应
  - 眼睛疼痛伴有红肿和视力下降
  - 近期被蜱虫叮咬，头痛、皮疹及流感样症状
  - 新发剧烈持久性疼痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 对常规偏头痛治疗无反应的持续偏头痛
  - 偏头痛强度或性质与以往不同
  - 头痛持续>12小时且无其他症状
  - 疼痛影响活动
  - 颞部疼痛且年龄>50岁

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 发热和喉咙痛
  - 肌肉和关节酸痛
  - 近期有压力性事件
  - 过敏史
  - 钝痛持续，颈部肌肉触痛紧绷
  - 最近停止饮用咖啡、吃巧克力或吸烟

**D级居家护理**
  - 每2小时在额头敷冷敷包或冰袋。
  - 在安静、昏暗的房间休息。
  - 服用常规止痛药（阿司匹林、对乙酰氨基酚、布洛芬）。不要给儿童使用阿司匹林。年龄<20岁者避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病或胃部问题存在时，或在怀孕情况下避免使用布洛芬。遵
  - 用常规的退烧措施降低体温。

## 疑似中风（STCC/Stroke_Suspected.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 既往中风史 / 充血性心力衰竭 / 房颤

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 突然一侧面部下垂，微笑偏斜，伸出舌头时偏向一边
  - 突然手臂或腿无力或麻木；如果要求其举起双臂，一只会掉下来
  - 言语不清或难以理解
  - 突然变得混乱或迷失方向
  - 突然头痛剧烈
  - 突然从一只眼或双眼看东西困难或出现不自主的眼球运动

**B级（立即急诊）**（立即前往急诊室）
  - 突然感到头晕或轻飘飘的
  - 突然平衡或协调能力受损，或行走困难
  - 突然失去记忆
  - 突然肌肉僵硬或身体某部分感觉丧失
  - 突然眼球不自主运动或从一只眼或双眼看东西困难
  - 突然行为改变

**C级（当天就医）**（当天内联系医生/诊所）
  - 无明显改善或状况恶化
  - 没有改善或病情加重
  - 突然一侧面部下垂，微笑偏斜，伸出舌头时偏向一边
  - 突然手臂或腿无力或麻木；如果要求其举起双臂，一只会掉下来
  - 言语不清或难以理解
  - 突然变得混乱或迷失方向

## 意识状态改变 (AMS)（STCC/Altered_Mental_Status_AMS.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 已知原因 / 用药情况 / 既往病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 无意识，停止呼吸

**B级（立即急诊）**（立即前往急诊室）
  - 当天多次失去意识
  - 拨打时无法唤醒
  - 药物/酒精过量
  - 呼吸困难
  - AMS伴有以下任一症状：
  - 剧烈头痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 头痛、发热或颈部僵硬疼痛
  - 近期头部受伤或创伤
  - 新出现癫痫发作或长时间的发作后状态
  - 持续高热
  - 新出现听觉（如听到声音、嗡嗡声、咔嚓声）、感觉（如虫爬）或视觉幻觉或妄想

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 短暂的失去意识发作

**D级居家护理**
  - 保持患者安全平躺直至完全清醒和有反应。
  - 若呕吐，将患者侧卧。
  - 不要让患者无人看管。
  - 如果糖尿病患者突然出现意识水平变化，立即给予糖分来源，但前提是患者足够清醒能够进食或饮水。快速作用的糖分来源包括橙汁或其他果汁、调味饮料（如酷爱）、常规可乐、糖果（如救生圈或软糖）或一茶匙糖或果酱。

## 混乱（STCC/Confusion.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 用药 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 近期有头部受伤史
  - 接触化学品或药物摄入
  - 糖尿病
  - 中风、高血压或心脏病史
  - 对姓名、日期、地点或情境定向障碍
  - 呼吸带有果味

**B级（立即急诊）**（立即前往急诊室）
  - 新发的幻觉或偏执症
  - 有药物或酒精滥用史
  - 儿童、老年人或免疫功能低下者体温 >101°F (38.3°C)
  - 癫痫发作病史

**C级（当天就医）**（当天内联系医生/诊所）
  - 正在服用已知可引起混乱的药物
  - 最近开始使用新药
  - 体温 >101°F (38.3°C)
  - 有痴呆或慢性脑综合征病史且精神状态改变
  - 最近突然停止使用（非处方药或处方药）、酒精或咖啡因

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 有痴呆或慢性脑综合征病史且无状态变化

**D级居家护理**
  - 如果正在服用可能导致谵妄的药物（如抗组胺药、颠茄碱、生物碱），请停止使用，并在1小时内如果没有改善，请回拨或联系家庭医生。
  - 让患者在一个光线充足、熟悉的环境中舒适地休息，旁边有人陪伴。
  - 给予常规退烧药（如对乙酰氨基酚）。用温水浴降低体温。不要给儿童服用阿司匹林。20岁以下者避免使用类似阿司匹林的产品。肝病存在时避免使用对乙酰氨基酚。肾病、胃部问题或怀孕的情况下避免使用布洛芬。请按照标

**回电/复诊标准**
  - 持续超过1小时的混乱状态
  - 退烧后仍持续混乱
  - 谵妄缓解后仍有其他症状存在

## 发作（STCC/Seizure.md）

**关键追问**：姓名 / 年龄 / 发作时间 / 既往发作史 / 过敏史 / 用药史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 多次发作
  - 呼吸困难
  - 发作持续>5分钟
  - 严重头痛
  - 持续异常嗜睡
  - 近期头部受伤史

**B级（立即急诊）**（立即前往急诊室）
  - 发作期间受伤
  - 有习惯性大量饮酒或用药史，且最近戒酒或停药
  - 在服用抗癫痫药物的情况下仍频繁发作

**C级（当天就医）**（当天内联系医生/诊所）
  - 停止服用抗癫痫药物
  - 有糖尿病、脑血管意外、癌症、心血管疾病或神经肌肉疾病的病史

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 有发作史，且从发作中苏醒后神志清醒

**D级居家护理**
  - 将患者侧卧或俯卧，头偏向一侧以防分泌物或呕吐物导致窒息。
  - 如呼吸有杂音，向前拉下巴和下颌。不要将手指、药物或其他物体放入抽搐者的口中。
  - 松开领带或其他紧身衣物。
  - 将发作中的人员移至远离可能造成伤害的物品的安全区域。

**回电/复诊标准**
  - 反复发作活动
  - 无改善或病情恶化
  - 发热对退热措施无效

## 发热性惊厥（STCC/Seizure_Febrile.md）

**关键追问**：姓名 / 年龄 / 发作时间 / 既往惊厥史 / 体温（如果已知） / 孩子是否感觉发热

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 多次惊厥
  - 呼吸困难或停止呼吸 >60 秒
  - 惊厥持续 >5 分钟

**B级（立即急诊）**（立即前往急诊室）
  - 首次发作的惊厥
  - 儿童年龄小于 6 个月或大于 5 岁
  - 严重头痛，颈部僵硬或疼痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 持续高热 >102°F (38.9°C) 对退烧措施无反应

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 惊厥后清醒且有方向感
  - 孩子希望在惊厥后睡觉，但可以被唤醒而不会烦躁

**D级居家护理**
  - 将患者侧卧或俯卧，头部转向一侧以防止因分泌物或呕吐引起的窒息。
  - 如果有呼吸嘈杂声，前拉下颚和下巴。不要将手指、药物或其他任何物体放入正在抽搐的人的口中。
  - 松开衣物。
  - 将抽搐者移动到远离可能造成伤害物体的安全区域。

**回电/复诊标准**
  - 反复发作的惊厥活动
  - 没有好转或病情恶化
  - 发热对退烧措施无反应

## 麻木和刺痛（STCC/Numbness_and_Tingling.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 位置 / 相关症状

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 精神状态改变

**B级（立即急诊）**（立即前往急诊室）
  - 剧烈头痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 身体一侧受影响
  - 突然的手臂和/或腿部无力
  - 言语困难，口齿不清
  - 视力模糊
  - 大小便失禁
  - 手指或脚趾比另一侧冷或蓝

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 近期有过重体力劳动或剧烈运动的病史
  - 区域疼痛、肿胀和/或温暖
  - 严重疼痛
  - 夜间手或臂部出现麻木、刺痛和/或锐痛
  - 近期有疾病或手术的病史
  - 逐渐起始的症状

**D级居家护理**
  - 避免长时间保持同一姿势。
  - 定期在受影响区域收紧和放松肌肉以促进血液循环。
  - 保护感觉麻木的部位以防受伤。
  - 为了减缓快速呼吸并控制麻木和刺痛，请参阅下一页。

**回电/复诊标准**
  - 20到30分钟内无改善或情况恶化
  - 症状影响日常生活

## Few-shot 示例

**用户问题**：我头磕到了桌子，有点头痛，没有晕倒，意识清醒。

**正确回答**：

【严重度判定】
**C级（当天内就医）**
头部轻微外伤，无意识丧失，当天内建议到医院排查颅内损伤。

【关键追问】
- 碰撞后有没有意识短暂丧失或"发呆"？
- 头痛是否在加剧？

【家庭护理 / 立刻行动】
- 在家安静休息，避免剧烈活动
- 每30分钟检查一次精神状态
- 不要服用阿司匹林或布洛芬（可能加重出血风险）

【回电 / 复诊标准】
出现以下情况立即呼叫120：
- 意识模糊或无法正常交流
- 持续或加重的头痛
- 呕吐超过1次
- 一侧肢体无力或口角歪斜
