# gastrointestinal 分诊知识片段

本片段整合了 STCC 指南中 **gastrointestinal** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 成人腹泻（STCC/Diarrhea_Adult.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 用药情况 / 既往病史 / 伴随症状

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 腹泻伴有严重虚弱、嗜睡或昏厥
  - 严重腹痛、肿胀和发热
  - 糖尿病患者出现呼吸急促或费力

**B级（立即急诊）**（立即前往急诊室）
  - 严重腹痛
  - 明显带血的大便
  - 脱水的迹象：
  - 尿量减少
  - 眼睛凹陷
  - 松弛干燥的皮肤

**C级（当天就医）**（当天内联系医生/诊所）
  - 每30至60分钟腹泻一次超过6小时
  - 腹泻持续超过5天
  - 粘液、脓液、黄色、绿色或泡沫状大便持续超过2天
  - 持续发热
  - 新出现的大便失禁
  - 家庭护理措施后无改善

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 最近饮食变化
  - 其他家庭成员也有腹泻
  - 最近前往外国旅行
  - 过去2个月内使用过抗生素
  - 新处方药

**D级居家护理**
  - 最初12至24小时内食用清澈液体饮食（小口喝水、平汽水、清汤、果冻[非红色]、调味冰）。
  - 接下来的12小时，逐渐过渡到喝汤（避免奶油汤）、干吐司、苏打饼干、白米饭、椒盐卷饼、香蕉、苹果酱和土豆。
  - 软便形成后恢复正常饮食。
  - 腹泻缓解后的2至5天内避免乳制品、柑橘类果汁、生水果和蔬菜以及油炸或辛辣食物。

**回电/复诊标准**
  - 家庭护理措施48小时后无改善或腹泻加重
  - 黄色、泡沫状、带血或绿色大便出现超过一次
  - 脱水的迹象
  - 发热、虚弱或嗜睡

## 腹泻，儿童（STCC/Diarrhea_Child.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 频率 / 用药情况 / 既往病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 腹泻并伴有严重的无力、疲倦、无精打采或昏厥
  - 呕吐伴右上腹疼痛
  - 3个月以下婴儿腹泻且体温 >100.4°F (38°C)
  - 寒冷和灰白的皮肤
  - 大量血便
  - 快速而困难地呼吸

**B级（立即急诊）**（立即前往急诊室）
  - 脱水迹象：
  - 尿量减少
  - 1岁以下儿童4至6小时无尿
  - 1岁以上儿童12小时以上无尿
  - 体温 >104°F (40°C)，年龄>3个月，对退热措施无反应
  - 哭泣时无泪

**C级（当天就医）**（当天内联系医生/诊所）
  - 脱水迹象
  - 尿量减少
  - 口腔干燥
  - 哭泣时无泪
  - 昏昏欲睡、无力或脱力

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 黄色、泡沫状、带血或绿色的大便发生不止一次
  - 体温升高、乏力或无精打采
  - 水样腹泻和呕吐清水超过3次

## 成人呕吐（STCC/Vomiting_Adult.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 病史 / 用药情况 / 伴随症状

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 意识状态改变
  - 呕吐鲜红色血液或类似暗咖啡色的呕吐物
  - 近期头部或腹部受伤并伴有呕吐
  - 胸痛或不适，呼吸困难，心悸或出汗
  - 持续严重的腹痛影响活动

**B级（立即急诊）**（立即前往急诊室）
  - 脱水迹象：
  - 尿量减少
  - 深黄色尿液
  - 眼窝凹陷
  - 松弛干燥的皮肤
  - 极度口渴

**C级（当天就医）**（当天内联系医生/诊所）
  - 持续超过24小时的呕吐，即使采取了家庭护理措施也未能缓解
  - 近期手术、住院或诊断程序
  - 新发黄疸
  - 持续呕吐和增加使用大麻或其他聚会/非法药物

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 腹泻或便秘
  - 腹部膨胀
  - 有出国旅行或露营的历史
  - 其他家庭成员生病
  - 过量摄入食物、酒精或液体
  - 近期服用抗生素、止痛药或新药物

**D级居家护理**
  - 最后一次呕吐后1小时内不要吃任何东西或喝水。
  - 最初12小时内，每次只喝少量的清澈液体，包括明胶、水、运动饮料、无气泡的汽水、清汤、冰块（带味）或苹果汁。不要喝柑橘类果汁。
  - 根据耐受情况逐渐增加液体摄入量。
  - 12小时后尝试吃小份的清淡食物，如米饭、土豆、苏打饼干、椒盐脆饼、干面包和苹果酱。

**回电/复诊标准**
  - 发热、虚弱或腹痛
  - 48小时内没有改善或状况恶化
  - 脱水迹象

## 呕吐，儿童（STCC/Vomiting_Child.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 疑似原因 / 病史 / 药物使用情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 意识状态改变：无力，异常易怒，困惑
  - 严重头痛、颈部僵硬或前屈时疼痛
  - 呕吐鲜红色血液或类似咖啡渣的暗色呕吐物
  - 近期头部或腹部受伤
  - 接触有毒物质（如药物、植物、清洁剂、杀虫剂或野生蘑菇）
  - 新生儿<1个月大

**B级（立即急诊）**（立即前往急诊室）
  - 脱水的迹象：
  - 尿量减少
  - 眼睛凹陷或前囟凹陷
  - 口腔干燥
  - 哭泣时无泪
  - 异常无力

**C级（当天就医）**（当天内联系医生/诊所）
  - 持续时间超过12小时的呕吐
  - 体温>101°F (38.3°C) 持续>24小时
  - 在过去6小时内呕吐超过3次
  - 持续性腹泻
  - 喂食后婴儿有剧烈呕吐

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 中度腹泻或便秘
  - 近期出国旅行或露营
  - 家中其他成员患病
  - 过量摄入食物或液体
  - 最近服用抗生素、止痛药或新药物
  - 耳痛、感冒、喉咙痛或发热

**D级居家护理**
  - 每隔5分钟给予1茶匙Lytren、Pedialyte、Infalyte或Kao Lectrolyte，并根据耐受情况逐渐增加。
  - 如果婴儿喝果汁，每隔5分钟给予1茶匙，然后根据耐受情况逐渐过渡到清淡液体。如果同时有腹泻，请不要给果汁。
  - 如果母乳喂养且婴儿呕吐3次以上，每30至60分钟提供4至5分钟的哺乳，并在两次哺乳之间补充1茶匙重水合液，每隔5至15分钟一次。通常不需要停止母乳喂养。
  - 如果是配方奶喂养，请使用少量、频繁的喂食方式。

**回电/复诊标准**
  - 高烧、虚弱或持续超过2小时的腹痛
  - 48小时内无改善或状况恶化
  - 脱水迹象

## 消化不良（STCC/Indigestion.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 症状通常发生在进食后不久 / 过敏史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸困难
  - 皮肤湿冷
  - 颈部、下颌、肩膀、背部或手臂疼痛
  - 面部、嘴唇、耳垂或指甲发蓝或灰暗
  - 心脏病史或糖尿病史
  - 劳累时出现疼痛

**B级（立即急诊）**（立即前往急诊室）
  - 呕吐黑色咖啡渣样物质
  - 剧烈腹痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 服用药物后不适持续存在
  - 状况恶化，需要更频繁地使用药物
  - 经常呕吐、体重减轻或食欲下降
  - 吞咽困难或疼痛

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 使用阿司匹林、布洛芬或类固醇后疼痛加剧
  - 弯腰、运动或饭后不久躺下时疼痛加剧
  - 经常嗳气或放屁
  - 口中酸味
  - 腹胀或饱胀感
  - 最近压力增加

**D级居家护理**
  - 尝试非处方药物（如Maalox、Mylanta、Riopan、Tums、Pepcid、Prilosec），并按照标签上的说明使用。液体通常比片剂提供更快的缓解作用。如果正在服用其他处方药，请咨询家庭医
  - 不要给儿童服用Pepto-Bismol。
  - 避免在睡前2到3小时内进食。
  - 饭后2到3小时内不要躺下，也不要在饭后不久弯腰或运动。

**回电/复诊标准**
  - >3天后持续不适且未见改善或状况恶化
  - 抗酸药无效
  - 吞咽困难或疼痛

## 便秘（STCC/Constipation.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 最后一次排便 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 严重腹痛，肿胀，膨胀或呕吐

**B级（立即急诊）**（立即前往急诊室）
  - 持续性呕吐和渐进性腹部肿胀
  - 剧烈疼痛或绞痛
  - 呕吐出棕色、黄色或苦味的液体
  - 有便血且无痔疮史或便秘伴随出血
  - 婴儿<1个月，母乳喂养，出现脱水迹象

**C级（当天就医）**（当天内联系医生/诊所）
  - 5至7天未排便，且家庭护理措施无效的便秘
  - 近期手术、受伤或分娩后，家庭护理措施未能缓解症状
  - 有憩室炎史并伴有发热
  - 婴儿超过6到10天无大便
  - 儿童/婴儿哭闹、腹胀、排出硬便，对家庭护理措施反应不佳
  - 24至48小时内不明原因的发热

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 干硬的大便
  - 排便时疼痛
  - 大便或排便习惯近期改变
  - 慢性便秘
  - 小而频繁出现的液体或硬便
  - 间歇性便秘

**D级居家护理**
  - 确保饮食量足够（数量），含纤维丰富，水分充足（除非在限制液体摄入的饮食中，否则每天应喝6到8杯[每杯8盎司]水）。将李子或李子汁、新鲜水果、豆类和全谷物加入饮食。
  - 避免导致便秘的食物（奶制品、米饭、香蕉、白面包和加工食品）。
  - 每天早晨喝一杯热饮，如咖啡、茶或加柠檬的热水。
  - 每天固定时间保证隐私并排便。

## 成人食欲减退（STCC/Appetite_Loss_Adult.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 用药情况 / 既往病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 意识障碍
  - 自杀念头
  - 幻觉（听觉、触觉或视觉）

**B级（立即急诊）**（立即前往急诊室）
  - 体温>101°F (38.4°C)且免疫系统减弱或年龄较大
  - 已知或疑似饮食障碍，以及坐立时头晕和心率持续增加

**C级（当天就医）**（当天内联系医生/诊所）
  - 突然体重减轻>5至10磅
  - 心脏病史、肢体肿胀、水肿、癌症、病毒感染、胃肠炎、寄生虫、甲状腺功能亢进或肠道疾病
  - 皮肤或眼白发黄
  - 体温>101°F (38.4°C)且退热措施无效
  - 突然体重增加>5磅
  - 水分摄入不足

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 不良饮食习惯
  - 近期手术
  - 压力/焦虑增加
  - 食欲减退新出现
  - 无其他症状
  - 活动减少

**D级居家护理**
  - 吃均衡的餐食。
  - 手术或疾病后缓慢增加食物量。
  - 定期锻炼。
  - 可尝试液体饮食补充剂（如Ensure、Boost）。向您的PCP或药剂师咨询其他非处方液态饮食补充剂。

## 脱水（STCC/Dehydration.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 药物 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 精神状态改变：倦怠、异常易怒、混乱、谵妄或难以唤醒
  - 严重头痛、颈部僵硬或前屈头部时疼痛
  - 呕吐鲜红色血液或类似深咖啡色的呕吐物
  - 血便或黑便
  - 严重腹痛或胸痛
  - 呼吸困难

**B级（立即急诊）**（立即前往急诊室）
  - 儿童4至6小时内无尿
  - 成人的脱水迹象：
  - 排尿不频繁（>12小时内<1次）
  - 尿液呈深黄色
  - 眼睛凹陷或前囟门凹陷
  - 捏起的皮肤无法迅速回弹

**C级（当天就医）**（当天内联系医生/诊所）
  - 头晕、头重脚轻、行走不稳或站立不稳

## Few-shot 示例

**用户问题**：我今天呕吐了3次，腹泻2次，喝水也吐，没有发烧。

**正确回答**：

【严重度判定】
**C级（当天内就医）**
频繁呕吐导致无法进水，有脱水风险，当天内建议就医补液。

【关键追问】
- 最后一次排尿是什么时候？
- 是否有头晕或站立时晕倒感？

【家庭护理 / 立刻行动】
- 暂停进食，分次小量饮用口服补液盐或清澈液体（每10分钟喝一小口）
- 平躺休息，保持温暖
- 避免乳制品、油腻食物

【回电 / 复诊标准】
立即就医信号：
- 8小时以上未排尿
- 出现极度口渴、眼眶凹陷
- 便血或呕血
