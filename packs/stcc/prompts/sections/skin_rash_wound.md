# skin_rash_wound 分诊知识片段

本片段整合了 STCC 指南中 **skin_rash_wound** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 成人皮疹（STCC/Rash_Adult.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 位置 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 突发严重荨麻疹或皮疹，呼吸困难，胸闷，喉咙后部或舌头肿胀，或意识改变

**B级（立即急诊）**（立即前往急诊室）
  - 突然发病并迅速发展为广泛红斑、鳞屑、发热和淋巴结肿大
  - 紫色或血色的平点或小点，头痛，颈部僵硬，呕吐，意识混乱或发热
  - 严重的面部或眼部肿胀

**C级（当天就医）**（当天内联系医生/诊所）
  - 新药物相关的皮疹，停止使用该药物。如果是处方药，请联系PCP。
  - 按照容器上的指示服用抗组胺药（如苯海拉明、氯雷他定、阿勒维特或氯-曲美通），直到皮疹和瘙痒消失。前列腺肿大者不要服用抗组胺药。

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 使用炉甘石洗剂或1%氢化可的松软膏。
  - 每2至3小时无肥皂冷浴或淋浴以缓解症状，自然晾干。
  - 在受影响区域使用婴儿爽身粉。
  - 尽可能在暴露后1小时内清洗受暴露区域。
  - 用冷水浸泡或冰敷20分钟，按需进行。
  - 清洗所有与这些植物接触过的衣物和动物。

## 荨麻疹（STCC/Hives.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 疑似原因 / 皮肤上的红肿痒斑块 / 过敏史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸困难、胸闷或喘息
  - 吞咽困难或舌头或喉咙后部肿胀
  - 困惑、激动或意识水平下降
  - 有蜂蜇史及对蜂蜇过敏
  - 对同一抗原的严重过敏反应史

**B级（立即急诊）**（立即前往急诊室）
  - 眩晕或头晕
  - 剧烈腹痛
  - 恶心或呕吐
  - 咳嗽快速发作
  - 突然声音嘶哑

**C级（当天就医）**（当天内联系医生/诊所）
  - 身体多处迅速出现红肿痒皮疹
  - 脸部或四肢肿胀
  - 服用抗组胺药两天后仍有荨麻疹

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 腹泻或发热
  - 关节肿胀或疼痛
  - 过去接触过需要激素治疗的林木植物（如毒葛、毒橡树或毒漆树）
  - 新用药物
  - 饮食中新添加的食物
  - 使用新的肥皂、洗发水、洗衣粉、衣物、化妆品、牙膏或其他个人护理用品

**D级居家护理**
  - 对于广泛分布的荨麻疹，可服用非处方抗组胺药（如苯那君、开瑞坦、阿拉维特），直到荨麻疹消失。请按照说明书上的指示使用。服药后不要驾驶或从事需要集中注意力的活动。如有其他产品建议，请咨询药师。
  - 如果怀疑药物是引起反应的原因，应停止使用并联系家庭医生。
  - 冷水浴、小苏打水浴或燕麦粉水浴可能有助于缓解瘙痒和不适感。
  - 涂抹Caladryl或炉甘石洗剂可帮助控制瘙痒和干燥病灶。请按照说明书上的指示使用。限制对幼儿使用Caladryl，因为它可能导致嗜睡。

**回电/复诊标准**
  - 持续超过7天的荨麻疹
  - 服用抗组胺药24小时后仍持续瘙痒

## 瘀伤（STCC/Bruising.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 瘀伤原因 / 瘀伤大小及颜色 / 用药情况（是否有使用华法林或阿司匹林的历史）

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 背部下侧、骨盆、胸部或腹部严重疼痛和瘀伤
  - 眼部受到撞击导致的瘀伤，并且有
  - 虹膜出血严重
  - 视力减退或复视
  - 眼球在各个方向移动困难
  - 眼睛剧烈疼痛

**B级（立即急诊）**（立即前往急诊室）
  - 肢体严重疼痛和瘀伤
  - 受伤部位严重肿胀
  - 原因不明的多处瘀伤
  - 疑似儿童虐待
  - 出血病史或使用抗凝药物的历史

**C级（当天就医）**（当天内联系医生/诊所）
  - 活动受限
  - 感染迹象：疼痛加剧、肿胀、发红、渗出物、发热、热感，或者从该区域延伸的红色条纹
  - 频繁跌倒
  - 最近突然停止使用药物（非处方药、处方药、娱乐性），酒精或咖啡因

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 轻微肿胀
  - 轻微不适

**D级居家护理**
  - 休息受影响区域。
  - 不要按摩或揉搓该区域；这可能会增加瘀伤。
  - 在受伤的24至48小时内，使用冰袋冷敷以减轻肿胀和疼痛。不要直接将冰放在皮肤上；应在冰与皮肤之间放置湿布或其他织物屏障。
  - 抬高受影响部位，以帮助减少肿胀和出血。

**回电/复诊标准**
  - 严重疼痛
  - 活动受限
  - 频繁瘀伤
  - 48小时后无改善或病情恶化

## 面部皮肤问题（STCC/Facial_Skin_Problems.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 面部突然下垂
  - 严重的面部肿胀
  - 迅速扩散的红或紫色皮疹，并在粘膜（唇、口、眼、生殖器）上形成水泡
  - 大面积皮肤剥落或脱落
  - 无法解释的广泛性疼痛

**B级（立即急诊）**（立即前往急诊室）
  - 感染迹象：肿胀、红斑、疼痛、发热（体温 >100°F (37.8°C)）、分泌物或红线
  - 大的开放性的排出液体的病变

**C级（当天就医）**（当天内联系医生/诊所）
  - 鼻孔周围或唇部有结痂、疼痛的病变
  - 面部一侧出现由1-2天前的疼痛和灼烧感先于皮疹的水泡
  - 持续存在的扁平或稍隆起的边界不规则的病变
  - 痣的颜色、大小或形状最近迅速变化

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 瘙痒性皮疹影响睡眠
  - 对抗生素和家庭护理措施无反应的持续性痤疮
  - 面部潮红
  - 唇部周围有水泡或红色、粗糙或疼痛区域
  - 皮肤下肿大且触痛的硬块
  - 突然出现痣或无痛性肿块

**D级居家护理**
  - 使用温和的肥皂（非处方产品，例如 Aveeno, Basis, Cetaphil, Dove, Neutrogena）清洗，并充分冲洗。
  - 对于感染迹象，每天4到6次，每次20分钟，将热湿敷布应用于患处。
  - 不要挤压或刺破病变部位。
  - 保持头皮头发远离面部。

**回电/复诊标准**
  - 口腔周围持续超过14天的溃疡
  - 没有改善或情况恶化
  - 感染迹象
  - 严重的疼痛或瘙痒影响活动

## 异物，皮肤（STCC/Foreign_Body_Skin.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 异物 / 过敏史 / 药物使用情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 无法移除嵌入的鱼钩
  - 异物嵌入关节空间
  - 深部异物
  - 无法移除外来物质
  - 外来物质粘附于皮肤上
  - 无法移除嵌入耳垂或其他部位的身体穿孔或异物

**B级（立即急诊）**（立即前往急诊室）
  - 无法移除大的刺
  - 嵌入玻璃、塑料或金属物体
  - 破伤风疫苗接种时间超过5年
  - 蜱虫头部嵌入皮肤且无法移除

**C级（当天就医）**（当天内联系医生/诊所）
  - 无法移除小刺
  - 持续感觉皮下有异物

**D级居家护理**
  - 鱼钩移除技巧：
  - 将倒钩推过皮肤，用钢丝钳剪掉倒钩，然后将其从进入的方向拉出。
  - 将倒钩向内和向下稍许推动以脱离皮肤，并将其从皮肤中拔出。
  - 通过鱼钩的弯曲部分套上钓鱼线。将倒钩向内压下并稍微保持不动以使倒钩脱离皮肤，快速拉动钓鱼线取出鱼钩。

**回电/复诊标准**
  - 48小时后仍无法移除异物
  - 感染迹象：红肿加剧、疼痛、发热、分泌物或红线
  - 剧烈疼痛

## 脱发（STCC/Hair_Loss.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 位置 / 过敏史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 新发脱发伴发热 >101°F (38.3°C)，且退热措施无效
  - 脱发区域感染迹象

**B级（立即急诊）**（立即前往急诊室）
  - 持续严重疲劳
  - 服用新药后突然脱发
  - 有甲状腺功能减退病史
  - 头皮上出现直径 ½ 至 1 英寸、中间清晰且发痒的圆形红色斑块

**C级（当天就医）**（当天内联系医生/诊所）
  - 压力水平升高
  - 持续脱发 >2 周
  - 有鳞屑、瘙痒、红秃斑点
  - 近期怀孕
  - 家族脱发史
  - 不良饮食习惯

**D级居家护理**
  - 请放心，突发性脱发后头发通常会再生。
  - 避免使用橡皮筋、发夹、编辫子或其他对头发施加张力的发型。
  - 避免拉扯头发。梳洗湿发时使用护发素或防缠结剂减少打结。
  - 如果头发因化学物质或美发技术严重受损，考虑改变发型。

## 瘙痒（STCC/Itching.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 药物使用情况 / 病史 / 伴随症状

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 全身多处严重瘙痒，广泛性荨麻疹，呼吸困难或面部、口腔、喉咙肿胀

**B级（立即急诊）**（立即前往急诊室）
  - 全身性瘙痒，或皮肤和眼睛发黄
  - 服用新药后出现皮疹并伴有瘙痒
  - 夜间特别严重瘙痒；皮肤褶皱处有红点；其他家庭成员也有类似症状
  - 持续的瘙痒影响日常活动
  - 头皮瘙痒和秃斑
  - 最近接触过毒葛或毒橡树，且家庭护理措施无效的持续性瘙痒皮疹

**C级（当天就医）**（当天内联系医生/诊所）
  - 多处昆虫叮咬
  - 肛门、阴道或生殖器周围瘙痒
  - 头皮或阴部瘙痒，头发根部有白色圆形斑点且无法去除
  - 经常接触水分或化学物质的手部瘙痒
  - 穿着新衣服后出现的瘙痒
  - 最近接触过毒葛或毒橡树的新发瘙痒皮疹

**D级居家护理**
  - 在受影响区域敷冷敷。将布料浸泡在冰水中。
  - 泡苏打水或燕麦浴，或者使用棉质布和煮熟的燕麦制成燕麦海绵，或者涂抹与白醋混合的小苏打膏。
  - 将Caladryl乳液或Domeboro溶液涂抹于昆虫叮咬处及毒葛或橡树皮疹。遵循标签上的说明。
  - 肛门周围的瘙痒可能是由痔疮或蛲虫引起的。请参阅相应的痔疮或蛲虫协议。

**回电/复诊标准**
  - 家庭护理措施超过3天后仍未改善或病情恶化
  - 其他家庭成员也有相同症状

## 虱子（STCC/Lice.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 过敏情况 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 持续性皮疹和瘙痒影响睡眠
  - 治疗1周后皮疹仍然存在
  - 溃疡扩散或出现感染迹象
  - 治疗后新卵出现
  - 皮疹消失后再复发
  - 对非处方药或处方治疗药物的局部皮肤反应

**B级（立即急诊）**（立即前往急诊室）
  - 正在接受虱子治疗并有关于用药或防止传播给他人的问题
  - 已知接触过有虱子的人或怀疑感染

**D级居家护理**
  - 在头发湿润时寻找虱子，每次梳理一小部分，并使用细齿梳、虱子梳或用于猫狗的跳蚤梳。每2到3天重复一次，持续2周。
  - 检查其他家庭成员是否有皮疹或瘙痒。如果存在，请用抗虱洗发水治疗。
  - 如果发现虱子，请将抗虱产品（处方药或非处方药——如Rid、Nix或Pronto洗发水）涂抹在干发上10分钟，并按照标签上的说明操作。不要对蒿草过敏的儿童使用Rid。不要让有哮喘的儿童使用Nix。请务必
  - 在水槽中用冷水冲洗（Nix杀死卵）。

## Few-shot 示例

**用户问题**：我身上突然出现了很多红疹，有点痒，不发烧，昨天吃了新的药物。

**正确回答**：

【严重度判定】
**C级（当天内就医）**
用药后出现皮疹，需要当天就医排查药物过敏，并评估是否需要停药。

【关键追问】
- 是什么药？什么时候开始吃的？
- 皮疹是否蔓延或出现水泡？

【家庭护理 / 立刻行动】
- 暂时停用疑似引起过敏的药物（非处方药），如是处方药先联系医生
- 穿宽松棉质衣物，避免搔抓
- 不要涂抹含香料的护肤品

【回电 / 复诊标准】
立即就医信号：
- 出现呼吸困难、喉咙肿胀
- 皮疹变成水泡或脱皮
- 伴随高烧（>38.5°C）

【依据】
参考：STCC/Rash_Adult.md
本建议仅供参考，最终请遵医嘱。如有疑虑请直接拨打120。
