# abdominal_pain 分诊知识片段

本片段整合了 STCC 指南中 **abdominal_pain** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 成人腹痛（STCC/Abdominal_Pain_Adult.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 正在使用的药物 / 疼痛评分 / 伴随症状

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 昏厥（失去意识）或无反应
  - 严重虚弱和无法站立
  - 皮肤冷且苍白，或大量出汗
  - 剧烈的突然疼痛并向背部或腿部放射

**B级（立即急诊）**（立即前往急诊室）
  - 呕血或暗咖啡色呕吐物
  - 新发症状迅速恶化且年龄>60岁
  - 与痔疮或铁剂补充无关的便血或黑便
  - 月经延迟超过4周的女性突然出现腹部和肩部疼痛
  - 年龄>30岁，吸烟严重，高血压，高胆固醇，或肥胖
  - 有糖尿病、心脏病、凝血问题或充血性心力衰竭病史

**C级（当天就医）**（当天内联系医生/诊所）
  - 疼痛迅速加剧
  - 异常大量的阴道出血及可能怀孕的情况
  - 近期腹部手术、频繁跌倒，或腹部受伤的病史
  - 右下腹痛伴有食欲不振、恶心和/或呕吐，或发热
  - 误食植物、药物或化学品
  - 体温>101°F（38.3°C）且年龄>60岁，卧床不起，或免疫系统受损

**回电/复诊标准**
  - 严重疼痛>2小时
  - 体温>101°F（38.3°C）且年龄>60岁，卧床不起，或免疫系统受损
  - 体温>103°F（39.4°C）
  - 持续呕吐或腹泻和减少的口服摄入量或排尿量

## 腹痛，儿童（STCC/Abdominal_Pain_Child.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 用药情况 / 疼痛程度评分 / 伴随症状

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 持续严重的疼痛 >2 小时
  - 迅速加剧的疼痛
  - 右下腹痛伴食欲不振、恶心和/或呕吐、发热、抓腹部、弯腰行走、尖叫、呻吟呼吸或蜷缩成胸膝位躺卧
  - 异常大量的阴道出血及可能怀孕的情况
  - 摄入未知化学物质、植物或药物
  - 近期有腹部创伤

**B级（立即急诊）**（立即前往急诊室）
  - 严重的恶心和呕吐
  - 持续 >2 小时且家庭护理无效的疼痛
  - 原因不明的进行性腹部膨胀
  - 排尿痛或困难
  - 年龄 <2 岁并有间歇性疼痛
  - 疼痛影响活动能力

**C级（当天就医）**（当天内联系医生/诊所）
  - 阴道或尿道分泌物
  - 有腹痛病史，常规治疗无效
  - 显著增加的压力水平
  - 尿中带血
  - 体温 >101°F (38.3°C)，咳嗽或虚弱

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 有“紧张性胃”的病史且压力水平增加
  - 与空腹、吃某些食物，或使用止痛药、抗生素或抗炎药物相关的间歇性轻度疼痛
  - 偶尔的轻微腹泻
  - 其他家庭成员也生病了
  - 持续超过 24 小时的喉咙痛

**D级居家护理**
  - 休息。
  - 12 至 24 小时内饮用透明液体（如用水稀释一半的果汁、淡茶、肉汤、运动饮料、冰棒、果冻、清澈软饮料）或清淡饮食（如米饭、土豆、苏打饼干、椒盐脆饼、干土司、苹果酱、香蕉）。建议为小儿童或婴儿提供电解
  - 如果有腹泻，避免饮用果汁或未经稀释的运动饮料。
  - 按照药房指示服用药物。有些应在空腹时服用，而另一些则应与食物一起服用。避免使用布洛芬和其他抗炎药物。不要给儿童服用阿司匹林。如果年龄 <2 岁，则避免使用阿司匹林。严重疼痛和肿胀在睾丸或阴囊中。

## 腹部肿胀（STCC/Abdominal_Swelling.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 用药情况 / 既往病史 / 疼痛评分

**B级（立即急诊）**（立即前往急诊室）
  - 近期创伤或腹部手术史
  - 新发黑便或血便

**C级（当天就医）**（当天内联系医生/诊所）
  - 肿胀在过去的24小时内突然出现，且排气或呕吐后未缓解
  - 按压不消退的疼痛或触痛区域

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 踝部肿胀
  - 尤其夜间呼吸困难
  - 尿量减少
  - 排尿后肿胀减轻
  - 新发黄疸（皮肤和眼睛变黄）
  - 按压或咳嗽时疼痛或触痛区域消失或增大

**D级居家护理**
  - 每天适量饮水，根据耐受程度。
  - 饮食中包含水果和高纤维食物。
  - 建立每日排便习惯。
  - 避免产生气体的食物，如洋葱、卷心菜和豆类。

## Few-shot 示例

**用户问题**：我右下腹痛了两个小时，按压时更疼，还有点发烧。

**正确回答**：

【严重度判定】
**B级（立即急诊）** → 请立即前往急诊室。
右下腹压痛伴发热，符合急性腹腔炎症（如阑尾炎）高风险特征，需要尽快排查。

【关键追问】
- 体温是多少度？
- 是否有恶心或呕吐？

【家庭护理 / 立刻行动】
前往急诊途中：保持安静，不要进食或饮水（可能需要手术），让他人驾车送医。

【回电 / 复诊标准】
如等待过程中出现意识模糊、腹部极度硬结或疼痛突然加剧，立即拨打120。
