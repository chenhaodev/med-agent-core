# fever 分诊知识片段

本片段整合了 STCC 指南中 **fever** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 成人发热（STCC/Fever_Adult.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 体温及测量方法 / 过敏史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 意识混乱，谵妄，难以唤醒或呼吸困难
  - 老年人或免疫抑制者（如艾滋病、化疗、服用类固醇）脱水迹象：
  - 尿量减少
  - 眼窝凹陷
  - 皮肤弹性差
  - 极度口渴，口腔干燥

**B级（立即急诊）**（立即前往急诊室）
  - 老年人或免疫抑制者体温>101°F（38.3°C）
  - 体温>104°F（40°C）且对退热措施无反应
  - 频繁或疼痛性排尿及背部或腰部疼痛
  - 糖尿病，使用类固醇，癌症，艾滋病，高龄，心脏病，肾脏或肝脏疾病，慢性病史，移植手术史，或怀孕
  - 静脉注射药物滥用，存在PICC线或外周静脉导管

**C级（当天就医）**（当天内联系医生/诊所）
  - 发热持续>72小时且无已知原因
  - 咳嗽伴彩色痰液
  - 频繁或疼痛性排尿
  - 阴道出血，疼痛或分泌物
  - 呕吐，腹泻或腹痛
  - 耳痛，咽痛及淋巴结肿大

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 充血、打喷嚏或酸痛
  - 其他家庭成员也生病了
  - 发热对自我护理措施有反应

**D级居家护理**
  - 增加液体摄入量（除非患者处于限液饮食状态或患有充血性心力衰竭）。
  - 服用常规药物（阿司匹林、对乙酰氨基酚或布洛芬）以缓解发热和酸痛。不要给儿童使用阿司匹林。<20岁者避免使用类似阿司匹林的产品。如存在肝病，避免使用对乙酰氨基酚。如果存在肾病或胃部问题，或怀孕中，请勿使
  - 用温水擦浴或浸泡；不要使用酒精擦拭或加入水中。
  - 穿着轻便衣物并使用薄被。

**回电/复诊标准**
  - 3天内无好转或情况恶化
  - 持续发热对退热措施无反应
  - 呕吐、腹泻、疲劳、嗜睡或颈部僵硬
  - 发热或肿胀

## 发热，儿童（STCC/Fever_Child.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 持续时间 / 体温 / 过敏史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸困难（非因阻塞）
  - 皮肤或嘴唇变蓝
  - 新出现的流涎
  - 无法吞咽
  - 皮肤上出现紫色或血色斑点
  - 首次发作且无热性惊厥史的新发抽搐

**B级（立即急诊）**（立即前往急诊室）
  - 前倾头部时颈部疼痛
  - 严重头痛或腹痛
  - 呼吸困难或急促，吞咽困难，或喘息
  - 3个月以下婴儿（近期未接种疫苗）体温>100.4°F (38.0°C)
  - 发热发生在最近一次接种后超过48小时
  - 2至6个月大的婴儿体温>105°F (40.6°C) 或有腹泻和呕吐

**C级（当天就医）**（当天内联系医生/诊所）
  - 发热>24小时，年龄>2岁且无其他症状
  - 发热>72小时对退热措施无反应或无其他症状
  - 尿频或排尿疼痛
  - 病情恶化或3天后未见改善
  - 脱水迹象
  - 发热持续48小时且无其他症状

**D级居家护理**
  - 将儿童穿轻便衣物。不要用毯子包裹婴儿。尽量保持室内温度为70°F。
  - 每2到4小时检查一次儿童的体温。如果无改善，通知主治医生或回电。
  - 遵循退热药物剂量指南：对乙酰氨基酚，每次15 mg/kg，每隔4小时；布洛芬，每次10 mg/kg，每隔6小时。如果儿童有腹痛、呕吐或脱水，请勿给予布洛芬。
  - 记住发热是身体对抗感染的正常反应；即使不治疗，发热也很少超过104°F到105°F (40°C到40.6°C)。

**回电/复诊标准**
  - 体温>104°F (40°C)
  - 皮疹、颈部疼痛或严重头痛
  - 尿频或排尿疼痛
  - 病情恶化或3天后未见改善

## 多汗（STCC/Sweating_Excessive.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 过敏史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 皮肤苍白、发冷和脉搏加快
  - 胸腔、喉咙、颈部、下颌、肩膀或手臂疼痛
  - 呼吸困难

**B级（立即急诊）**（立即前往急诊室）
  - 长时间暴露于高温环境，伴有疲劳、虚弱、头晕或恶心
  - 年轻人、老年人或免疫力低下者出现脱水迹象：
  - 8小时以上尿量减少
  - 眼睛凹陷
  - 哭泣时无泪
  - 皮肤被捏后无法迅速恢复原状

**C级（当天就医）**（当天内联系医生/诊所）
  - 夜间多汗、体重下降、持续咳嗽、痰中带血
  - 持续发热和疲劳
  - 最近突然停止使用药物（非处方药、处方药或街头药物）、酒精或咖啡因

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 体重下降、食欲增加、无力、睡眠困难
  - 年龄超过38岁，无发热多汗，月经不规律或停经
  - 体温 >100°F (37.8°C)
  - 饮酒后或多剂量服用阿司匹林后出汗
  - 穿着合成材料衣物时出汗
  - 仅手和脚出汗

**D级居家护理**
  - 请记住，出汗通常是发烧、荷尔蒙变化、运动和压力的自然反应。
  - 肥胖会导致出汗增多。减重应减少出汗量。
  - 服用常规止痛药（阿司匹林、对乙酰氨基酚、布洛芬）以降低体温。不要给儿童服用阿司匹林。如果年龄<20岁，避免使用类似阿司匹林的产品。如果有肝病，避免使用对乙酰氨基酚。如果有肾病或胃部问题，或者处于孕期时
  - 如果出汗与热暴露有关，请用冷敷、冷水浴或淋浴降温，并穿轻薄的棉质衣物，喝凉爽的饮料。

## Few-shot 示例

**用户问题**：我女儿5岁，发烧38.5度，有点流鼻涕，精神还好。

**正确回答**：

【严重度判定】
**D级（居家观察）**
5岁儿童，低热（38.5°C），精神状态良好，暂无急诊指征。

【家庭护理 / 立刻行动】
- 穿轻便衣物，保持室温约22°C
- 鼓励多喝水（温水、清汤）
- 可按体重给予对乙酰氨基酚（布洛芬）退热，剂量请参照药品说明书
- 每2-4小时测量一次体温

【回电 / 复诊标准】
出现以下情况需联系医生或就诊：
- 体温升至40°C以上
- 发热超过48小时仍无改善
- 出现皮疹、颈部疼痛、严重头痛
- 精神状态变差（异常嗜睡、极度烦躁）
