# pediatric_specific 分诊知识片段

本片段整合了 STCC 指南中 **pediatric_specific** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 新生儿问题（STCC/Newborn_Problems.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 出生史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 无反应
  - 直肠温度 >100.4°F (38°C)
  - 体温 <96.8°F (36°C) 且对保暖无反应
  - 囟门凸起伴有呕吐、发热或行为异常
  - 跌落硬表面并有明显受伤
  - 不褪色的皮疹

**B级（立即急诊）**（立即前往急诊室）
  - 跌落硬表面但无明显受伤
  - 拒绝饮水 >6 小时
  - 脐带出血超过 10 分钟的直接压迫
  - 脐周有红色条纹或红斑区域
  - 身体某部位有一簇水泡
  - 呼吸急促

**C级（当天就医）**（当天内联系医生/诊所）
  - 囟门凸起但婴儿行为正常
  - 脐带间歇性出血 >3 天
  - 接近脐带的丘疹或水泡
  - 中等量脐部引流但无发热
  - 眼睛有黄色粘液分泌物
  - 红眼或肿胀的眼睑

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 间歇性囟门凸起但无疾病迹象或症状
  - 脐带少量出血
  - 肚脐内的粉红色组织
  - >2 天的脐带有恶臭（无发热或红斑）
  - 呼吸时有喘息声但无呼吸困难

**D级居家护理**
  - 出生后第一周内乳房肿胀是正常的。
  - 眼球白色部分的红色条纹与分娩过程有关，将在 2 到 3 周内消退。
  - 水汪汪的眼睛通常是由于泪管堵塞所致，一年内会自行解决。注意观察感染迹象，如红、肿或有分泌物。
  - 阴道出现清澈、白色或粉红色分泌物是正常的，并会在 2 到 3 天内消失。

## 包皮环切术后护理（STCC/Circumcision_Care.md）

**关键追问**：姓名 / 年龄 / 包皮环切日期 / 症状出现时间 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 包皮环切部位难以控制持续出血
  - 阴茎头部呈深蓝色或黑色

**B级（立即急诊）**（立即前往急诊室）
  - 超过8小时仅有少量尿液或无尿液排出
  - 年龄小于12周且体温高于100.4°F（38.0°C）
  - 出现新的水泡
  - 阴茎部有红色条纹延伸

**C级（当天就医）**（当天内联系医生/诊所）
  - 包皮环切部位有分泌物或引流液
  - 塑料环超过14天仍未脱落
  - 塑料环移至阴茎部

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 包皮环切部位发红

**D级居家护理**
  - 每次更换尿布时，在阴茎头部涂抹凡士林，以防止头4天内的结痂粘附到尿布上。
  - 每天两次和弄脏时用温水轻轻清洗阴茎，并用柔软的毛巾拍干。将尿布松散地系好，以防摩擦和出血。
  - 不要使用婴儿湿巾或酒精清洁包皮环切部位。
  - 如果包皮环切处发红，可正常使用抗生素软膏（如新霉素或多粘菌素）。请按照标签上的说明使用。

**回电/复诊标准**
  - 包皮环切部位出血
  - 阴茎部有红色条纹延伸
  - 阴茎头部呈深蓝色或黑色
  - 超过8小时仅有少量尿液或无尿液排出

## 喉炎（STCC/Croup.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 咳嗽描述 / 先前治疗 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 流口水，吞咽困难，孩子看起来病态
  - 嘴唇发绀或呈灰暗色
  - 严重呼吸困难
  - 呼吸时胸部凹陷

**B级（立即急诊）**（立即前往急诊室）
  - 吸气时有鸡鸣样声音，经过20分钟的蒸汽处理后仍未缓解，或者在过去24小时内出现三次以上此类症状
  - 孩子看起来非常病态

**C级（当天就医）**（当天内联系医生/诊所）
  - 肺炎或其他肺部疾病史
  - 哮喘史且在家使用呼吸治疗和常规支气管扩张剂后无改善
  - ⚠️ 以下情况也应判 C 级（STCC 原文为"24小时内就医"，在我们的体系映射为 C 而非 D）：
    - 犬吠样咳嗽影响睡眠（即使今天有所好转）
    - 躺下时喉炎加重
    - 发烧 >40°C 伴犬吠样咳嗽

**D级（居家观察）**（无以上加重因素时居家护理）
  - 轻微犬吠样咳嗽，不影响睡眠，精神好
  - 症状持续好转中，无新发症状

**D级居家护理**
  - 温湿空气可能有助于病情：通过覆盖在口鼻上的温暖湿润的毛巾呼吸，或者坐在充满蒸汽的浴室中20到30分钟。
  - 持续的冷雾可能有所帮助。使用冷雾加湿器或外出进入凉爽的夜晚空气中。
  - 喝温热透明液体以缓解咳嗽。
  - 按照常规剂量给发热的孩子服用药物（儿童美林、布洛芬、儿童泰诺、对乙酰氨基酚）。不要给孩子服用阿司匹林。如果年龄小于20岁，避免使用类似阿司匹林的产品。如果有肝病，避免使用对乙酰氨基酚。如果有肾病或胃部

**回电/复诊标准**
  - 无改善或病情加重
  - 呼吸困难
  - 体温 >104°F (40°C)
  - 哮喘儿童在家进行呼吸治疗后病情持续存在

## 婴儿过度哭闹（STCC/Crying_Excessive_in_Infants.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸困难
  - 极度疲倦
  - 筋疲力尽的家长表示有伤害婴儿的恐惧
  - 12周以下婴儿体温超过38°C（100.4°F）

**B级（立即急诊）**（立即前往急诊室）
  - 喷射性呕吐
  - 持续哭闹超过2小时，且家庭护理措施无效
  - 间歇性的疲倦或烦躁
  - 12周以上婴儿体温超过40°C（104°F）

**C级（当天就医）**（当天内联系医生/诊所）
  - 新发哭闹
  - 家长认为婴儿看起来生病了
  - 伴有发热、呕吐或拉耳朵的哭闹
  - 烦躁不安超过48小时
  - 夜间不喂食无法入睡
  - 尝试入睡时哭泣

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 母亲正在母乳喂养且饮食无限制
  - 12周以下婴儿主要在晚上6点到午夜期间哭闹
  - 近期接种疫苗后伴有发热
  - 体温超过38.3°C（101°F）
  - 出牙期，牙龈红肿，婴儿4至8个月大

**D级居家护理**
  - 当婴儿开始哭泣时，检查其是否需要食物、换尿布、拥抱或是否有发热。
  - 不要每次婴儿哭泣都喂食；尝试安抚15到20分钟。如果哭声持续，让婴儿平躺15至20分钟后重复此过程。寻找其他原因：拉耳朵、腹泻、眼睫毛在眼睛里或手指、脚趾、阴茎周围有头发。抱起、安慰并拍嗝婴儿。烦躁不
  - 如果主要是在晚上或夜间哭泣，婴儿可能患有肠绞痛。节奏性的安抚活动如摇晃、摆动或怀抱通常会有帮助。用烘干机加热的毯子或毛巾放在婴儿的肚子上；给婴儿洗个温水澡，提供奶嘴，并播放单调的声音。保持白天喂养间隔
  - 如果正在母乳喂养，请避免过量摄入咖啡因和吸烟。按需哺乳，而不是严格按照固定的时间表哺乳。

**回电/复诊标准**
  - 哭闹变得严重、持续，并且影响婴儿睡眠超过2小时
  - 任何新的症状或体征
  - 哭闹两天后仍未缓解
  - 家长/照顾者感到筋疲力尽

## 尿布疹（STCC/Diaper_Rash.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 病史 / 药物 / 皮疹描述

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 大面积疼痛性水泡或开放性伤口
  - 脓液、疖子、丘疹或结痂
  - 影响睡眠的不适
  - 皮疹扩散到尿布区域之外
  - 鲜红色固定皮疹
  - 阴茎末端溃疡或结痂

**B级（立即急诊）**（立即前往急诊室）
  - 皮疹超过3天
  - 腹股沟和尿布区域红或渗出性皮疹

**D级居家护理**
  - 尽可能让皮肤暴露于空气中—每天至少1小时。
  - 频繁更换尿布。每小时检查一次湿或脏情况。
  - 每次换尿布时用温水清洁该区域。每日使用温和的肥皂清洗一次并晾干。
  - 皮肤干燥后，涂抹保护层，例如非处方A和D软膏、迪塞汀或氧化锌。这有助于保护皮肤，尤其是在腹泻的情况下。

## 婴儿睡眠呼吸暂停（STCC/Sleep_Apnea_Infant.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 婴儿停止呼吸
  - 皮肤变蓝

**B级（立即急诊）**（立即前往急诊室）
  - 呼吸暂停超过1分钟
  - 皮肤曾变为灰色或蓝色但现已恢复正常颜色
  - 发作后出现异常呼吸
  - 需要进行人工呼吸

**C级（当天就医）**（当天内联系医生/诊所）
  - 呼吸暂停少于1分钟
  - 没有皮肤颜色变化
  - 发作后婴儿正常呼吸
  - 不需要进行人工呼吸

**D级居家护理**
  - 给予安慰。有些婴儿在快速呼吸几次之后会有不到15秒的停顿。
  - 强调让婴儿仰睡的重要性。
  - 与初级保健医生讨论使用呼吸监测器的问题。
  - 获取心肺复苏培训。快速识别呼吸暂停并及时进行心肺复苏可以成功使婴儿复苏，避免严重问题的发生。

## 婴儿溢奶（STCC/Spitting_Up_Infant.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸困难
  - 面部、嘴唇、指甲或耳垂呈蓝色或灰色
  - 反应迟钝

**B级（立即急诊）**（立即前往急诊室）
  - 持续呕吐
  - 之后有窒息或咳嗽
  - 吐出物中有血液或深绿色胆汁
  - 体温 >100.4°F (38°C) 且年龄 <3 个月
  - 脱水迹象：
  - 尿量减少

**C级（当天就医）**（当天内联系医生/诊所）
  - 持续烦躁
  - 家长担心婴儿体重增长不足

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 体温升高且年龄 >3 个月
  - 婴儿摄入超过1到2口食物后频繁溢奶
  - 体重减轻
  - 便秘加剧
  - 持续溢奶
  - 家长担忧

**D级居家护理**
  - 请记住，喂食后不久非剧烈地吐出少量胃内容物是婴儿的常见情况。婴儿开始坐立后应该会自然好转。
  - 在喂养期间多次拍嗝。
  - 给予小而频繁的喂养，避免过度喂养或喂养过快。
  - 喂食后让婴儿处于直立位置，如放在摇篮或婴儿背带中。避免喂养后紧抱或晃动婴儿。

**回电/复诊标准**
  - 家庭护理措施无效或情况恶化
  - 体重减轻或未能正常增重
  - 便秘加剧
  - 持续溢奶

## 长牙（STCC/Teething.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 药物使用情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 极度烦躁不安、间歇性嗜睡及高烧
  - 持续哭闹超过2小时且与肠绞痛无关
  - 持续高烧
  - 孩子看起来病态

**B级（立即急诊）**（立即前往急诊室）
  - 烦躁不安超过2天
  - 哭闹影响孩子的睡眠
  - 孩子拉扯耳朵

**C级（当天就医）**（当天内联系医生/诊所）
  - 长牙，牙龈红肿，且孩子年龄在4到8个月之间
  - 体温<101°F (38.3°C)
  - 食欲不振

**D级居家护理**
  - 用手指按摩肿胀或发炎的牙龈2分钟。
  - 给予对乙酰氨基酚缓解不适。不要给儿童服用阿司匹林。如果年龄<20岁，避免使用类似阿司匹林的产品。如果有肝病，避免使用对乙酰氨基酚。如果有肾病、胃部问题或怀孕的情况下，避免使用布洛芬。请按照标签上的指示
  - 提供冷的牙胶圈、调味冰块、冷冻香蕉或冷冻贝果供婴儿咀嚼。
  - 避免食用咸或酸的食物。

**回电/复诊标准**
  - 极度烦躁不安、间歇性嗜睡及高烧
  - 持续哭闹超过2小时
  - 持续高烧
  - 婴儿停止进食

## Few-shot 示例

**用户问题**：我的新生儿15天，今天哭了很久，肚子看起来有点硬，不喝奶。

**正确回答**：

【严重度判定】
**B级（立即急诊）** → 请立即前往儿科急诊。
15天新生儿异常哭闹伴腹部发硬和拒奶，新生儿病情变化快，须立即评估。

【关键追问】
- 最后一次排便是什么时候？大便颜色正常吗？
- 有没有呕吐？

【家庭护理 / 立刻行动】
- 立即带孩子就医，不要等待或观察
- 记录哭闹开始时间、持续多久、喂奶情况

【回电 / 复诊标准】
就诊后如新生儿再次出现异常哭闹、拒奶或体温异常，立即复诊。
