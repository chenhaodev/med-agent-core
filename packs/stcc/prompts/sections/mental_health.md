# mental_health 分诊知识片段

本片段整合了 STCC 指南中 **mental_health** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 自杀未遂、威胁（STCC/Suicide_Attempt_Threat.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 住址 / 电话号码

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 无意识
  - 严重的呼吸困难、胸痛或腹痛
  - 自杀未遂，如身体伤害或药物过量
  - 正在进行中的自杀企图
  - 威胁要伤害自己或他人
  - 有具体的自杀计划（可用的方法如武器或药片）

**B级（立即急诊）**（立即前往急诊室）
  - 拒绝继续交谈且被认为是高自杀风险
  - 有先前自杀未遂的历史
  - 有自杀念头但无计划或伤害
  - 最近药物变化

**D级居家护理**
  - 所有涉及自杀念头的电话都应转介给适当的精神健康专业人士。
  - 如果来电者独自一人且风险高，尽可能长时间保持电话联系或在不中断连接的情况下将电话转移到自杀热线。必要时追踪电话。获取来电者的地址和电话号码。
  - 所有表示有自杀念头的电话都应被视为紧急情况，直到由精神健康专业人士或急诊医生确认无误为止。
  - 注意背景噪音。

## 抑郁症（STCC/Depression.md）

**关键追问**：姓名 / 年龄 / 起病时间 / 诱因 / 最近的相关事件 / 既往自杀未遂情况及最后一次尝试的日期

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 有自杀念头，并且有实施计划和手段
  - 意图伤害他人
  - 有自杀念头并且已经自伤
  - 服药过量
  - 精神状态改变：混乱，妄想，精神病性症状

**B级（立即急诊）**（立即前往急诊室）
  - 有自杀念头但没有实施计划或手段
  - 新发的妄想想法
  - 因抑郁症曾住院治疗
  - 青少年行为叛逆、挑衅或冒险行为
  - 抑郁新发作，最近改变了药物或新增了药物

**C级（当天就医）**（当天内联系医生/诊所）
  - 有既往自杀尝试史
  - 抑郁症影响工作能力或功能
  - 食欲下降且吃得少
  - 突然停止服用非处方药、处方药、酒精或咖啡因
  - 药物或酒精滥用

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 难以集中注意力
  - 睡眠困难
  - 对性活动的兴趣减少或勃起功能障碍
  - 月经不规律或缺失
  - 对活动缺乏兴趣
  - 人际关系的变化

**D级居家护理**
  - 如果正在接受咨询，给咨询师打电话预约。
  - 联系当地危机干预 ___________________________________
  - 社区咨询服务 ______________________________
  - 家庭服务 ___________________________________________

**回电/复诊标准**
  - 有自杀念头但没有实施计划或手段
  - 抑郁症影响日常活动
  - 持续无法入睡

## 酒精问题（STCC/Alcohol_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 饮酒习惯（量和频率） / 上次饮酒后的时间（小时/天数） / 服用药物情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 新发听觉（声音、嗡嗡声、咔嗒声）、感觉（虫子爬行）或视觉幻觉或妄想
  - 呕血或咖啡色样呕吐物
  - 停酒24至48小时后，出现戒断症状如心跳加速或不规则、出汗、呼吸困难、颤抖或震颤
  - 极度焦虑、恐惧感、激动或偏执
  - 意识状态改变(AMS)
  - 呼吸暂停或呼吸困难

**B级（立即急诊）**（立即前往急诊室）
  - 过去戒断时有抽搐或震颤谵妄(DTs)的病史
  - 欲伤害自己或他人
  - 新发黑色或带血的大便
  - 急性焦虑
  - 感知扭曲
  - 持续呕吐超过24小时，且家庭措施无效

**C级（当天就医）**（当天内联系医生/诊所）
  - 胃部不适、腹泻、胃灼热或睡眠困难
  - 近期突然停止饮酒
  - 请求帮助以停止饮酒

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 恶心、呕吐或腹泻
  - 总体感觉不适
  - 停酒12小时后，出现轻微震颤或焦虑、食欲不振、恶心或呕吐、无力或身体疼痛

**D级居家护理**
  - 确保饮酒者安全，不允许其驾驶或从事可能导致跌倒或受伤的高风险行为。
  - 增加非酒精饮料的摄入量，直到尿液呈淡黄色（这可能是适当水分补充的标志，可能需要长达2天的时间）。
  - 增加水果、蔬菜、土豆、米饭、谷物、全麦食品、鸡蛋、肉类、家禽和乳制品的摄入量。
  - 根据需要服用抗酸药以缓解消化不良。遵循标签上的说明。

## 儿童虐待（STCC/Child_Abuse.md）

**关键追问**：姓名 / 地址和电话号码 / 年龄 / 发生时间 / 原因 / 虐待史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 通话时正在发生虐待行为
  - 严重伤害
  - 无反应

**B级（立即急诊）**（立即前往急诊室）
  - 不同愈合阶段的瘀伤或出现在不寻常部位，黄色瘀伤>18小时
  - （如香烟或热水造成的）烧伤
  - 四肢肿胀、触痛或变形
  - 疑似骨折或脱臼
  - 多处擦伤
  - 面部或身体上的咬痕、撞击或掌掴痕迹

**C级（当天就医）**（当天内联系医生/诊所）
  - 疑似儿童虐待但没有明显的体征或症状
  - 请求与医疗专业人员讨论涉嫌虐待的情况

**D级居家护理**
  - 记住必须为儿童提供一个安全的环境。
  - 将父母或看护者转介给当地资源：儿童保护服务；社会服务机构；县或州卫生部门；地区住房避难所；当地警察；儿童虐待登记热线；家庭服务；咨询服务；或危机热线。
  - 遵循报告涉嫌虐待或忽视的州法律要求。
  - 在1小时内对父母或看护者进行回访。

## 家庭暴力（STCC/Domestic_Abuse.md）

**关键追问**：姓名 / 年龄 / 药物或酒精问题 / 起始时间 / 原因 / 虐待史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 在通话时有人正在遭受虐待或受到威胁
  - 受害者严重受伤或无反应
  - 大量出血
  - 呼吸困难

**B级（立即急诊）**（立即前往急诊室）
  - 有淤青，包括手印或指纹痕迹
  - 香烟、热水、摩擦或化学烧伤
  - 肢体肿胀、触痛或变形
  - 疑似骨折或脱位
  - 其他打击造成的伤害或割伤
  - 人咬或抓痕

**C级（当天就医）**（当天内联系医生/诊所）
  - 虐待发生在2天前以上且伤势不严重
  - 来电者请求帮助或转介

**D级居家护理**
  - 需要问的重要问题：
  - 你现在是否在一个安全的环境中？
  - 你一个人吗？
  - 是否有处于危险中的儿童？

## 老年人虐待（STCC/Elder_Abuse.md）

**关键追问**：姓名 / 地址和电话号码 / 年龄 / 初次出现时间 / 原因 / 虐待史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 打电话时正在发生的虐待
  - 严重伤害
  - 无反应

**B级（立即急诊）**（立即前往急诊室）
  - 处于不同愈合阶段或出现在不寻常区域的瘀伤，黄色瘀伤超过18小时
  - 烧伤（如香烟或热水造成的）
  - 四肢肿胀、触痛或变形
  - 疑似骨折或脱位
  - 多处擦伤
  - 面部或身体上的咬痕、压痕或掌掴痕迹

**C级（当天就医）**（当天内联系医生/诊所）
  - 疑似存在但无身体症状的虐待
  - 显著突然体重下降
  - 无法解释的瘀伤
  - 对家人或看护者的突然恐惧
  - 请求与医疗专业人员讨论疑似虐待的情况

**D级居家护理**
  - 记住，必须为该人提供一个安全的环境。
  - 对于患有晚期痴呆症或某些类型阿尔茨海默病的人，如果存在被害妄想和攻击性行为，可能难以发现虐待。注意观察其基线行为的变化。此人是否更加内向、营养不良、行为改变、出现褥疮、蓬头垢面、不明原因的伤口或疤痕？
  - 记住，服用抗凝药物的人容易瘀伤。询问瘀伤的时间和方式。检查其他部位是否有瘀伤。
  - 如果你怀疑或发现虐待迹象，请私下与该人交谈以了解发生了什么，并采取措施报告并防止再次发生。

## 临终关怀问题（STCC/Hospice_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 终末期状况 / 确认临终关怀患者身份 / 药物使用情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 严重的无法控制的疼痛，即使采取了止痛措施也没有缓解
  - 在家意外死亡且没有不复苏指令（DNR）

**B级（立即急诊）**（立即前往急诊室）
  - 呼吸困难、意识水平下降、尿量减少或无尿，家庭希望撤销家庭死亡计划
  - 没有DNR指令，且家属希望在家照顾临终患者或不确定如何行动
  - 无法服用药物
  - 新出现的不安、烦躁或混乱
  - 发热和感染症状
  - 最近停用抗生素后发热

**C级（当天就医）**（当天内联系医生/诊所）
  - 护理者焦虑且难以应对，或无法维持疼痛控制
  - 计划的死亡和DNR指令已到位
  - 昏睡且可能用药过量

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 对进食缺乏兴趣
  - 嘴唇开裂，口腔干燥
  - 高热、昏迷和呼吸变化

**D级居家护理**
  - 按照处方给予止痛药和抗焦虑药物，确保最大程度的舒适。检查泵以确保其正常工作。
  - 联系宗教服务或精神社区领导者。
  - 记住，在临终过程中失去对食物的兴趣是正常的。
  - 记住，脱水有助于人在接近死亡时感到更加舒适。

**回电/复诊标准**
  - 无法服用药物
  - 新出现的不安、烦躁或混乱
  - 发热和感染症状
  - 最近停用抗生素后发热

## 失眠（STCC/Insomnia.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 过敏史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 自杀念头

**B级（立即急诊）**（立即前往急诊室）
  - 持续性的疼痛、瘙痒、咳嗽或发热影响睡眠且对家庭护理措施无效
  - 持续性抑郁、焦虑或压力

**C级（当天就医）**（当天内联系医生/诊所）
  - 持续超过7天的睡眠困难
  - 连续无法入睡超过2到3小时
  - 请求开安眠药
  - 尿频或大便问题频繁打断睡眠
  - 问题影响工作、学习或其他日常活动
  - 摄入咖啡因产品

**D级居家护理**
  - 增加日常运动。避免在睡前2至3小时内进行剧烈运动。睡前10分钟做轻柔的伸展运动。
  - 睡前阅读与工作无关的材料或听舒缓的音乐。
  - 睡前11小时避免摄入咖啡因和其他兴奋剂。
  - 睡前2小时洗个热水澡或淋浴。

**回电/复诊标准**
  - 家庭护理措施无效或情况恶化
  - 问题影响工作、学习或其他日常活动
  - 感到无法承受

## Few-shot 示例

**用户问题**：我朋友说他最近很绝望，觉得活着没意思，我很担心他。

**正确回答**：

【严重度判定】
**B级（立即寻求帮助）** → 请陪同或帮助其立即联系心理危机热线或精神科急诊。
表达绝望感和"活着没意思"是自杀风险信号，须认真对待，立即介入。

【关键追问】
- 他有没有说过想结束生命的具体计划？
- 手边有没有可能用于自伤的物品？

【家庭护理 / 立刻行动】
- 不要独自离开他，保持陪伴
- 认真倾听，不要评判或反驳
- 帮助拨打心理危机热线：北京 010-82951332 / 全国 400-161-9995
- 如有即时危险，拨打120

【回电 / 复诊标准】
危机过后应帮助其尽快预约心理科门诊，持续关注情绪变化。

参考：STCC/Suicide_Attempt_Threat.md
本建议仅供参考，最终请遵医嘱。如有疑虑请直接拨打120。
