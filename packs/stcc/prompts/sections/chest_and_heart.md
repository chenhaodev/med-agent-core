# chest_and_heart 分诊知识片段

本片段整合了 STCC 指南中 **chest_and_heart** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 胸痛（STCC/Chest_Pain.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 既往心肌梗死史 / 冠状动脉疾病史 / 糖尿病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 持续或间歇性疼痛、紧绷感、压迫感或不适伴随有：
  - 呼吸短促
  - 头晕或无力
  - 皮肤湿冷
  - 恶心或呕吐
  - 颈部、肩部、下颚、背部或手臂的疼痛

**B级（立即急诊）**（立即前往急诊室）
  - 近期有创伤，且疼痛随活动加剧。将热敷袋放在受影响区域，每天4次，每次20分钟。
  - 咳嗽持续或加重，采取上述措施后无改善。

**C级（当天就医）**（当天内联系医生/诊所）
  - 没有明显的创伤或其他原因，且疼痛在休息时减轻但在活动时加剧。服用硝酸甘油（按医生指示）；如果3-5分钟后仍无缓解，再服用一次，并有人陪同前往急诊室或呼叫救护车。

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 疼痛与已知的创伤或其他原因相符，且采取上述措施后有所改善。继续观察症状并在需要时重复使用热敷或止痛药。

## 心率问题（STCC/Heart_Rate_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 心率 / 药物

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 胸部、颈部、下颌或手臂疼痛或不适
  - 呼吸困难
  - 皮肤冷湿或热而干燥
  - 脸部或嘴唇发蓝、灰暗或非常苍白

**B级（立即急诊）**（立即前往急诊室）
  - 持续快速心率>150次/分钟超过30分钟
  - 头晕目眩或晕厥
  - 持续快速心率和有甲状腺疾病或心脏病史
  - 植入体内除颤器反复电击
  - 缓慢的心率并极度疲劳或频繁出现缓慢的心率
  - 持续的慢速心率和间歇>3秒（数1,001，1,002，1,003）

**C级（当天就医）**（当天内联系医生/诊所）
  - 频繁出现快速心率
  - 持续缓慢的心率并有心脏病史、全身不适或频繁跌倒
  - 最近有持续呕吐或腹泻的病史

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 之前有过治疗快速心率的历史
  - 最近服用利尿剂、减肥药、去充血剂、感冒药物、β受体阻滞剂、甲状腺药物、新药物或娱乐性毒品
  - 有使用支气管扩张剂的历史，且有新的处方或剂量增加
  - 过量摄入咖啡因、烟草、酒精或草本兴奋剂
  - 睡眠困难或持续疲劳
  - 压力增大

**D级居家护理**
  - 减慢心率：
  - 深呼吸；屏住呼吸同时捏紧鼻子。轻轻尝试通过鼻子呼气
  - 深呼吸，像排便一样用力
  - 试图吹起一个气球

**回电/复诊标准**
  - 问题持续或恶化
  - 头晕目眩

## 心力衰竭（STCC/Congestive_Heart_Failure.md）

**关键追问**：姓名 / 年龄 / 心力衰竭的发病时间 / 是否正在接受心力衰竭治疗 / 服用药物情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 突发严重呼吸困难
  - 胸痛或压迫感
  - 精神状态改变
  - 唇部、舌头或指甲发绀（蓝紫色）
  - 面色苍白或灰暗
  - 因呼吸困难无法说出>2-3个词的句子

**B级（立即急诊）**（立即前往急诊室）
  - 焦虑加剧
  - 平躺或活动时呼吸困难逐渐加重
  - 劳力性呼吸困难
  - 说话断句
  - 有发热和咳嗽的上呼吸道感染
  - 1至4天内体重突然增加3到4磅

**C级（当天就医）**（当天内联系医生/诊所）
  - 睡眠困难
  - 踝部水肿加重
  - 疲劳或无力加剧
  - 频繁干咳
  - 1天内体重增加>3磅
  - 喘息加剧

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 充血、打喷嚏、咳嗽但无发热
  - 看护者或个人担心，但无其他症状
  - 踝部间歇性肿胀

**D级居家护理**
  - 按处方服药。
  - 与主治医生和实验室检测保持定期约见。
  - 每日早餐前排便后测量体重。记录下来，并在就诊时携带。
  - 遵循医护人员指导的低盐饮食。

**回电/复诊标准**
  - 呼吸困难、喘息、咳嗽或疲劳加重
  - 因呼吸问题或尿量增加而睡眠困难
  - 1天内体重增加>3磅
  - 踝部肿胀加剧

## 高血压（STCC/Hypertension.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 已知高血压情况 / 最后一次升高的血压读数 / 服用的药物

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 除高血压外还有胸痛

**B级（立即急诊）**（立即前往急诊室）
  - 有主动脉或腹主动脉夹层病史
  - 剧烈头痛、视力模糊、恶心或呕吐
  - 嗜睡或意识混乱
  - 手脚持续麻木和刺痛感
  - 咳血或带血的痰
  - 呼吸困难

**C级（当天就医）**（当天内联系医生/诊所）
  - 开始新的降压药后出现间歇性眩晕期
  - 偶发鼻出血
  - 血压升高且最近改变了降压药物
  - 血压升高且服用非处方感冒药

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 持续血压读数 >140/90 mm Hg
  - 近期压力增大
  - 酒精消耗量增加
  - 血压升高且漏服药物

**D级居家护理**
  - 如果血压 >140/90 mm Hg，每周测量一次。
  - 避免吸烟和饮酒。
  - 均衡饮食，低脂肪、低盐。如体重超标，开始减肥计划。停止使用咖啡因、尼古丁和甘草。
  - 定期锻炼。从散步开始。

**回电/复诊标准**
  - 开始新的降压药后出现间歇性眩晕、便秘、阳痿或舌肿胀期
  - 正在接受血压控制治疗且持续升高血压 >160/100 mm Hg
  - 血压 >140/90 mm Hg 持续超过3个月

## 休克，疑似（STCC/Shock_Suspected.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 心跳微弱且快速
  - 呼吸急促而浅表
  - 焦虑不安
  - 昏厥、头晕、无力、精神状态改变、无反应

**D级居家护理**
  - 清理气道，并在必要时提供心肺复苏。
  - 让患者平躺，头部垫枕头，双腿抬高约12英寸（30厘米）。
  - 如果怀疑头部、颈部、背部或胸部受伤，则保持腿部平直并支撑颈部。
  - 通过直接按压出血部位来控制任何出血。

## 胸部创伤（STCC/Chest_Trauma.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药情况 / 疼痛评分

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 严重呼吸困难
  - 意识改变
  - 嘴唇或面部发绀、非常苍白或灰暗
  - 吸气时胸伤口有气泡
  - 胸部或胸骨上方的剧烈疼痛
  - 异物刺入胸壁

**B级（立即急诊）**（立即前往急诊室）
  - 移动或呼吸时疼痛加剧
  - 呼吸困难加重
  - 咳血或粉红色泡沫痰

**C级（当天就医）**（当天内联系医生/诊所）
  - 持续性疼痛 >48 小时
  - 绿、黄或棕色痰 >48 小时
  - 24小时后出现头晕

**D级居家护理**
  - 对于吸吮性胸伤口，用塑料薄膜或几层胶带覆盖伤口以防止空气进入。在患者呼气时密封。
  - 不要移除刺入胸部的异物。
  - 服用常规止痛药（阿司匹林、对乙酰氨基酚或布洛芬）。不要给儿童服用阿司匹林。如果年龄 <20 岁，避免使用类似阿司匹林的产品。如果有肝病，避免使用对乙酰氨基酚。如果有肾病、胃部问题或怀孕的情况，避免使用
  - 如果移动时疼痛加剧，可用枕头支撑疼痛的肋骨。

**回电/复诊标准**
  - 呼吸困难加重、发热、疼痛或头晕
  - 绿、黄或棕色痰

## 低血压（STCC/Hypotension.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 最近的收缩压读数 / 用药情况 / 病史

**B级（立即急诊）**（立即前往急诊室）
  - 冷、苍白、湿润的皮肤
  - 嗜睡或意识模糊
  - 发热和快速脉搏
  - 肩部或腹部疼痛
  - 最近受伤、跌倒或受到钝力冲击
  - 快速脉搏>120 bpm

**C级（当天就医）**（当天内联系医生/诊所）
  - 坐起或从坐位起身时感觉要昏倒
  - 持续出血

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 开始使用新的降压药物后出现头晕期
  - 在服用降压药物或利尿剂期间，低血压和症状持续存在
  - 持续的收缩压<90 mm Hg且无其他症状
  - 来电者对低血压感到担忧但没有其他症状

**D级居家护理**
  - 改变姿势时动作要缓慢，如从坐位到站起。
  - 按照医生的处方服用药物。
  - 节食或减肥过程中避免突然显著减轻体重；在减肥中使用利尿剂时也要注意这一点。
  - 如果正在服用降压药，应避免使用含有抗组胺和去充血成分的感冒药。在服用这类药物前咨询主治医师（PCP）。

**回电/复诊标准**
  - 开始使用新的降压药物后出现头晕期
  - 在接受医疗护理以控制血压的情况下，持续的收缩压<90 mm Hg
  - 反复发生昏厥

## Few-shot 示例

**用户问题**：我胸口疼了20分钟，有点出冷汗，今年60岁。

**正确回答**：

【严重度判定】
**A级（立即呼救）** → 请立即拨打120，不要自行驾车就医。
60岁，胸痛持续超过15分钟伴出冷汗，高度怀疑心脏急性事件，须立即急救。

【关键追问】
- 疼痛是否放射到左臂、下颌或背部？
- 是否有呼吸困难或恶心？

【家庭护理 / 立刻行动】
- 立即停止活动，坐下或半卧位休息
- 拨打120，告知症状和地址
- 若无阿司匹林过敏，可咀嚼一片成人阿司匹林（300mg）
- 不要进食或饮水，等待救护车

【回电 / 复诊标准】
等待急救期间如意识改变或无法呼吸，保持电话畅通由调度员指导。
