# toxin_exposure 分诊知识片段

本片段整合了 STCC 指南中 **toxin_exposure** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 毒物中毒，疑似（STCC/Poisoning_Suspected.md）

**关键追问**：姓名 / 年龄 / 起病时间 / 原因 / 既往史 / 相关症状

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 意识状态改变或无反应
  - 严重呼吸困难或呼吸频率<10次/分钟
  - 自杀企图
  - 惊厥发作
  - 意识水平变化

**B级（立即急诊）**（立即前往急诊室）
  - 大量出汗或唾液分泌过多
  - 喘息或呼吸困难
  - 嘴唇、口腔或指甲床发绀

**C级（当天就医）**（当天内联系医生/诊所）
  - 恶心、呕吐、腹泻
  - 唇部、舌头或皮肤烧伤
  - 头痛、烦躁或发热
  - 有精神疾病史

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 儿童周围有敞开的溢出容器、药瓶，或面部、皮肤、衣物上有物质
  - 呼吸或衣物上可闻到产品气味
  - 怀疑摄入了阿司匹林、对乙酰氨基酚或其他非处方药物、致幻剂或未知蘑菇

**D级居家护理**
  - 按照毒物控制中心的指示行事。向中心提供以下信息：
  - 确认摄入的物质。从容器标签上读取准确名称。
  - 描述摄入量或容器中缺失的数量。
  - 指出物质被摄入的时间。

## 涉嫌食物中毒（STCC/Food_Poisoning_Suspected.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 伴随症状 / 病史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 精神状态改变
  - 呼吸困难、吞咽困难或视力模糊
  - 呕吐带血
  - 便中带血

**B级（立即急诊）**（立即前往急诊室）
  - 疼痛严重、加剧或持续>4小时
  - 恶心和呕吐超过12小时且对家庭治疗或非处方药无反应
  - 腹泻超过48小时且对家庭治疗或非处方药无反应
  - 便中带血
  - 脱水迹象：
  - 尿量减少

**C级（当天就医）**（当天内联系医生/诊所）
  - 食用未冷藏的肉类、禽类、鱼类、蛋黄酱或鸡蛋制品后出现恶心、呕吐、腹泻或腹痛
  - 同餐的其他人也感到不适

**D级居家护理**
  - 频繁（每5到10分钟）小口喝清液（苹果汁、茶、肉汤、运动饮料、透明汽水、姜汁汽水），直到恶心和呕吐缓解。在呕吐缓解后的头24小时内，根据耐受情况逐渐增加量。
  - 腹泻时避免食用乳制品。
  - 呕吐和腹泻缓解后48小时内避免食用辛辣食物、酒精、乳制品和咖啡。
  - 避免服用阿司匹林。

**回电/复诊标准**
  - 症状持续>48小时或恶化
  - 粪便或呕吐物中带血（不仅仅是条纹或斑点）
  - 体温>101°F（38.3°C）

## 血液/体液暴露（STCC/BloodBody_Fluid_Exposure.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 来源 / 风险

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 与已知感染 HIV 或乙型肝炎或感染风险高的人接触，并且发生了以下任何一种情况：
  - 无保护性接触
  - 开放性伤口（溃疡、割伤、擦伤）、眼睛或口腔接触到感染者的血液或体液
  - 被污染的针头刺伤

**B级（立即急诊）**（立即前往急诊室）
  - 与 HIV 或乙型肝炎状态未知的人接触，并且发生了以下任何一种情况：
  - 被污染的针头刺伤
  - 人咬伤
  - 开放性伤口（溃疡、割伤、擦伤）、眼睛或口腔接触到感染者的血液或体液

**C级（当天就医）**（当天内联系医生/诊所）
  - 与 HIV 或乙型肝炎状态未知的人接触，并且发生了以下任何一种情况：
  - 无保护性接触
  - 怀疑被药物迷奸
  - 受伤，且未接种破伤风疫苗、免疫状态不明或最后一次破伤风疫苗接种超过 5 年

**D级居家护理**
  - 如果存在伤口，可用碘伏（贝塔迪恩）或肥皂和水清洗，并观察感染迹象：红肿加剧、疼痛、流脓、发热、温暖感或有条纹。
  - 用流动水冲洗暴露的眼睛或口腔 5 分钟。
  - 咨询初级保健医生或卫生部门（电话号码 ____________），以进行：
  - HIV 检测（自己和接触者）。

## 异物吞咽（STCC/Foreign_Body_Swallowing_of.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 异物 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 过多唾液、流口水或干呕
  - 吞咽困难
  - 咳嗽、窒息或呼吸困难
  - 自杀企图

**B级（立即急诊）**（立即前往急诊室）
  - 喉咙或胸部疼痛不适
  - 异物为电池或尖锐物体

**C级（当天就医）**（当天内联系医生/诊所）
  - 金属物体
  - 大于五分硬币的物体

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 木头或塑料物体
  - 暗淡的玻璃物体（如瓶子、杯子的一部分）
  - 小于一分钱硬币大小的物体
  - 已知吞下某种物质但无症状

**D级居家护理**
  - 如果没有症状，试着喝一口水。如果没有困难，可以尝试吃面包或软食。
  - 暗淡的玻璃物体，如瓶子、杯子或戒指的一部分，通常在3到4天内会随粪便排出而无困难。
  - 不要给予泻药。增加饮食中的纤维（水果、蔬菜、全谷物）以帮助自然排泄。
  - 检查粪便中的异物。

**回电/复诊标准**
  - 间歇性呛咳或干呕
  - 7天内未见异物随粪便排出的证据

## HIV 暴露（STCC/HIV_Exposure.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 开放性伤口（如溃疡、割伤、擦伤）、眼睛或口腔接触到了感染者的血液或体液
  - 被污染的针头刺伤
  - 无保护的性接触

**B级（立即急诊）**（立即前往急诊室）
  - 持续发热和夜间盗汗
  - 持续腹泻
  - 颈部、腋下或腹股沟淋巴结持续肿大
  - 异常溃疡及溃疡数量增加

**C级（当天就医）**（当天内联系医生/诊所）
  - 无保护的性接触
  - 开放性伤口（如溃疡、割伤或擦伤）、眼睛或口腔接触到另一个人的血液或体液
  - 体重下降
  - 手脚麻木或疼痛

**D级居家护理**
  - 如果有伤口，用肥皂和水清洗，并注意感染的迹象：红肿加剧、疼痛、流脓、发热、温热或从伤口发出红色条纹。
  - 用流动水冲洗暴露的眼睛或口腔5分钟。
  - 咨询初级保健医生或卫生部门（电话号码：___________）进行自我及接触者的 HIV 检测。
  - 了解非亲密接触不会传播疾病。

**回电/复诊标准**
  - 知道与 HIV 感染者、甲型、乙型或丙型肝炎患者或性传播疾病患者的血液或体液接触
  - 持续发热和夜间盗汗
  - 持续腹泻
  - 颈部、腋下或腹股沟淋巴结持续肿大

## 药物过量（STCC/Overdose.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 药物或物质的名称 / 剂量 / 既往史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 严重呼吸困难或呼吸频率<10次/分钟
  - 自杀企图
  - 抽搐活动
  - 酒精和药物复合过量
  - 17岁以下受害者中毒
  - 使用可卡因/裂纹并伴有胸痛

**B级（立即急诊）**（立即前往急诊室）
  - 喘息或呼吸短促
  - 严重腹痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 恶心、呕吐或腹泻
  - 服用阿司匹林、对乙酰氨基酚或其他非处方药、致幻剂或未知蘑菇
  - 受害儿童在打开的药物容器附近，如药瓶，或者面部、皮肤或衣物上有物质残留
  - 呼吸或衣物上有疑似摄入产品的气味

**D级居家护理**
  - 按照毒物控制中心的指示行事。（电话：1-800-222-1222）。向中心提供以下信息：
  - 确定摄入的物质。从容器标签上读取准确名称，包括剂量，或描述药片或胶囊上的标记。
  - 说明摄入了多少量或容器中缺少多少。
  - 指出物质被摄入的时间。

## Few-shot 示例

**用户问题**：我2岁的孩子刚喝了一口洗碗液，现在在哭，没有呕吐。

**正确回答**：

【严重度判定】
**B级（立即急诊）** → 请立即前往急诊室并携带洗碗液包装。
幼儿误服化学品，需立即医学评估，部分洗涤剂可造成食道化学灼伤。

【家庭护理 / 立刻行动】
- 不要催吐（可能加重食道损伤）
- 可给孩子喝少量水或牛奶（约120ml）稀释
- 携带洗碗液包装前往急诊，方便医生判断成分

【回电 / 复诊标准】
若出现呕吐、流涎过多、哭闹加剧或呼吸困难，立即拨打120。

【依据】
参考：STCC/Poisoning_Suspected.md
本建议仅供参考，最终请遵医嘱。如有疑虑请直接拨打120。
