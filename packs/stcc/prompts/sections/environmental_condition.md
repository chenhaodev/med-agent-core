# environmental_condition 分诊知识片段

本片段整合了 STCC 指南中 **environmental_condition** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 日晒伤（STCC/Sunburn.md）

**关键追问**：姓名 / 年龄 / 起病时间 / 过敏史 / 用药情况 / 破伤风免疫状态

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 皮肤凉、头晕、昏厥
  - 当体温 >105°F (40.6°C) 时，干燥、发热的皮肤及昏厥
  - 意识混乱
  - 尿量显著减少
  - 失去意识或精神状态改变

**B级（立即急诊）**（立即前往急诊室）
  - 视力变化
  - 严重肿胀
  - 手部或生殖器出现水泡
  - 指端或肢体周围有烧伤圈
  - 关节处的烧伤

**C级（当天就医）**（当天内联系医生/诊所）
  - 多个水泡
  - 开放性水泡
  - 破伤风免疫接种 >10 年前或无免疫史
  - 感染迹象：疼痛、肿胀、脓液，或从水疱区域延伸的红色条纹
  - 眼睛疼痛、视力下降、对光敏感

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 皮肤发红且无水泡
  - 非处方止痛药无效的疼痛
  - 疼痛 >48 小时
  - 少数几个水泡

**D级居家护理**
  - 将冷敷布贴于烧伤处，或每天 4 次用凉水浸泡 10 分钟。可在水中加入 Aveeno 或半杯小苏打。注意不要让该区域变得麻木；可能会导致冻伤。不要直接在皮肤上放置冰块。
  - 预计不适感将持续长达 48 小时。使用您常用的止痛药（阿司匹林、对乙酰氨基酚、布洛芬）。不要给儿童服用阿司匹林。如果年龄 <20 岁，避免使用类似阿司匹林的产品。如果有肝病史，请勿使用对乙酰氨基酚。如
  - 不要在烧伤区域涂抹油腻物质或牙膏。
  - 用水冷却后，如果没有开放性水泡，可将局部抗生素、芦荟胶，或将苯那君糖浆和氧化镁乳液等量混合物涂抹于烧伤处以达到舒缓效果。

**回电/复诊标准**
  - 感染迹象
  - 严重肿胀
  - 没有改善或病情恶化

## 热暴露问题（STCC/Heat_Exposure_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 体温 / 药物

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 体温 >102°F (38.9°C)，皮肤灼热干燥，并且有以下任何一种情况：
  - 皮肤潮红
  - 不出汗
  - 突然头晕、头痛、虚弱、昏厥或失去意识
  - 混乱、谵妄或方向感丧失
  - 视觉障碍

**B级（立即急诊）**（立即前往急诊室）
  - 大量出汗
  - 头晕、虚弱或昏厥
  - 严重肌肉痉挛或不协调
  - 心跳缓慢
  - 尿色深黄或橙色
  - 呕吐和无法耐受液体

**C级（当天就医）**（当天内联系医生/诊所）
  - 体温 >100.5°F (38.1°C) 且年龄 >60 岁、糖尿病患者、卧床不起或免疫力低下者

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 轻微肌肉痉挛
  - 皮肤潮红
  - 轻度恶心或头晕
  - 感觉发热

**D级居家护理**
  - 帮助受害者躺下在阴凉的地方，松开衣服或脱掉衣物。
  - 在等待医疗援助时保持受害者冷静。使用冷毛巾并用风扇帮助降温。不要直接将冰放在皮肤上。不要让受害者发抖，因为这会增加体温。用温水擦拭受害者。不要使用酒精擦洗。
  - 抬高双腿至高于心脏的位置。
  - 如果意识清醒，给受害者喝冷水（水、运动饮料、软饮料、果汁）。不要饮用含酒精或咖啡因的饮料。过快饮水会导致恶心。每15分钟喝一杯，如能耐受且无恶心感可继续。

## 床虫接触或疑虑（STCC/Bedbug_Exposure_or_Concerns.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 过敏史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 严重瘙痒、水泡或荨麻疹
  - 持续性皮疹和瘙痒干扰睡眠
  - 治疗1周后皮疹仍不消退
  - 疮口扩散或出现感染迹象
  - 皮疹消失后再复发
  - 发热、疲倦或淋巴结肿大

**B级（立即急诊）**（立即前往急诊室）
  - 红色或棕色斑点，中心较暗或红
  - 皮肤上成线或集群的多个斑点
  - 脸部、颈部、手臂或手部突然出现扁平或隆起的斑点
  - 正在接受床虫治疗，并对药物使用或防止床虫接触传播给他人有疑问
  - 已知或疑似暴露于床虫侵扰中

**D级居家护理**
  - 如果怀疑被床虫咬伤：
  - 在夜间检查昆虫，因为它们最活跃。检查墙壁裂缝、床垫和弹簧底座之间的空间以及带衬垫的家具和床头板后面。
  - 寻找深色斑点（床虫排泄物）、浅棕色空壳（脱皮前变为成虫）以及床上用品上的血迹（被咬后留下）。

## 咬伤，动物/人类（STCC/Bites_AnimalHuman.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 咬伤部位 / 破伤风状态

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸困难

**B级（立即急诊）**（立即前往急诊室）
  - 直接压迫无法控制出血
  - 受伤肢体畸形或无法使用
  - 头部、面部、颈部或手部撕裂伤
  - 有血友病史

**C级（当天就医）**（当天内联系医生/诊所）
  - 动物未接种狂犬疫苗
  - 动物不可用于观察
  - 手臂、腿部或躯干的撕裂伤
  - 感染迹象：红肿、疼痛、发红条纹从伤口蔓延、分泌物或脓液
  - 猫咬伤

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 无破伤风免疫接种，免疫状态未知，或最后一次破伤风疫苗接种超过5年前
  - 有糖尿病史或免疫力低下
  - 小撕裂伤/擦伤/刺伤

**D级居家护理**
  - 每天用肥皂布清洁受伤区域，然后用水彻底冲洗。
  - 按照标签说明，每日3次涂抹常规使用的抗生素软膏（例如：Mycitracin Triple Antibiotic [bacitracin, neomycin, and polymyxin B]、Neos
  - 用干净干燥的敷料覆盖伤口2天，之后让伤口暴露于空气中，除非有血液渗出。
  - 如果敷料粘在伤口上，请用水冲洗。

**回电/复诊标准**
  - 感染迹象：疼痛加剧、红肿、发热、从伤口蔓延的红色条纹或分泌物
  - 24至48小时内没有改善
  - 感觉伤口内有异物

## 虫咬（STCC/Bites_Insect.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 虫类类型 / 过敏史 / 被咬部位

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 胸部紧绷或呼吸困难或吞咽困难

**B级（立即急诊）**（立即前往急诊室）
  - 对同种昆虫有过严重过敏反应的历史
  - 被棕色隐士蜘蛛或黑寡妇蜘蛛咬伤
  - 精神状态改变
  - 在被咬后突然出现出汗和皮肤苍白
  - 舌头、喉咙或嘴唇肿胀
  - 嘴内蜜蜂蜇伤

**C级（当天就医）**（当天内联系医生/诊所）
  - 被蜇部位以外的荨麻疹、皮疹、瘙痒或肿胀突然出现
  - 肌肉僵硬、腹痛和坐立不安
  - 恶心、呕吐或腹部痉挛
  - 多处被蜇
  - 有引流液、发热、红线或者除了红肿外还有脓包
  - 剧烈疼痛

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 头痛、寒战、发热
  - 在被咬后大约48小时出现的感染迹象：加剧的疼痛、红肿、温热感、红线、引流或发热

## 海洋动物咬伤（STCC/Bites_Marine_Animal.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 动物种类 / 过敏史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 头晕、眩晕或混乱
  - 视力改变
  - 胸闷、呼吸困难或吞咽困难、嘴唇或舌头肿胀
  - 有被同种动物咬伤或蜇伤后出现严重全身过敏反应的历史
  - 咬伤或蜇伤后突然出现出汗和皮肤苍白
  - 心跳快/不规则

**B级（立即急诊）**（立即前往急诊室）
  - 突发荨麻疹
  - 被僧帽水母或𫚉鱼蜇伤
  - 整只手臂或腿部肿胀
  - 剧烈疼痛影响活动能力
  - 伤口脏重且破伤风免疫状态超过5年

**C级（当天就医）**（当天内联系医生/诊所）
  - 蜇伤部位的疼痛、红肿
  - 刺留在体内未取出
  - 无破伤风疫苗接种记录，免疫状态未知，或最后一次破伤风疫苗接种超过5年

**D级居家护理**
  - 对于以下海洋脊椎动物刺伤，将受伤部位浸入热水中30至90分钟以缓解疼痛：
  - 鲶鱼、圆锥蜗牛、鲉鱼、海胆、鲨鱼、𫚉鱼、某些海星、石头鱼和锯齿鱼。
  - 僧帽水母和水母：不要刮除刺或使用冰块。用醋冲洗刺，然后用镊子拔出。可以使用专用止痛喷雾。热敷有助于减少损伤。切勿用海水清洗。
  - 蚂蟥：应用盐、酒精或醋去除。

**回电/复诊标准**
  - 感染迹象，包括红肿加重、扩散，疼痛加剧，肿胀，渗出物，或局部温度升高
  - 症状没有改善或恶化

## 蛇咬（STCC/Bites_Snake.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 蛇的种类 / 被咬部位 / 先前治疗

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 有毒蛇咬伤：如响尾蛇、铜头蛇、水蝮蛇或珊瑚蛇
  - 胸闷或呼吸、吞咽困难
  - 被咬后出现紫癜、发热、口周麻木和刺痛、皮肤苍白或出汗

**B级（立即急诊）**（立即前往急诊室）
  - 无法识别的蛇造成的穿刺伤或獠牙痕迹
  - 有对蛇咬反应的历史
  - 精神状态变化

**C级（当天就医）**（当天内联系医生/诊所）
  - 突然出现荨麻疹、皮疹、瘙痒或非咬伤部位的肿胀
  - 非毒蛇造成的多处咬伤
  - 伤口感染迹象：红肿、渗出物、发热、红线或温热感
  - 伤口周围剧烈疼痛和肿胀
  - 未接种破伤风疫苗，免疫状态不明或最后一次破伤风疫苗接种超过5年

**D级居家护理**
  - 如果在偏远地区且无法获得医疗救助，请按照紧急蛇咬急救包中的指示操作。
  - 保持冷静，不要跑动。
  - 如有可能识别蛇的种类。
  - 抬高和固定受影响部位，并立即寻求紧急医疗援助。

## 蜱虫叮咬（STCC/Bites_Tick.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 已知的蜱虫类型 / 叮咬部位 / 过敏史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 叮咬部位以外突然出现荨麻疹、皮疹、瘙痒或肿胀
  - 胸闷或呼吸困难或吞咽困难

**B级（立即急诊）**（立即前往急诊室）
  - 蜱虫叮咬后2至14天内，广泛性皮疹和流感样症状，如发热、寒战、咽喉痛或头痛
  - 感染迹象，如红肿、疼痛、伤口分泌物或局部发热
  - 有对蜱虫叮咬过敏的历史

**C级（当天就医）**（当天内联系医生/诊所）
  - 不愿意或无法移除蜱虫，且蜱虫头部嵌入皮肤中
  - 无破伤风免疫接种记录、免疫状态不明或最后一次破伤风疫苗接种已超过5年
  - 蜱虫叮咬后14天以上出现皮疹或流感样症状，如发热、寒战、咽喉痛或头痛
  - 在蜱虫叮咬处周围出现靶心状皮疹

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 没有尝试移除蜱虫

**D级居家护理**
  - 使用镊子，施加稳定的向上拉力直到蜱虫松开（尽量靠近皮肤抓住其头部）。如果无法使用镊子，则可以用手指、塑料包装纸或纸以相同方式操作，或者在蜱虫颚部系一根线并牢固地向上拉动。
  - 小型蜱虫可用刀刃或信用卡刮除。
  - 尝试避免在移除过程中或之后压碎蜱虫。压碎会增加疾病传播的风险。
  - 如果只移除了蜱虫身体而头部仍留在皮肤中，使用无菌针头将其取出。

## Few-shot 示例

**用户问题**：我在户外工作了几个小时，现在头晕、恶心、皮肤很热但不出汗。

**正确回答**：

【严重度判定】
**A级（立即呼救）** → 请立即拨打120。
皮肤热但不出汗、头晕恶心，符合热射病（重度中暑）特征，是医疗急症。

【家庭护理 / 立刻行动】
- 立即转移至阴凉处或有空调的室内
- 脱去多余衣物
- 用湿毛巾擦拭颈部、腋窝和腹股沟快速降温
- 不要给意识不清的人喂水
- 等待120

【回电 / 复诊标准】
等待过程中若意识改变或抽搐，保持侧卧位（防止窒息），告知120调度员。
