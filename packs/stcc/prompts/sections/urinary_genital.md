# urinary_genital 分诊知识片段

本片段整合了 STCC 指南中 **urinary_genital** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 尿痛（STCC/Urination_Painful.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 过敏史（如怀孕超过20周 / 请参阅孕期尿路问题(491)） / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 无法排尿
  - 尿中大量血迹
  - 腰背或腹部疼痛且超过8小时无法排尿

**B级（立即急诊）**（立即前往急诊室）
  - 最近进行过泌尿系统手术，出现意外的疼痛或发热
  - 生殖器或腹部近期受伤
  - 使用抗凝药物且尿液呈粉红色或红色
  - 排尿时极度不适
  - 腰背痛
  - 体温>102°F (39°C)

**C级（当天就医）**（当天内联系医生/诊所）
  - 阴茎分泌物
  - 云状或恶臭的尿液（<2岁）
  - 儿童呕吐或液体摄入不足
  - 急迫感或频率超过3天
  - 膀胱部位疼痛
  - 云状或恶臭的尿液

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 排尿结束时疼痛加剧
  - 反复出现泌尿系统感染
  - 性接触后尿痛
  - 频繁泡浴或生殖器区域有肥皂残留（尤其是在年幼的女孩中）
  - 急迫感或频率小于3天

**D级居家护理**
  - 多喝水，尤其是蔓越莓汁。
  - 避免摄入咖啡因和酒精。
  - 性交前后排尿。
  - 从前向后擦拭会阴部区域。

## 阴道出血（STCC/Vaginal_Bleeding.md）

**关键追问**：姓名 / 年龄 / 发病时间（如果怀孕 / 请参见怀孕阴道出血(493)） / 浸透的卫生巾或棉条数量及大小 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 近期遭受强奸或创伤
  - 怀孕超过20周且突然出血
  - 1小时内浸透一个全尺寸卫生巾并伴有虚弱感
  - 意识水平下降
  - 皮肤苍白湿润
  - 突然出现鲜红色血液

**B级（立即急诊）**（立即前往急诊室）
  - 盆腔疼痛伴出血（不同于常规月经不适）
  - 异常大量出血
  - 绝经后突然出现出血或点滴出血（未服用安宫黄体酮[Provera]）
  - 坐起时感到头晕或轻度头昏
  - 近期流产且出血增加、疼痛加剧或发热

**C级（当天就医）**（当天内联系医生/诊所）
  - 怀孕并伴有出血

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 少量出血：
  - 性交后
  - 服用避孕药前三个月内
  - 周期中期漏服避孕药
  - 绝经后使用安宫黄体酮
  - 出血持续>10天

**D级居家护理**
  - 请记住，在排卵期、服用避孕药前三个月内、服用安宫黄体酮期间，或是情绪危机时出现突破性出血并不罕见。
  - 频繁更换棉条。
  - 如果使用棉条时出现发热、寒战或肌肉疼痛，请停止使用，并在症状持续或加重时通知家庭医生。
  - 如怀疑流产，请勿冲洗马桶；保留组织和血块供医生检查。

## 遗尿（STCC/Bed-Wetting.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 频率 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 体温 >101°F (38.3°C)
  - 排尿时疼痛或灼热感
  - 尿急或尿频
  - 腹痛或背痛
  - 尿液中有血或脓
  - 恶心或呕吐

**B级（立即急诊）**（立即前往急诊室）
  - 遗尿变得更加频繁
  - 儿童之前已经连续数月或数年没有尿床
  - 3岁以上儿童有白天膀胱控制问题
  - 3岁以上儿童在遗尿时弄脏内裤
  - 家族中有遗尿史

**D级居家护理**
  - 鼓励白天多喝水，但在睡前2小时内限制饮水。
  - 不要惩罚孩子或强迫孩子夜间使用尿布。使用防水内裤和厚垫子或塑料覆盖物来保护床垫。
  - 口头表扬孩子或使用“干爽夜晚”日历或图表奖励孩子。

**回电/复诊标准**
  - 遗尿变得更加频繁或严重
  - 6岁以上的儿童在遵循家庭护理指导1个月后仍无改善
  - 体温 >101°F (38.3°C)
  - 尿急、尿频，或排尿时疼痛/灼热感

## 乳房问题（STCC/Breast_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药情况 / 病史（如果正在哺乳

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 产后期间出现寒战、发热和头痛
  - 近期乳房受到创伤或穿刺，并伴有撕裂伤或感染迹象
  - 产后期间肌肉疼痛、发热以及乳房上的红肿痛区
  - 乳头有恶臭分泌物
  - 剧烈疼痛
  - 假体乳房的穿孔和渗漏

**B级（立即急诊）**（立即前往急诊室）
  - 有乳房红肿热痛的历史
  - 皮肤溃疡
  - 血性分泌物
  - 非妊娠女性的乳头分泌物
  - 乳房突然出现烧灼、刺痛或剧痛
  - 新纹身并伤口处疼痛，寒战，感觉不适或头痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 近期乳房受到创伤，并伴有疼痛、肿胀或瘀伤
  - 乳房有肿块但无其他症状
  - 肿块与月经前期无关
  - 男性乳房肿块
  - 女性最近一次乳腺检查超过1年前
  - 肿块在月经前一周出现并在经期后消失

**D级居家护理**
  - 注意感染迹象：发热；乳头或裂口处有分泌物；红、热、痛区域或红色条纹。在受影响的区域使用湿热敷10至15分钟，每天4次，直到症状消失。
  - 如果是创伤相关，请在最初的24到48小时内每2到4小时使用冰袋，之后则改为每次10至15分钟，每天4次的热敷。不要直接将冰块或热水袋放在皮肤上。应在冰袋或热源与皮肤之间放置一层布。
  - 保持乳头清洁干燥。
  - 避免穿戴刺激乳头的衣物。

**回电/复诊标准**
  - 创伤后48小时内肿块未消退
  - 状况持续或恶化
  - 感染迹象
  - 血性、绿色、棕色或黄色分泌物

## 紧急避孕 (EC)（STCC/Contraception_Emergency_EC.md）

**关键追问**：姓名 / 年龄 / 发生时间（自上次无保护性交以来的小时数） / 原因 / 正在使用的药物 / 避孕史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 最近服用了紧急避孕药并出现过敏反应迹象（呼吸困难或吞咽困难，喉咙或舌头突然肿胀，无法说话，或胸痛）

**B级（立即急诊）**（立即前往急诊室）
  - 阴道或肛门撕裂或出血
  - 请求进行性侵犯检查和证据收集
  - 受害者是未成年人

**C级（当天就医）**（当天内联系医生/诊所）
  - 发生了性侵犯或被迫发生性行为，且请求不收集证据的医疗检查
  - 无保护性交发生在120小时内（5天），请求预防性传播疾病和怀孕
  - 请求使用铜宫内节育器作为紧急避孕措施；无保护性交发生在过去5天内

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 对紧急避孕药有疑问或担忧
  - 无保护性行为超过120小时，且对怀孕有顾虑
  - 已服用紧急避孕药，并存在副作用的担忧；或出现恶心/呕吐、腹痛、头晕、疲劳、头痛、月经变化、乳房触痛

**D级居家护理**
  - 提供安慰和一般信息：
  - 紧急避孕药仅用于应急，不应作为长期避孕手段。
  - 紧急避孕药不能预防性传播疾病。
  - 紧急避孕药在第一次72小时内最有效，但可在无保护性行为后120小时内使用。（紧急避孕产品“Next Choice”需在72小时内服用[电话866-9WATSON]。“Plan B”需在120小时内服用

**回电/复诊标准**
  - 持续超过24小时的恶心/呕吐、腹痛、头晕、疲劳或乳房触痛
  - 伴随或不伴随发热的阴道出血
  - 突然大量出汗
  - 意识水平下降

## 阴道异物（STCC/Foreign_Body_Vagina.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 异物 / 过敏史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 阴道内嵌有锐利物体
  - 大量出血
  - 剧烈疼痛
  - 性行为侵犯
  - 突然发作：皮疹、发热、手或脚脱皮、全身不适感、呕吐或腹泻

**B级（立即急诊）**（立即前往急诊室）
  - 多次尝试后仍无法取出异物
  - 水肿和触痛
  - 异味分泌物
  - 阴道内放置的卫生棉条超过24小时且无法取出
  - 无法取出避孕或娱乐设备

**D级居家护理**
  - 不要尝试移除阴道内的锐利物体或已经断裂的物体。
  - 按照常规服用止痛药（阿司匹林、对乙酰氨基酚、布洛芬）。不要给儿童服用阿司匹林。10岁以下避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病或胃部问题存在，或怀孕时避免使用布洛芬。请按照标签
  - 注意感染的迹象：疼痛加剧、分泌物增多、发热或肿胀。

**回电/复诊标准**
  - 多次尝试后仍无法取出异物
  - 水肿和触痛持续或加重
  - 异味排泄物或发热
  - 皮疹、发热、手或脚脱皮、全身不适感、呕吐或腹泻

## 生殖器病变（STCC/Genital_Lesions.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 过敏史 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 剧烈疼痛
  - 感染迹象：疼痛加剧、红肿、流脓、发热、温暖感

**B级（立即急诊）**（立即前往急诊室）
  - 开放性疮口
  - 新出现的散在簇状水泡
  - 发烧和全身不适
  - 腹股沟淋巴结肿大
  - 阴道或阴茎分泌物或出血
  - 剧烈瘙痒或烧灼感

**C级（当天就医）**（当天内联系医生/诊所）
  - 确诊患有生殖器疱疹、尖锐湿疣或有STD暴露史并要求治疗
  - 排尿或排便时疼痛
  - 生殖器或肛门区域出现持续超过24小时的无痛性皮疹或硬结

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 发痒的红色皮疹
  - 近期剧烈活动后出汗的历史
  - 大腿内侧、腹股沟或阴囊出现红斑、鳞屑和瘙痒
  - 凸起的红色、触痛或白色/硬结
  - 持续时间小于24小时的无痛性皮疹或肿物

**D级居家护理**
  - 用温水泡澡。
  - 避免使用泡沫浴、刺激性强的或有香味的肥皂、带香味的卫生纸或个人清洁用品。
  - 在症状消退前避免性活动。
  - 保持患处干净干燥。

**回电/复诊标准**
  - 感染迹象：疼痛、红肿、流脓、温暖感、红色条纹或腹股沟淋巴结肿大
  - 2天后无改善或病情恶化
  - 疼痛或肿胀加剧
  - 出现分泌物或发热

## 男性生殖器问题（STCC/Genital_Problems_Male.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 使用冰袋冷敷30分钟后仍然持续勃起疼痛
  - 剧烈疼痛或肿胀
  - 阴茎受到创伤伴有变形或出血
  - 阴茎内有异物

**B级（立即急诊）**（立即前往急诊室）
  - 无法将包皮拉回到阴茎头部
  - 无法排尿
  - 排尿时疼痛
  - 腰部疼痛
  - 排尿后腹股沟疼痛且体温>100°F（37.8°C）
  - 阴囊或睾丸肿胀或疼痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 持续的阴茎肿胀、硬块或溃疡
  - 已知或怀疑接触过性传播疾病
  - 疼痛性皮疹或溃疡
  - 超过24小时的阴茎分泌物
  - 阴茎头部红肿
  - 尿中带血

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 性交时或性交后疼痛
  - 勃起困难或维持勃起困难，且有糖尿病病史或正在服用抗抑郁药、抗焦虑药物、降压药或利尿剂
  - 丧失性欲
  - 阴茎被拉链夹住
  - 婴儿阴茎肿胀和割伤
  - 超过24小时的无痛皮疹或生长物

**D级居家护理**
  - 如果性交时疼痛，可以考虑使用非处方润滑剂（如K-Y润滑剂）。不要使用凡士林。
  - 如果性交后阴茎头部疼痛，探索可能的原因，比如对避孕霜或避孕套的过敏，并改用替代方法。
  - 如果因摩擦尿布导致阴茎头红肿，在给婴儿换尿布时将阴茎向下压。
  - 如果客户有性功能障碍问题且正在服用处方药、患有糖尿病或情绪问题，应与家庭医生讨论。

## Few-shot 示例

**用户问题**：我排尿时有灼烧感，频繁想上厕所，小便量很少，没有发烧。

**正确回答**：

【严重度判定】
**C级（当天内就医）**
尿频、尿痛是典型泌尿道感染症状，当天内应就医确认并治疗，防止上行感染。

【关键追问】
- 有无腰背痛或发烧？（提示肾脏受累）

【家庭护理 / 立刻行动】
- 多饮水，促进细菌排出
- 避免憋尿
- 不要自行购买抗生素服用，需要尿液检测确认感染类型

【回电 / 复诊标准】
出现以下情况需急诊：
- 高烧（体温>38.5°C）
- 腰部或背部剧烈疼痛
- 出现寒战或恶心呕吐

参考：STCC/Urination_Painful.md
本建议仅供参考，最终请遵医嘱。如有疑虑请直接拨打120。
