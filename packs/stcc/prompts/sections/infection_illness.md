# infection_illness 分诊知识片段

本片段整合了 STCC 指南中 **infection_illness** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 流感（STCC/Influenza.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 症状 / 已知接触史或社区疫情 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 精神状态改变
  - 因非鼻塞原因导致的呼吸困难
  - 体温>104.9°F (40.5°C)
  - 面部或躯干出现紫斑或暗红色斑点，颈部僵硬或疼痛
  - 严重头痛
  - 皮肤或嘴唇发绀

**B级（立即急诊）**（立即前往急诊室）
  - 颈部僵硬或疼痛
  - 体温>103.1°F (39.5°C)
  - 发热，且儿童或老年人显得非常病态、昏睡或极度烦躁
  - 脱水迹象

**C级（当天就医）**（当天内联系医生/诊所）
  - 已知接触史，并伴有以下任何症状：疲劳、体温<103.1°F (39.5°C)、干咳、咽痛、胃肠道症状、流鼻涕或鼻塞、肌肉酸痛
  - 有心力衰竭病史、免疫抑制、6周至23个月大的婴儿、年龄>65岁、怀孕、长期护理机构居民、哮喘、慢性阻塞性肺疾病（COPD）、代谢障碍

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 轻微症状
  - 无症状，但父母或个人表示担忧

**D级居家护理**
  - 经常用肥皂和水洗手，或使用酒精基手部消毒剂。
  - 强调流感具有高度传染性。保持良好的呼吸道礼仪；咳嗽或打喷嚏时用纸巾遮住口鼻。
  - 避免与生病的人接触。
  - 如果感到不适，避免与其他人接触。咳嗽和打喷嚏时，在与人近距离接触时佩戴外科口罩以防止飞沫传播。如果口罩变脏或湿润，请更换。

**回电/复诊标准**
  - 颈部僵硬或疼痛
  - 体温>103.1°F (39.5°C)
  - 发热，且儿童或老年人显得非常病态、昏睡或极度烦躁
  - 脱水迹象

## 水痘（STCC/Chickenpox.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 水痘接触史 / 暴露日期 / 怀孕状况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 难以唤醒
  - 意识模糊或谵妄
  - 呼吸困难

**B级（立即急诊）**（立即前往急诊室）
  - 严重头痛
  - 呼吸急促
  - 颈部僵硬
  - 红痛的皮肤区域或红色条纹
  - 面部剧烈疼痛或肿胀
  - 1个月以下婴儿且有皮损

**C级（当天就医）**（当天内联系医生/诊所）
  - 咳嗽或发热对家庭护理措施无反应
  - 呕吐超过3到4次
  - 干燥的棕色结痂变软和金色，或排出脓液
  - 发热持续 >4天
  - 严重瘙痒对家庭护理措施无反应
  - 没有水痘病史或免疫接种且怀孕

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 发热和全身疲倦
  - 皮疹从红斑发展到水泡
  - 肿胀和触痛的淋巴结

**D级居家护理**
  - 增加液体摄入量。
  - 每3至4小时洗一次凉浴，并服用常规药物（对乙酰氨基酚）退热。请遵循标签上的说明，不要服用阿司匹林。
  - 为了控制瘙痒，可以尝试非处方药（如口服的苯海拉明、Caladryl、外用的Cortaid或Cortizone），并遵循标签上的指示，或者洗燕麦浴（Aveeno）。
  - 患者应留在家中，直到所有皮损干燥。

**回电/复诊标准**
  - 感染迹象：红肿加剧、疼痛、分泌物、脓液、红色条纹
  - 瘙痒影响睡眠
  - 持续发热 >4天
  - 咳嗽或呼吸困难

## 禽流感（“禽流”）暴露（STCC/Avian_Influenza_“Bird_Flu”_Exposure.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 过去10天内是否前往疑似地点（已知的家禽或人类中存在禽流感感染） / 最近是否有接触过家禽或家养鸟类（如家禽农场 / 养有家禽的家庭或鸟市）

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 出汗且伴有头晕或虚弱感
  - 严重呼吸困难
  - 面色苍白或发灰，并有病毒性肺炎病史
  - 意识混乱、谵妄或难以唤醒

**B级（立即急诊）**（立即前往急诊室）
  - 可能接触过受污染的鸟类，包括被分泌物/排泄物污染的表面，并伴有结膜炎和发热、咳嗽、喉咙痛或身体疼痛等症状
  - 不褪色的暗红色或紫色皮疹
  - 照顾已知患有禽流感的人后出现结膜炎
  - 儿童体温 >105°F (40.6°C)
  - 成人体温 >103°F (39.4°C)
  - 有免疫抑制病史、年龄>60岁、镰状细胞贫血、糖尿病或卧床不起且体温 >101°F (38.3°C) 的情况

**C级（当天就医）**（当天内联系医生/诊所）
  - 照顾已知患有禽流感的人并伴有类似流感的症状或结膜炎
  - 喉咙痛持续超过2天或发热 > 2 天
  - 发热且有哮喘、癌症、慢性阻塞性肺病（COPD）、充血性心力衰竭（CHF）、糖尿病、心脏病或肾病史
  - 绿色、棕色或黄色痰液或鼻分泌物 >72 小时
  - 脱水迹象

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 绿色、棕色或黄色痰液或鼻分泌物 <72小时
  - 痰中带血丝
  - 对潜在的禽流感暴露有担忧但无其他症状（或轻度症状）

**D级居家护理**
  - 出现类似流感症状的人应在最后一次发热后24小时内留在家中（除非看医生）。
  - 接触患者及其环境（包括洗衣物）的护理人员应经常用肥皂和水或含酒精的手部消毒剂洗手。
  - 咳嗽和打喷嚏时要遮住口鼻，并将纸巾丢入袋子中。
  - 脏盘子和餐具应用洗碗机清洗或使用温水和肥皂手洗。

**回电/复诊标准**
  - 护理人员出现类似流感的症状
  - 喉咙痛持续超过2天或发热 > 2 天
  - 绿色、棕色或黄色痰液或鼻分泌物 >72 小时
  - 脱水迹象

## 埃博拉：已知或疑似暴露（STCC/Ebola:_Known_or_Suspected_Exposure.md）

**关键追问**：名字 / 年龄 / 与埃博拉患者的接触史 / 前往或居住在已知埃博拉疫情爆发地区的旅行史 / 药物使用史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 没有反应或精神状态改变
  - 呼吸极度困难
  - 发热>100.4°F (38.0°C)，严重头痛、肌肉疼痛、腹痛、呕吐、腹泻、无法解释的异常出血

**B级（立即急诊）**（立即前往急诊室）
  - 无发热>100.4°F (38.0°C)，严重头痛、肌肉疼痛、腹痛、呕吐、腹泻、无法解释的异常出血等症状

**C级（当天就医）**（当天内联系医生/诊所）
  - 无已知接触史，但存在发热>100.4°F (38.0°C)，严重头痛、肌肉疼痛、腹痛、呕吐、腹泻、无法解释的异常出血等症状，并对埃博拉感染感到担忧

**D级居家护理**
  - 如果前往急诊科或拨打911，提前通知提供者以便他们能够准备好适当的防护装备和隔离地点。不要使用公共交通工具。
  - 如果在过去的21天内接触了埃博拉患者，在保持无症状的21天期间避免公共接触。
  - 照顾者或其他附近人员应避免与感染者的体液（包括血液、呕吐物、粪便、精液、唾液、尿液、汗水）接触。
  - 安慰他们，告知埃博拉的主要传播途径是通过体液接触，并非空气传播。作为额外的防护措施，如果埃博拉感染者咳嗽或打喷嚏，请保持6英尺的距离以避免接触到喷出的唾液。经常洗手。

**回电/复诊标准**
  - 如果呼叫者同意给出的建议，记录通话，并鼓励呼叫者在问题恶化时回拨或咨询家庭医生。
  - 如果呼叫者不同意给出的建议，重新评估并建议呼叫者跟进家庭医生、诊所或急诊科。

## 腺体，肿胀或触痛（STCC/Glands_Swollen_or_Tender.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 过敏史 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 肿大的淋巴结大于2英寸
  - 肿大淋巴结附近有红色条纹
  - 淋巴结处发热或发红
  - 剧烈疼痛
  - 淋巴结影响呼吸、吞咽或颈部活动
  - 淋巴结流脓

**B级（立即急诊）**（立即前往急诊室）
  - 肿大的淋巴结1到2英寸大小
  - 持续发热且退热措施无效
  - 腹股沟、后颈或腋窝区域有肿大的淋巴结
  - 肿大淋巴结附近或远端区域的感染迹象
  - 身体其他部位的持续性皮疹、瘙痒或肿胀
  - 持续疲劳

**C级（当天就医）**（当天内联系医生/诊所）
  - 间歇性的淋巴结肿大史
  - 伴有发热、咽喉痛或鼻塞的颈部淋巴结肿大
  - 有过敏史
  - 月经前腋下区域肿胀和触痛

**D级居家护理**
  - 不要挤压肿大的淋巴结。
  - 根据需要服用常规止痛药（阿司匹林、对乙酰氨基酚、布洛芬）以缓解不适或发热。不要给儿童使用阿司匹林。20岁以下者避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病、胃部问题患者或孕妇避免使用
  - 注意感染的迹象。

## 乙型肝炎（STCC/Hepatitis.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 乙型肝炎或其他类型肝炎（A / B或C）的已知或疑似暴露情况 / 用药史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 严重腹痛

**B级（立即急诊）**（立即前往急诊室）
  - 有胃肠道出血迹象
  - 意识混乱或精神状态下降

**C级（当天就医）**（当天内联系医生/诊所）
  - 严重呕吐
  - 容易出现淤青或流血
  - 站立时感到头晕
  - 免疫功能低下
  - 体重增加并伴有呼吸困难

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 眼睛或皮肤发黄
  - 腹部疼痛，尤其是右上腹痛
  - 大便呈粘土色
  - 尿液颜色深
  - 年龄小于10岁或大于70岁
  - 糖尿病

**D级居家护理**
  - 充分休息，保持良好的饮食习惯（尝试一天吃几顿小餐）。
  - 增加水分摄入：果汁、肉汤、运动饮料、水、茶和每天至少8杯（每杯8盎司）的水。
  - 在排便后、吃饭前或准备食物前彻底洗手。
  - 餐具应在洗碗机中清洗，或在200°F (93.3°C) 的烤箱中烘烤15到20分钟，并用最热的温度设置清洗衣物和床上用品。

**回电/复诊标准**
  - 眼白或皮肤发黄且为新出现的情况
  - 腹痛、粘土色大便或深色尿液
  - 因呕吐而出现脱水迹象

## 葡萄球菌性脓疱病（STCC/Impetigo.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 已知的葡萄球菌性脓疱病诊断或接触史 / 有蜜黄色结痂的伤口 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 红色或可乐色尿液
  - 鲜红且疼痛的脸部
  - 侧腹痛
  - 非常快速地扩散
  - 腹部肿胀

**B级（立即急诊）**（立即前往急诊室）
  - 水泡或伤口直径>1英寸（约2.5厘米）
  - 从伤口延伸出的红色条纹
  - 糖尿病、类固醇治疗、癌症、HIV或其他免疫抑制病史

**C级（当天就医）**（当天内联系医生/诊所）
  - 1岁以下儿童
  - 两个或以上的家庭成员有伤口
  - 超过两处伤口
  - 鼻孔、耳道或嘴周的伤口
  - 靠近伤口区域的淋巴结肿大
  - 体温>100°F（37.8°C）或喉咙痛

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 伤口直径<1英寸（约2.5厘米）
  - 小红点变成浑浊的水泡或痘痘，然后成为有黄褐色结痂的伤口

**D级居家护理**
  - 去除结痂时，可将浸泡在漂白水溶液中的布（1杯漂白剂兑入4杯水中）敷于结痂处。轻轻擦拭结痂，不要用力搓洗。
  - 用抗菌肥皂清洗伤口。
  - 每天3次在清洁后的伤口上涂抹抗生素软膏（如Mycitracin、Neosporin、Polysporin）。请遵循标签上的说明。
  - 避免触摸、抓挠或挑破伤口以防止扩散和感染。

**回电/复诊标准**
  - 水泡或伤口直径>1英寸（约2.5厘米）
  - 从伤口延伸出的红色条纹
  - 体温>100°F（37.8°C）或喉咙痛
  - 其他家庭成员也出现了类似症状

## 流行性腮腺炎（STCC/Mumps.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 已知或疑似流行性腮腺炎 / 病史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 颈部僵硬或严重头痛
  - 反复呕吐
  - 剧烈腹痛
  - 意识混乱
  - 意识水平下降
  - 呼吸困难

**B级（立即急诊）**（立即前往急诊室）
  - 腺体肿胀超过8天
  - 发热超过5天
  - 腺体上方皮肤发红
  - 睾丸疼痛
  - 剧烈疼痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 耳前及下颌周围的腺体肿大且触痛
  - 咀嚼时疼痛加剧
  - 未接种过流行性腮腺炎疫苗
  - 16至18天前接触过流行性腮腺炎

**D级居家护理**
  - 隔离流行性腮腺炎患者，直至肿胀消退。
  - 发热时休息。
  - 为了缓解不适，可将冰领或冰袋敷在肿大的腺体上。不要直接将冰放在皮肤上；应在冰和皮肤之间放置一层布料。
  - 服用常规止痛药（阿司匹林、对乙酰氨基酚、布洛芬）。不要给儿童使用阿司匹林。20岁以下者避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病或胃部问题存在，或怀孕的情况下避免使用布洛芬。请按照

**回电/复诊标准**
  - 腺体肿胀超过8天
  - 发热超过5天
  - 腺体上方皮肤发红
  - 睾丸疼痛

## Few-shot 示例

**用户问题**：我发烧38.8度，全身酸痛，头痛，周围有人得流感。

**正确回答**：

【严重度判定】
**D级（居家观察）**
症状符合流感典型表现，成人无高危因素，可先居家护理，但需密切观察。

【家庭护理 / 立刻行动】
- 充分休息，避免外出传染他人
- 多喝水（每天至少2升），退热期出汗多需补充水分
- 可按说明书服用对乙酰氨基酚或布洛芬退热止痛
- 与家人保持距离，戴口罩

【回电 / 复诊标准】
以下情况需就医：
- 体温超过39.5°C或持续超过5天
- 呼吸困难、胸痛
- 精神状态明显变差（极度嗜睡、意识混乱）

【依据】
参考：STCC/Influenza.md
本建议仅供参考，最终请遵医嘱。如有疑虑请直接拨打120。
