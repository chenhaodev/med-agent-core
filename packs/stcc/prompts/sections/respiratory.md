# respiratory 分诊知识片段

本片段整合了 STCC 指南中 **respiratory** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 呼吸问题（STCC/Breathing_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药情况 / 既往史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 嘴唇或舌头发绀
  - 皮肤湿冷
  - 窒息感
  - 泡沫状粉红色或大量白色痰液
  - 意识改变
  - 突然发作的严重呼吸短促

**B级（立即急诊）**（立即前往急诊室）
  - 因剧烈疼痛而难以深呼吸
  - 过去2小时内开始的严重气促、喘息或有声呼吸
  - 近期创伤、手术或分娩史
  - 吸入异物
  - 接触过之前引起显著反应的物质（如蜇伤、药物、植物、化学物质、食物或动物）
  - 断续说话

**C级（当天就医）**（当天内联系医生/诊所）
  - 断续说话
  - 紧绷的咳嗽
  - 休息时有轻微可听的喘息声
  - 呼吸时疼痛加剧
  - 上呼吸道感染且之前因相同症状住院
  - 因咳嗽或呼吸困难无法连续睡眠1-2小时以上

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 体温>101°F (38.3°C)
  - 峰流速为基线的50%至80%
  - 胸部不适持续不缓解或加重
  - 呼吸困难且无法缓解
  - 突然发作的严重症状，如胸痛、意识丧失等

## 哮喘（STCC/Asthma.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 既往哮喘史 / 严重程度 / 峰流速测量值

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 治疗后持续喘息
  - 呼吸困难或气短
  - 无法躺下呼吸；必须坐起来才能呼吸
  - 胸部疼痛或紧绷感
  - 嘴唇、舌头或脸部发绀或发蓝
  - 药物、食物、蜜蜂叮咬或接触已知过敏原或哮喘触发因素后突然出现喘息

**B级（立即急诊）**（立即前往急诊室）
  - 呕吐并无法保留药物
  - 上呼吸道感染症状和既往史：
  - 激素治疗
  - 因相同症状住院的历史
  - 气管插管历史
  - 只能说部分句子

**C级（当天就医）**（当天内联系医生/诊所）
  - 体温>100.5°F（38.1°C）
  - 药物无法缓解的咳嗽
  - 当前药物只能提供最小或暂时的哮喘症状缓解
  - 黄色或绿色痰液
  - 峰流速值>50%至80%的基线水平

**D级居家护理**
  - 增加液体摄入量；饮用清水和透明液体。
  - 早期治疗症状以减少哮喘发作的严重性。按照初级保健医生（PCP）的处方使用预防药物。
  - 使用加湿器、蒸汽浴室或冷湿空气帮助缓解症状。
  - 遵循初级保健医生（PCP）制定的治疗计划或管理计划，正确使用峰流速计、药物和吸入器以最佳控制哮喘。

## 咳嗽（STCC/Cough.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 既往治疗 / 相关症状 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 是 转至 窒息处理方案（130）
  - 否 下行至 B

**B级（立即急诊）**（立即前往急诊室）
  - 突然出现的呼吸急促、快速呼吸或喘息
  - 是 转至 呼吸困难处理方案（101）
  - 否 下行至 C

**C级（当天就医）**（当天内联系医生/诊所）
  - 是 转至 胸痛处理方案（118）
  - 否 下行至 D

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 嘴唇或舌头发绀
  - 窒息感
  - 咳出泡沫状粉红色痰液
  - 呼吸困难且无法说话
  - 吸入烟雾、火焰或气体后的呼吸困难
  - 接触某种之前引起过显著反应的物质（如刺伤、药物、植物、化学品、食物或动物）后突然发作

**D级居家护理**
  - 每天喝 6 到 8 杯水。
  - 温热的雾气可能有助于改善状况。通过覆盖在口鼻上的湿布呼吸，或使用加湿器。
  - 如果没有蒸汽设备，可以用热水洗澡时吸入蒸气。
  - 避免吸烟和二手烟。

**回电/复诊标准**
  - 情况未见好转或恶化
  - 发热持续 >72 小时
  - 出现绿色、棕色或灰色痰液且持续 >72 小时
  - 咯血（不仅仅是条纹或斑点）

## 喘息（STCC/Wheezing.md）

**关键追问**：姓名 / 年龄 / 发作时间 / 用药情况 / 插管史 / 住院史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 严重的呼吸困难
  - 无法说话
  - 胸骨凹陷
  - 吸入异物
  - 嘴唇或面部发绀
  - 严重胸痛

**B级（立即急诊）**（立即前往急诊室）
  - 对药物治疗无反应
  - 对家庭护理措施无反应
  - 需要坐起来才能呼吸
  - 类似以前需要住院或注射的喘息发作
  - 有心力衰竭、心脏病、慢性阻塞性肺疾病、肺栓塞或腿部血栓病史
  - 断断续续地说句子

**C级（当天就医）**（当天内联系医生/诊所）
  - 峰流速值>80%基线值（如果是哮喘患者）
  - 咳出绿色、黄色或铁锈色的痰
  - 婴儿、老人或免疫抑制者

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 首次喘息发作，短时间内可缓解
  - 能完整地说句子

**D级居家护理**
  - 按家庭医生的指导服用药物。
  - 使用冷雾加湿器。
  - 确定原因并避免接触刺激物。

## 慢性阻塞性肺疾病 (COPD)（STCC/Chronic_Obstructive_Pulmonary_Disease_COPD.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 既往 COPD 病史 / 严重程度 / 峰流速测量（如使用）

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 治疗后持续喘息
  - 呼吸困难或气短
  - 胸痛或胸部紧绷感
  - 不能平躺；必须坐起来才能呼吸
  - 嘴唇、舌头或脸部发绀或蓝色
  - 用药后、接触已知过敏原或其他触发因素后突然出现喘息

**B级（立即急诊）**（立即前往急诊室）
  - 呕吐且无法保留药物
  - 上呼吸道感染症状，并有以下病史：
  - 使用皮质类固醇治疗
  - 因相同症状住院的历史
  - 插管历史
  - 峰流速为正常基线的50%到80%，并使用雾化器或吸入器后无改善

**C级（当天就医）**（当天内联系医生/诊所）
  - 发热 >100.5°F (38.1°C)
  - 药物无法缓解的咳嗽
  - 当前药物对 COPD 症状仅有轻微或暂时性缓解
  - 长期咳嗽，痰液呈透明、白色、黄色或绿色
  - 脚踝或腿部肿胀
  - 早晨头痛（由睡眠期间呼吸减少导致氧气不足和二氧化碳增加引起）

**D级居家护理**
  - 停止吸烟，避免接触任何可能的刺激物。
  - 定期使用医生开具的所有药物和治疗方案。
  - 监测症状变化，并根据需要调整药物剂量。
  - 保持适当的活动量，以增强呼吸肌肉的力量和耐力。

## 气道阻塞（STCC/Choking.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 疼痛评分 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 患者有意识但无法说话、咳嗽或呼吸

**B级（立即急诊）**（立即前往急诊室）
  - 患者无意识且不呼吸

**C级（当天就医）**（当天内联系医生/诊所）
  - 呼吸困难
  - 口唇或面部发绀

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 异物吸入肺部
  - 排出异物后咳血或严重疼痛
  - 无法从喉咙中移除异物且无其他症状
  - 可以说话和咳嗽
  - 没有呼吸困难
  - 经常性唾液、食物或液体引起窒息

**D级居家护理**
  - 对于经常性气道阻塞，慢慢吃东西，每次取较小的口。
  - 如果感觉有鱼刺卡在喉咙里，试着用面包和牛奶将其冲下。

**回电/复诊标准**
  - 喉咙中有鱼刺且持续2小时以上的刺激感
  - 感染迹象：持续咽喉痛、发热、流脓
  - 吞咽困难
  - 症状未见好转或病情加重

## 过度换气（STCC/Hyperventilation.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 是 → 参见胸痛协议（123）
  - 否 ↓ 转至 B

**B级（立即急诊）**（立即前往急诊室）
  - 嘴唇或指甲发绀
  - 无法说话
  - 必须坐起来才能呼吸
  - 是 → 参见呼吸问题协议 (106)
  - 否 ↓ 转至 C

**C级（当天就医）**（当天内联系医生/诊所）
  - 严重疼痛
  - 发热且对退烧措施无反应
  - 异常困倦
  - 是 → “在2到4小时内寻求医疗护理”
  - 否 ↓ 转至 D

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 面部、口唇、手指或脚趾周围的麻木和刺痛感
  - 感觉要昏倒
  - 手脚抽搐
  - 焦虑或情绪激动
  - 有类似症状的病史或先例
  - 是 → “如果无改善，请回电或联系家庭医生预约”

**D级居家护理**
  - 坐下并专注于放慢呼吸，每5秒一次呼吸。
  - 用纸袋盖住口鼻，深呼吸10次。
  - 如果没有改善，继续使用袋子呼吸5到15分钟。
  - 不用袋子休息几分钟。

**回电/复诊标准**
  - 无改善或状况恶化
  - 发作期间突然发热
  - 频繁发作

## 吸入异物（STCC/Foreign_Body_Inhaled.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 吸入物体 / 病史 / 用药情况

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 窒息且无法说话、咳嗽或呼吸
  - 失去意识的人没有呼吸

**B级（立即急诊）**（立即前往急诊室）
  - 呼吸困难
  - 嘴唇或脸部变蓝
  - 无法哭泣或说话
  - 自杀企图

**C级（当天就医）**（当天内联系医生/诊所）
  - 异物吸入肺部
  - 咳血或在异物从喉咙移除后感到剧烈疼痛
  - 无法从喉咙中取出异物，但没有其他症状
  - 窒息感
  - 流口水
  - 短语说话

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 断句说话
  - 吸入异物、气溶胶或烟雾后间歇性咳嗽或喘息
  - 能说话和咳嗽
  - 无呼吸困难
  - 频繁因唾液、食物或液体而窒息
  - 完整句子说话

**D级居家护理**
  - 对于经常性窒息，慢慢吃东西，每次吃得少一些。
  - 如果有鱼刺卡在喉咙的感觉，试着用面包和牛奶将鱼刺冲下去。

**回电/复诊标准**
  - 喉咙里有鱼刺或鸡骨，并且持续感到喉咙痛超过2小时
  - 感染迹象：持续的喉咙痛、发热或流脓
  - 吞咽困难
  - 没有改善或情况恶化

## Few-shot 示例

**用户问题**：我儿子8岁，哮喘发作，喘得比平时厉害，吸入药已经用了但没什么效果。

**正确回答**：

【严重度判定】
**B级（立即急诊）** → 请立即前往急诊室。
哮喘急性发作，用药后无缓解，须立即就医避免病情恶化。

【家庭护理 / 立刻行动】
- 保持孩子坐直，不要平躺
- 前往急诊途中让孩子保持冷静
- 记录最后一次用药时间和剂量，告知急诊医生
- 如途中呼吸极度困难（无法说话或嘴唇变蓝），立即拨打120

【回电 / 复诊标准】
此次就诊后，如症状夜间加重或用药效果持续不佳，请次日复诊调整方案。

参考：STCC/Asthma.md
