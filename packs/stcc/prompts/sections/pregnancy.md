# pregnancy 分诊知识片段

本片段整合了 STCC 指南中 **pregnancy** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 妊娠，疑似临产（STCC/Pregnancy_Suspected_Labor.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 孕周 / 妊娠次数

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 即将分娩，胎儿头部已露出
  - 大量阴道出血
  - 抽搐或惊厥
  - 剧烈腹痛
  - 羊膜破裂伴脐带脱垂

**B级（立即急诊）**（立即前往急诊室）
  - 严重头痛
  - 已知胎儿为臀位或横位，在首次出现宫缩或流液时
  - 流出棕色或绿色阴道液体
  - 当前有疱疹发作，并伴有规律且强烈的宫缩
  - 宫缩和孕周<37周
  - 无胎动

**C级（当天就医）**（当天内联系医生/诊所）
  - 第一次怀孕，强烈且规律的宫缩（每5分钟一次，持续45至60秒，持续1到2小时）
  - 非首次怀孕，强烈且规律的宫缩（每5到10分钟一次，持续45至60秒，持续1到2小时）
  - 流出清澈阴道液体
  - 已知有高血压、糖尿病或其他妊娠并发症史
  - 快速分娩史
  - 前往分娩机构路程较长

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 不规律的宫缩
  - 点状出血
  - 背部、脚部或腹股沟疼痛
  - 排尿时有急迫感、频次增加或疼痛

**D级居家护理**
  - 散步、淋浴、按摩、背部使用热敷或冷敷以缓解不适，改变体位或进行骨盆摇摆运动帮助早期分娩。
  - 吃清淡的食物。
  - 监测宫缩：频率 = 从一次宫缩的开始到下一次宫缩的开始（即5分钟）；持续时间 = 宫缩持续的时间（即30至45秒）。
  - 监测胎动；如果1小时内少于10次胎动，请寻求医疗帮助。

## 妊娠，阴道出血（STCC/Pregnancy_Vaginal_Bleeding.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 孕周 / 妊娠次数

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 严重腹痛
  - 胎儿头部即将娩出
  - 胎膜破裂伴有疑似脐带脱垂
  - 意识水平下降
  - 1小时内湿透的卫生巾多于一片并伴有无力感
  - 连续2小时每小时湿透的卫生巾多于一片或12小时内超过6片卫生巾被湿透

**B级（立即急诊）**（立即前往急诊室）
  - 无胎动且妊娠已超过20周
  - 中等程度的阴道出血（需要使用卫生巾但不会完全湿透）
  - 鲜红色血液，沿腿流下
  - 已知前置胎盘或胎盘早剥并伴有出血
  - 近期有外伤史

**C级（当天就医）**（当天内联系医生/诊所）
  - 1小时内少于10次的胎动
  - 强而规律的宫缩
  - 清澈羊水渗漏
  - 卫生巾、内衣、床单或衣物上有少量鲜红色血液

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 有少许暗褐色或粉红色血迹，擦拭时可见或内裤上少量出现

**D级居家护理**
  - 近期性交后或进行过阴道检查后可能出现暗褐色或粉红色血迹。
  - 监测胎动；如果1小时内少于10次的胎动，请寻求医疗援助。
  - 监测宫缩。

**回电/复诊标准**
  - 擦拭时或内裤上出现暗褐色或粉红色斑点
  - 没有改善或情况恶化

## 产后问题（STCC/Postpartum_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 分娩日期 / 用药情况 / 既往史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸困难或呼吸急促
  - 阴道出血需要每小时使用超过一片全尺寸卫生巾并伴有虚弱、头晕
  - 意识改变
  - 皮肤苍白、湿润且发凉
  - 分娩后48小时内下腹部剧烈疼痛
  - 有伤害婴儿或自己的想法

**B级（立即急诊）**（立即前往急诊室）
  - 体温>100°F (37.8°C) 在分娩后4到10天内
  - 伴有发热或腹痛的出血
  - 分娩第一周内出血量增加和腹部痉挛（<1小时内一片卫生巾被鲜红血液浸透）
  - 异常阴道分泌物伴有出血、发热、疼痛，或分泌物有恶臭
  - 呕吐、腹泻、发热或皮疹
  - 小腿肿胀、疼痛或发红

**C级（当天就医）**（当天内联系医生/诊所）
  - 持续抑郁>2到3周
  - 排尿时的急迫感、频尿或疼痛
  - 无发热的异常阴道分泌物
  - 会阴切开术伤口处疼痛增加
  - 剖宫产伤口部位疼痛加剧、肿胀、恶臭引流、红热或温暖

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 乳房充血、触痛、硬结
  - 子宫收缩痛
  - 会阴切开术部位的拉伸或牵拉感

**D级居家护理**
  - 产后头两周平衡活动和休息。
  - 通常在6周检查后（剖宫产为8周）如果得到医生许可可以返回工作。
  - 记住，产后3到4天的子宫收缩是正常的，并且是由哺乳刺激产生的。
  - 排便可能在产后1到2天内不发生。每天喝6到8杯水和果汁并增加水果、蔬菜和麸皮的摄入量。轻泻剂（Metamucil）可能有帮助。请遵循说明书使用。

**回电/复诊标准**
  - 家庭护理措施无效或状况恶化
  - 发热、腹痛或异常阴道分泌物
  - 感染迹象：红肿加剧、疼痛、从伤口延伸出的红线、温暖感、恶臭或浓绿引流，或发热
  - 持续抑郁

## 孕期，感冒症状（STCC/Pregnancy_Cold_Symptoms.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 孕周 / 怀孕次数

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸急促，说话困难或无法讲话
  - 体温 >39.5°C (103°F)
  - 非鼻塞原因引起的呼吸困难

**B级（立即急诊）**（立即前往急诊室）
  - 无胎动

**C级（当天就医）**（当天内联系医生/诊所）
  - 强烈且规律的宫缩
  - 体温 >38.4°C (101°F)
  - 呼吸频率 >每分钟20次
  - 减少的胎动
  - 喘息声，对家庭护理措施无反应
  - 1小时内少于10次胎动

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 绿色或棕色痰液或鼻分泌物
  - 严重咽喉痛
  - 哮喘、糖尿病、心脏病或免疫抑制病史
  - 耳痛或流脓
  - 喉咙痛
  - 体温 <38.4°C (101°F)

**D级居家护理**
  - 每4小时测量体温一次。
  - 增加每日饮水量至6到8杯。
  - 使用温湿气雾化器，并每天更换水。
  - 用盐水漱口；喝热鸡汤或吸吮冰冻硬糖以缓解不适。

**回电/复诊标准**
  - 绿色或棕色痰液或鼻分泌物
  - 严重咽喉痛
  - 体温 >38.4°C (101°F)
  - 情况恶化或家庭护理措施后无改善

## 妊娠，高血压（STCC/Pregnancy_Hypertension.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 病因 / 孕周 / 妊娠次数

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 抽搐活动
  - 无反应
  - 孕20周以上且有：
  - 剧烈腹痛
  - 显著阴道出血
  - 即将分娩，胎儿头部露出

**B级（立即急诊）**（立即前往急诊室）
  - 鲜红色阴道出血
  - 剧烈头痛
  - 无胎动
  - 手、脸、腿或腰部突然肿胀加剧
  - 上腹痛（右上象限）
  - 视觉障碍（视物模糊或双重视觉，眼前有黑点，或看到星点或物体周围光环）

**C级（当天就医）**（当天内联系医生/诊所）
  - 家庭护理措施无法缓解的头痛
  - 1小时内胎动少于10次
  - 规律性强收缩
  - 血压升高（在家监测后休息时）
  - 呕吐且不能摄入液体

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 脚或腿逐渐肿胀
  - 家庭护理措施可缓解的头痛
  - 每周体重增加>2磅

**D级居家护理**
  - 左侧卧休息。
  - 监测胎动；如果1小时内胎动少于10次，寻求医疗帮助。

## 妊娠，阴道流液（STCC/Pregnancy_Leaking_Vaginal_Fluid.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 孕周 / 妊娠次数

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 羊膜破裂伴有脐带脱垂
  - 胎儿头部即将娩出，临产迫在眉睫
  - 明显阴道出血（每小时浸湿1片卫生巾超过2小时或12小时内浸湿6片）

**B级（立即急诊）**（立即前往急诊室）
  - 剧烈腹痛
  - 无胎动
  - 绿色、棕色或红色液体
  - 已知胎儿为臀位或横位
  - 孕期<37周
  - 当前有疱疹爆发，伴有规律且强烈的宫缩或阴道流液

**C级（当天就医）**（当天内联系医生/诊所）
  - 阴道流出清澈液体
  - 强烈、规律的宫缩
  - 发热或寒战
  - 1小时内少于10次胎动

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 丧失黏液栓
  - 不规则宫缩
  - 阴道分泌物增多，呈黏液状
  - 如果出现阴道流液且脐带脱垂，平静地告诉患者：
  - 将电话交给他人拨打救护车
  - 采取膝胸位，头低臀高

**D级居家护理**
  - 请记住，在真正临产前几周丧失黏液栓并不罕见。
  - 监测胎动；如果1小时内少于10次胎动，请就医。
  - 监测宫缩；如果宫缩强烈、持续45至60秒，并且以每5分钟一次的规律模式持续1到2小时，或宫缩强度增加，请前往分娩机构。
  - 排空膀胱。

**回电/复诊标准**
  - 点滴出血
  - 1小时内少于10次胎动
  - 强烈规律宫缩
  - 发热或寒战

## 妊娠、恶心和呕吐（STCC/Pregnancy_Nausea_and_Vomiting.md）

**关键追问**：姓名 / 年龄 / 起始时间 / 原因 / 频率 / 妊娠周数

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸困难
  - 精神状态改变
  - 妊娠超过20周，且存在以下任一症状：
  - 呕吐鲜红色血液或咖啡渣样呕吐物
  - 最近头部受伤
  - 出汗或心悸

**B级（立即急诊）**（立即前往急诊室）
  - 24周以上妊娠且无胎儿活动

**C级（当天就医）**（当天内联系医生/诊所）
  - 体温 >100.4°F (38.9°C)
  - 脱水症状
  - 糖尿病
  - 阵痛或规律的强烈宫缩
  - 超过24小时无法保持水分摄入
  - 尿液呈深琥珀色

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 家庭护理措施无法缓解的恶心和呕吐或胃灼热
  - 有恶心但能保持部分水分或食物摄入
  - 尿液呈黄色
  - 胃灼热

**D级居家护理**
  - 在最后一次呕吐后1小时内不要进食或饮水。
  - 在首次呕吐后的12小时内，每次只喝少量的透明液体。
  - 根据耐受情况逐渐增加水分摄入。
  - 如果12小时内无再次呕吐，尝试吃清淡食物（如饼干、干面包、香蕉）。

## 妊娠问题（STCC/Pregnancy_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 妊娠期 / 孕次 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 即将分娩，胎儿头部露出
  - 严重头痛、视力重影或模糊、方向感丧失、头晕或易怒
  - 二尖瓣区域先前受过创伤、跌倒、腹部受到撞击等
  - 排出大血块
  - 剧烈腹痛
  - 阴道流液伴随脐带脱垂

**B级（立即急诊）**（立即前往急诊室）
  - 无胎动
  - 面部、手部、腿部或腰部突然肿胀
  - 头痛、眼前黑点、头晕、昏厥或呕吐
  - 规律宫缩或阴道流液

**C级（当天就医）**（当天内联系医生/诊所）
  - 已知高血压且血压升高
  - 腿痛、肿胀、发红或负重时疼痛加剧
  - 疼痛、阴道出血或发热
  - 对止痛药无反应的持续头痛
  - 所有食物和液体及/或腹痛持续呕吐
  - 1小时内少于10次胎动

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 急迫的排尿感、疼痛或困难
  - 尿液颜色异常（如深黄、红色或棕色）
  - 无法解释的体重减轻
  - 疲劳或虚弱加剧
  - 阴道出血量增多
  - 持续性头痛、头晕或视力问题

## Few-shot 示例

**用户问题**：我怀孕38周，现在开始有宫缩，每5分钟一次，每次持续约1分钟。

**正确回答**：

【严重度判定】
**B级（立即前往医院）** → 请立即前往产科就诊或分娩。
38周孕妇，宫缩规律（每5分钟一次，持续1分钟），已进入活跃分娩期，须立即就医。

【关键追问】
- 是否有破水（阴道有液体流出）？
- 胎动正常吗？

【家庭护理 / 立刻行动】
- 不要独自驾车，让他人送往医院
- 携带待产包和产检资料
- 保持冷静，记录宫缩间隔和持续时间

【回电 / 复诊标准】
出现以下情况需立即拨打120：
- 破水但无法快速到达医院
- 宫缩突然消失（可能提示胎盘问题）
- 大量阴道出血

【依据】
参考：STCC/Pregnancy_Suspected_Labor.md
本建议仅供参考，最终请遵医嘱。如有疑虑请直接拨打120。
