# ear_eye_nose 分诊知识片段

本片段整合了 STCC 指南中 **ear_eye_nose** 类别的分诊标准。
回答时严格基于以下内容，并遵守输出格式规范。

## 耳痛，耳漏（STCC/Earache_Drainage.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 用药情况 / 疼痛程度评分

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 耳痛、颈强直和发热

**B级（立即急诊）**（立即前往急诊室）
  - 面部一侧肿胀、疼痛和红斑
  - 耳部受到撞击后出现剧烈疼痛、听力丧失、耳后淤青、耳道出血或显著肿胀
  - 对止痛药无反应的严重疼痛
  - 有糖尿病或免疫抑制史
  - 耳朵向外偏移
  - 耳后骨触痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 突然听力下降伴疼痛，耳漏或眩晕
  - 血液性或透明的耳漏（不同于常规耳垢）
  - 肿胀、疼痛、温暖感、耳漏或发热
  - 移动或触碰耳朵时疼痛加剧
  - 水疱或溃疡
  - 发热、鼻塞或咽痛

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 无法通过药物清除耳蜡栓并伴有疼痛或听力下降
  - 耳朵晒伤
  - 冷暴露后疼痛
  - 听力模糊但无痛感
  - 游泳或水接触后的疼痛
  - 突然疼痛伴随破裂声，听力下降和鼻塞

**D级居家护理**
  - 如果疼痛与伤害有关或怀疑鼓膜破裂（突然疼痛、听力下降、出血或流出物、耳鸣、眩晕），不要向耳朵内滴入液体。
  - 对于晒伤的耳朵应用冷敷，对因撞击造成的肿胀区域使用冰袋。
  - 为了清除过多的耳蜡，按照说明使用Debrox（过氧化脲）最长3天，或每天在受影响的耳朵里滴两次矿物油，连续2天。
  - 用温暖的热敷布每次15到20分钟，一天4次，直到症状缓解或等待预约时。将袜子装半满米粒，打结后放入微波炉加热至温热状态，然后将其贴在耳朵上。使用前请先用手腕内侧测试温度。

**回电/复诊标准**
  - 3天内无改善或病情恶化
  - 持续发热且退热措施无效
  - 呕吐、腹泻、疲劳、嗜睡或颈强直
  - 发热或肿胀

## 眼部问题（STCC/Eye_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 药物 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 突然或剧烈疼痛
  - 视力突然丧失、模糊或双重视觉
  - 瞳孔大小突然不等
  - 视野中有一块阴影
  - 眼前飞絮增多
  - 光周围有光环或彩虹

**B级（立即急诊）**（立即前往急诊室）
  - 疼痛、红肿、流泪、有分泌物，佩戴隐形眼镜
  - 发热和红肿的睫毛
  - 施加压力或眼球移动时疼痛加剧
  - 眼球或眼角有病灶
  - 眼睛完全肿胀闭合

**C级（当天就医）**（当天内联系医生/诊所）
  - 家庭护理措施无效，持续疼痛
  - 持续的瘙痒、红肿、灼热感和分泌物
  - 摘除隐形眼镜并冲洗后仍然疼痛
  - 颈部淋巴结肿大，眼睛周围红肿
  - 麦粒肿复发或出血
  - 位于鼻侧下眼睑的麦粒肿对家庭护理措施无反应

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 眼球白色部分出血超过3天
  - 傍晚时分有眼痛病史
  - 长时间佩戴隐形眼镜或清洁不当的病史
  - 泪腺区域肿胀疼痛
  - 上睑或下睑小而红、肿胀和触痛的区域
  - 接触烟雾、气体、烟霾、泳池水、已知过敏原或太阳灯

**D级居家护理**
  - 如果有分泌物，鼓励家人使用单独的毛巾和洗脸巾。眼部感染非常容易传染。
  - 避免揉眼睛或接触眼睛。
  - 用温水浸湿的棉球清洁结痂或分泌物。使用后丢弃棉球。不要用同一个棉球清洁两只眼睛。清洁后洗手。
  - 每天4次，每次15到20分钟地对眼睛进行热敷。

**回电/复诊标准**
  - 48小时后情况未改善或恶化
  - 黄色或绿色分泌物
  - 红肿的眼睑

## 口臭（STCC/Bad_Breath.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 用药情况 / 既往病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 呼吸有水果味或氨味
  - 伴有腹胀和疼痛的恶臭
  - 口腔或喉咙内有发热和溃疡
  - 口腔或舌部剧烈疼痛

**B级（立即急诊）**（立即前往急诊室）
  - 牙龈持续出血或肿胀
  - 咳嗽并伴有恶臭痰液
  - 牙齿松动、缺失或腐烂
  - 频繁使用铸铁炊具或餐具
  - 近期摄入大剂量维生素或矿物质
  - 有胃肠疾病或慢性肺部疾病的病史

**D级居家护理**
  - 使用抗菌漱口水（如李施德林）漱口。
  - 每天刷牙、假牙、牙龈和舌头两次，并定期使用牙线。
  - 每日取下并浸泡桥冠和假牙。
  - 如果认为维生素或矿物质补充剂是导致口臭的原因，减少其剂量。

## 隐形眼镜问题（STCC/Contact_Lens_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 隐形眼镜类型 / 疼痛程度

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 突然失明
  - 剧烈疼痛
  - 穿透性损伤（如硬质镜片嵌入眼球表面）
  - 佩戴未充分清洁和冲洗的隐形眼镜后出现剧烈疼痛

**B级（立即急诊）**（立即前往急诊室）
  - 眼睛内硬质镜片破裂
  - 持续性的疼痛，家庭护理无效
  - 视力模糊或改变

**C级（当天就医）**（当天内联系医生/诊所）
  - 眼部分泌物呈脓状
  - 家庭护理后状况未改善或恶化
  - 无法摘除隐形眼镜
  - 持续感觉有异物在眼中
  - 对光线极度敏感

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 不适感
  - 眼睛红肿但视力无变化
  - 沙涩感
  - 频繁流泪

**D级居家护理**
  - 在进行任何眼部护理前彻底洗手。
  - 摘除硬质隐形眼镜：
  - 手电筒或放大镜有助于操作。
  - 可能需要翻转眼睑。

**回电/复诊标准**
  - 持续的眼睛红肿或刺激感
  - 脓状分泌物或频繁流泪
  - 对光线敏感
  - 状况未改善或恶化

## 耳部伤害，异物（STCC/Ear_Injury_Foreign_Body.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 用药 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 协调能力丧失
  - 面瘫或面部下垂
  - 天旋地转的眩晕

**B级（立即急诊）**（立即前往急诊室）
  - 听力下降
  - 耳道出血
  - 最近头部受到撞击且耳道有清澈或带血的分泌物
  - 无法移除异物或嵌入的穿孔饰物
  - 剧烈疼痛
  - 持续性耳鸣

**C级（当天就医）**（当天内联系医生/诊所）
  - 持续疼痛超过48小时
  - 空气冲击、噪音或头部受到撞击后的耳痛
  - 外耳红肿
  - 最近使用了棉签
  - 小割伤且破伤风疫苗接种已超过10年
  - 家庭护理3天后没有好转

**D级居家护理**
  - 伤后24至48小时内，每天将冰袋敷于耳朵上20分钟，每日四次。不要直接将冰放在皮肤上；在冰和皮肤之间使用毛巾或其他织物作为屏障。
  - 将耳朵朝向光源，并向上拉耳以鼓励昆虫爬向光线方向离开耳朵。
  - 如果无法通过将耳朵向下并向地面轻轻摇晃头部同时向上拉耳来移除异物，请不要尝试强行取出。
  - 为减轻不适，可服用常规止痛药（阿司匹林、对乙酰氨基酚或布洛芬）。不要给儿童使用阿司匹林。年龄小于20岁者避免使用类似阿司匹林的产品。肝病患者避免使用对乙酰氨基酚。肾病、胃部问题存在或怀孕情况下避免使用

## 耳鸣（STCC/Ear_Ringing.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 药物使用情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 过量或频繁服用阿司匹林或含阿司匹林的产品

**B级（立即急诊）**（立即前往急诊室）
  - 耳内有异物
  - 家庭护理措施无效的严重耳痛

**C级（当天就医）**（当天内联系医生/诊所）
  - 近期服用过阿司匹林、含奎宁或链霉素的药物
  - 近期头部受伤或最近接受过耳部手术的历史
  - 头晕或眩晕
  - 恶心或呕吐
  - 持续性耳痛
  - 耳道分泌物

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 上呼吸道感染
  - 过敏引起的充血
  - 单侧或双侧耳朵感到堵塞
  - 耳垢过多积聚
  - 有梅尼埃病类似症状的历史

**D级居家护理**
  - 服用减充血剂以缓解耳朵充血。按照标签上的说明使用。向药剂师咨询产品建议。
  - 如果正在服用含阿司匹林、奎宁或链霉素的药物，停止服用并通知家庭医生。
  - 不要用手指或棉签戳耳朵或试图取出耳垢。这些措施会加重问题。
  - 如果有梅尼埃病的历史，减少盐分、咖啡因和酒精的摄入量。

**回电/复诊标准**
  - 严重的听力损失或眩晕
  - 耳痛或流脓
  - 持续性呕吐、耳鸣加剧、头痛

## 眼部损伤（STCC/Eye_Injury.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 是否佩戴隐形眼镜 / 过敏史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 眼睛或眼睑有割伤或穿透性损伤
  - 眼部受到撞击并突然失明
  - 眼球突出
  - 受损眼睛流出透明胶状物
  - 眼部钝器伤害
  - 瞳孔大小不等

**B级（立即急诊）**（立即前往急诊室）
  - 肿胀、疼痛和流泪超过30分钟
  - 暴露于强光下，如焊接弧光或太阳灯
  - 由热水、化学品、辣椒喷雾或异物造成的伤害，在家庭护理治疗后仍持续疼痛，或者眼白变浑浊
  - 持续模糊视觉或复视

**C级（当天就医）**（当天内联系医生/诊所）
  - 伤害后或移除异物后24小时仍持续不适或刺激感
  - 伤后出现感染迹象：疼痛、肿胀、发红、分泌物或发热

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 眼周区域出现青紫
  - 白眼仁出血超过3天

**D级居家护理**
  - 眼内化学品：立即用冷自来水冲洗眼睛20到30分钟。将头部倾斜至受伤的眼睛下方，在拉开眼睑的同时，让水流过眼球的内部到达外侧。
  - 对于辣椒素暴露（如辣椒、辣椒喷雾），使用整盒牛奶冲洗眼睛。它会像清洁剂一样作用，去除有害物质。
  - 不要揉眼睛。
  - 在最初24小时内应用冰袋或冷敷以减轻肿胀。不要直接将冰块放在皮肤上；在冰和皮肤之间放置一条毛巾或其他布料作为屏障。

## 面部问题（STCC/Facial_Problems.md）

**关键追问**：姓名 / 年龄 / 发病时间 / 原因 / 药物使用情况 / 病史

**A级（立即呼救 → 拨打120）**（立即呼叫救护车（120））
  - 突发视力丧失
  - 面部一侧严重疼痛，眼周模糊视觉和红眼
  - 成人肩部、胸部、颈部或臂部疼痛
  - 青光眼病史
  - 突然出现面部一侧下垂

**B级（立即急诊）**（立即前往急诊室）
  - 突发剧烈疼痛影响活动能力
  - 疼痛、肿胀、发红、发热或有分泌物

**C级（当天就医）**（当天内联系医生/诊所）
  - 午后或弯腰时疼痛加剧
  - 鼻腔流出绿色、棕色或黄色的分泌物
  - 鼻子与下眼睑之间的棱线上疼痛
  - 体温 >101°F (38.3°C)
  - 持续肿胀
  - 面部皮疹、水泡或病变

**D级（居家观察）**（居家观察，按家庭护理指导）
  - 近期面部出现红色水疱状皮疹
  - 面部受伤后的疼痛、肿胀或瘀伤
  - 近期感冒史
  - 食用冰冻食物或饮料后疼痛加剧
  - 鼻腔和眼部有分泌物

**D级居家护理**
  - 交替使用冷热敷于前额和脸颊，每次1分钟，每天4次。袜子内装满在微波炉中加热的米粒效果很好。
  - 增加水分摄入量。
  - 对于面部伤害，在最初24小时内每天4次将冰袋敷于脸上10到20分钟以帮助减少肿胀。不要直接将冰放在皮肤上，应在冰和皮肤之间放置湿布或其他织物屏障。
  - 每天多次坐在蒸汽浴室中20分钟以促进鼻窦排液。

**回电/复诊标准**
  - 持续性疼痛或病情恶化
  - 体温 >101°F (38.3°C)
  - 感染迹象：疼痛、肿胀、发红、发热、分泌物或红线
  - 持续性鼻腔分泌物

## Few-shot 示例

**用户问题**：我右耳痛了两天，有点流脓，听力感觉下降了。

**正确回答**：

【严重度判定】
**C级（当天内就医）**
耳痛伴分泌物提示耳道或中耳感染，当天内应就医检查，避免感染扩散。

【家庭护理 / 立刻行动】
- 不要用棉签掏耳，避免加重损伤
- 不要向耳内滴任何液体
- 侧卧可略微缓解耳压

【回电 / 复诊标准】
立即就医信号：
- 耳后红肿或发热（乳突炎可能）
- 极度剧烈的耳痛
- 突然完全失聪
