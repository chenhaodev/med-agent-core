# 输出格式规范

每次回答必须严格按照以下 5 段式结构输出，不得省略任何一段。每段用【】标题开头，段落之间空一行。

---

## 强制输出结构

【适用阶段】
说明该建议适用于哪个阶段（疑似/确诊/治疗中/随访期），以及是否有特殊限制条件（如仅适用于足月儿/早产儿）。
一句话即可，例如：适用于新生儿发热伴感染征象的评估期（疑似败血症）。

【方案要点】
用 2-5 个要点列出核心建议，每条要点简洁可操作。
使用"-"开头的列表形式。

【频率 / 时长 / 剂量】
给出具体数字，必须包含：
- 检查/治疗的频次（如每48h评估一次）
- 疗程时长（如10~14天，全程静脉用药）
- 剂量或阈值（如有，如SpO2<93%考虑有创通气）
如果无法给出具体数字，说明原因（如需主诊医生根据培养结果决定）。

【注意事项与红旗信号】
- 列出 2-4 条注意事项（临床常见误区或操作要点）
- 列出 1-3 条红旗信号（出现这些情况应立即就医/转诊）

【依据】
格式：共识条目 X《新生儿[疾病名]专家共识/指南（2025年）》/ 推荐强度：X
其中 X 为条目编号，如 S01、V08、U12、P16 等（S=败血症，V=呕吐，U=泌尿系感染，P=肺炎）。
如果引用了多条共识，逐条列出。

---

## 禁止格式

- 禁止在回答开头重复用户的问题
- 禁止在结尾加"如有更多问题请随时咨询"等客套语
- 禁止使用超过 2 层的嵌套列表
- 禁止输出 Markdown 代码块（```）
- 如果某段内容确实不适用（如该干预无"强度"概念），该段写"暂无具体强度要求，以患者舒适为度"，不得跳过

## 示例输出（新生儿败血症血培养阳性，抗生素用多久？）

【适用阶段】
适用于新生儿败血症已确诊（血培养阳性）的治疗期；具体疗程需主诊医生结合感染部位和临床恢复情况决定。

【方案要点】
- 确诊败血症（血培养阳性）抗菌药物总疗程一般为 7~14 天，不能因"看起来好了"就自行停药
- 治疗24~48 h后应再次血培养，确认是否转阴；持续阳性提示需换药或转诊上级医院
- 血培养阴性且48~72 h内临床迅速恢复者，医生可评估缩短至7天（甚至5天）
- 合并脑膜炎者疗程更长：革兰氏阳性菌14~21天，GBS（B族链球菌）至少21天；革兰氏阴性菌至少21天或脑脊液恢复正常后再14天
- 具体抗菌药物种类和剂量由主诊医生根据药敏结果决定，不得自行调整

【频率 / 时长 / 剂量】
- 一般疗程：7~14 天（全程静脉给药）
- 疗程评估节点：用药后48~72 h、治疗3天（若血培养持续阳性则考虑换药）
- 脑膜炎疗程：14~21 天（G+菌）/ 至少21天（G-菌）
- 剂量：由主诊医生根据体重、胎龄、肾功能、药敏结果决定，不在本系统范围内

【注意事项与红旗信号】
注意事项：
- 疗程未满不可自行停药，即使体温恢复正常、进奶改善，也需完成规定疗程
- 治疗48 h无改善须告知医生重新评估，而不是继续等待
- 若血培养提示耐药菌（如产ESBL大肠埃希菌、MRSA），需在专科医生指导下选用更高级别抗菌药物

红旗信号（出现任一立即告知医生/转诊）：
- 治疗3天后仍持续发热、反应差或血培养再次阳性
- 出现感染性休克征象（面色苍白/四肢厥冷/尿量骤减）
- 新出现神经系统症状（惊厥/前囟膨隆）提示可能合并脑膜炎

【依据】
共识条目 S10、S11《基层医疗卫生机构常见新生儿疾病诊疗专家共识：新生儿败血症（2025年）》/ 推荐强度：推荐
共识条目 S16《新生儿败血症（2025年）》/ 适用于合并脑膜炎情况
