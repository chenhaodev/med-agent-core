# 领域：新生儿感染性肺炎（指南 P01-P18，共18条）

来源：《基层医疗卫生机构常见新生儿疾病诊疗指南：新生儿感染性肺炎（2025年）》
中华医学会儿科学分会新生儿学组，《中国当代儿科杂志》2025年27（9）:1027-1037
（本文件为GRADE推荐强度指南，1=强推荐，2=弱推荐；A/B/C=高/中/低证据质量；GPS=良好实践声明）

---

## 核心概念

新生儿感染性肺炎是新生儿期常见病、多发病，病死率与胎龄/出生体重成反比。
早发病例病死率高于晚发病例。临床表现不典型，难与其他肺部疾病鉴别。

---

## 知识片段

### 1. 分类与传播途径（P01-P02）

**按发病时间（GPS）：**
- 早发型：生后48h内（或1周内）发生
- 晚发型：生后1周后发生

**按感染途径（1B）：**
- 宫内感染性肺炎：宫内吸入被感染羊水，或病原体经胎盘传播
- 出生时感染性肺炎：出生时吸入感染羊水或母亲阴道定植微生物
- 出生后感染性肺炎：接触传播、血行传播（脐炎/皮肤感染/败血症）、医源性感染（器械/手卫生/血制品）

---

### 2. 危险因素（P03-P04）

**早发型危险因素（1B）：**
- 早产、低出生体重
- 母亲绒毛膜羊膜炎（围产期常见危险因素）
- 胎膜早破：早产儿>18h，足月儿产程发动前破膜>24h
- 母亲GBS（B族链球菌）定植，或既往分娩GBS感染新生儿
- 母亲产时发热（≥38.0°C）

**晚发型危险因素（1B）：**
- 早产、低出生体重
- 长时间机械通气（呼吸机相关性肺炎风险）
- 使用胃酸阻滞剂
- 气道畸形
- 重度基础心肺疾病
- 住院时间较长
- 神经功能受损
- 接触有呼吸道感染的养护人员或家庭成员

---

### 3. 临床表现（P05）——推荐强度1B

**非特异性感染征象：**
- 体温不稳定（过高/过低）
- 呼吸暂停
- 嗜睡
- 心动过速或心动过缓
- 灌注不良

**早发型突出表现：**
- 出生时或出生后不久发生的呼吸窘迫（呻吟、气促、三凹征）
- 严重者进展为脓毒性休克、急性呼吸窘迫综合征、肺出血、肺动脉高压

**晚发型常见表现：**
- 呼吸急促、喂养困难
- 黄疸、呕吐
- 进乳呛咳
- 新发脓痰或痰性质改变，呼吸道分泌物增加及咳嗽
- 口吐泡沫（社区获得性感染性肺炎明显）
- 肺部听诊：呼吸音粗糙/减低、细湿啰音、哮鸣音

**依赖呼吸机者：**
- 原有疾病加重或迁延
- 对氧及呼吸机需求增加
- 脓性气道分泌物出现

---

### 4. 病原体（P06-P08）

**细菌性肺炎病原（P06，1B）：**

| 肺炎类型 | 主要病原体 |
|----------|------------|
| 早发型 | GBS（足月儿多见）、大肠埃希菌（早产儿多见） |
| 晚发型院内 | 葡萄球菌、大肠埃希菌，需警惕机会致病菌及耐药菌（产ESBL菌）|
| 晚发型社区 | 病毒为主（RSV、腺病毒等）|

**病毒性肺炎病原（P07，1B）：**
- 单纯疱疹病毒（HSV）：进展性间质/出血性肺炎，可致呼吸衰竭
- 巨细胞病毒（CMV）：宫内或产后（母乳/输血）传播
- RSV、流感病毒、副流感病毒、腺病毒：社区获得性常见
- SARS-CoV-2
- 肠道病毒、副肠孤病毒

**不典型病原体（P08，1A——高证据质量）：**
- 沙眼衣原体：生后**2周**后发病
- 脲原体：与支气管肺发育不良（BPD）及脑膜炎相关
- 肺炎支原体
- 百日咳鲍特菌

---

### 5. 辅助检查（P09-P10）

**病原学检查（P09，1A）：**
- 下呼吸道/肺泡灌洗液培养（金标准；建议留取下呼吸道样本）
- 鼻咽样本/血液：抗原/抗体检测、NAAT（核酸扩增试验，灵敏度高）
- 痰涂片镜检：与培养结果一致具有重要参考价值
- 胸腔积液（需治疗取样时）：革兰氏染色和培养
- **宏基因组二代测序（mNGS）**：重症患儿常规检查阴性且治疗3d无效时，有条件医院可考虑（首选肺泡灌洗液）

**影像学检查（P10，2C）：**

*胸部X线（主要诊断方法）：*
- 细菌性：两肺弥漫性模糊影或点片状浸润影
- 病毒性：间质病变或肺气肿为主
- 伴胸腔积液支持肺炎诊断
- 影像学改变未在48h内消失

*肺部超声：*
- 良好灵敏度和特异度
- 特征性表现：肺实变伴支气管充气征或支气管充液征
- 实变区胸膜线异常、A-线消失；非实变区可见B-线或肺间质综合征改变
- 对胸腔积液诊断准确性高

*胸部CT（不常规）：*
- 仅在临床症状/体征与常规影像学结果不一致时，或怀疑气道/肺部/血管畸形、肿瘤时考虑

---

### 6. 诊断标准（P11，GPS）

综合诊断（无统一金标准）需同时满足：
1. **高危因素**存在
2. **临床表现**：发热/呼吸急促/呼吸窘迫/呼吸暂停/肺部啰音/喂养困难
3. **影像学证据**：胸片示浸润影（>48h未消失）；超声示肺实变+支气管充气征
4. **实验室支持**：WBC/CRP/PCT异常，病原学阳性

宫内感染性肺炎：生后8~12h内气管抽吸物培养有助于诊断。

---

### 7. 鉴别诊断（P12，1A）

需鉴别所有引起新生儿呼吸窘迫的疾病：

| 疾病 | 主要特征 |
|------|----------|
| 新生儿暂时性呼吸增快（TTN）| 晚期早产儿/足月剖宫产儿；12~24h后恢复（严重者持续72h）|
| 新生儿呼吸窘迫综合征（NRDS）| 早产儿；PS治疗有效；高危因素+病原学阴性有助鉴别 |
| 胎粪吸入综合征（MAS）| 有胎粪污染羊水史；特征性胸片 |
| 先天性心脏病 | 心脏杂音、休克、发绀、超声心动图确认 |

---

### 8. 抗感染治疗（P13-P15）

**细菌性肺炎（P13，GPS）：**

培养结果前根据发病时间和流行病史经验性启动，明确病原后按药敏调整。

*早发型经验性方案：*
- 氨苄西林（或青霉素）+ 第三代头孢菌素

*晚发型院内感染经验性方案：*
- 万古霉素（或替考拉宁）+ 第三代头孢菌素
- 参照本地近期病原流行趋势和药敏优先窄谱

*培养阴性者：*
- 经验性IV抗生素至少48~72h，持续临床评估
- 高度提示肺炎：治疗**7~10天**（研究显示5天安全有效）
- 48h内临床症状缓解：4天与7天疗效相当

**常见病原菌用药（P13，表2）：**

| 病原体 | 首选药物 |
|--------|----------|
| B族链球菌（GBS）| 氨苄西林或青霉素 |
| 大肠埃希菌（氨苄西林敏感）| 氨苄西林 |
| 大肠埃希菌（氨苄西林耐药）| 头孢噻肟/头孢他啶/头孢吡肟 |
| 多重耐药革兰氏阴性菌（产ESBL）| 美罗培南 |
| MSSA（甲氧西林敏感金黄色葡萄球菌）| 萘夫西林或头孢唑林 |
| MRSA（耐甲氧西林）| 万古霉素/克林霉素/利奈唑胺 |
| 单核细胞增多性李斯特菌 | 氨苄西林或青霉素 |
| 沙眼衣原体/肺炎支原体 | 红霉素或阿奇霉素（红霉素耐药时） |
| 梅毒螺旋体 | 青霉素 |
| 单纯疱疹病毒（HSV）| 阿昔洛韦 |
| 巨细胞病毒（CMV）| 更昔洛韦或缬更昔洛韦 |
| 流感病毒 | 奥司他韦 |

**病毒性肺炎（P14，1B）：**
- 一般无特异性治疗，以对症支持治疗为主
- CMV肺炎：更昔洛韦或缬更昔洛韦
- 流感：奥司他韦

**不典型病原体肺炎（P15，1A——高证据质量）：**
- 脲原体/衣原体肺炎：**首选红霉素**，红霉素耐药者用阿奇霉素

### 9. 呼吸支持（P16-P17）

**适应证（P16，2B）：**
出现低氧血症及呼吸衰竭时给予吸氧或呼吸支持。

**基层可用氧疗设备：**
- 鼻导管、头罩、面罩及暖箱内供氧

**经鼻高流量氧疗（HFNC）（有条件机构）：**
- 初始FiO₂：**30%~40%**
- 初始流量：**3~8 L/min**
- 根据临床反应调整

**nCPAP（经鼻持续气道正压通气，首选无创）：**
- 如HFNC无效，升级为nCPAP等无创辅助通气

**有创呼吸支持指征：**
- nCPAP压力>**6 cmH₂O** + FiO₂>**30%** 仍SpO₂<**93%** → 考虑气管插管机械通气

**氧合监测（P17，GPS）：**
- 推荐脉搏血氧仪监测（防止低氧和高氧损伤均重要）
- 辅以动脉血气分析

---

### 10. 转诊指征（P18，GPS）

**重症肺炎指征（满足任意1项即需评估转诊）：**
- 一般情况差
- 拒食或脱水征
- 意识障碍
- 呼吸频率明显增快（**>70次/min**）
- 发绀
- 呼吸困难（鼻翼扇动、三凹征加重）
- SpO₂≤**92%**（海平面）或<**90%**（高原）
- 多肺叶或单肺叶≥**2/3**受累
- 胸腔积液
- 肺外并发症（脓毒性休克/急性呼吸窘迫综合征/DIC）
- 生命体征不稳定需要高级生命支持

**其他新生儿危重病评分参考（需转诊）：**
- HR<80次/min或>180次/min
- 收缩压<40 mmHg或>100 mmHg
- 呼吸<20次/min或>100次/min
- 需气管插管机械通气或反复呼吸暂停
- 严重心律失常
- 弥散性血管内凝血（DIC）

---

### 11. 红旗信号

- 呼吸频率持续>70次/min（安静状态）
- 发绀（口唇/甲床青紫）
- SpO₂持续<92%（海平面）
- 呼吸暂停发作
- 新发脓性分泌物（依赖呼吸机者）
- 一般情况急剧恶化：极度嗜睡/拒乳/肌张力下降
