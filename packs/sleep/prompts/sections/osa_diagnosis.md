# OSA 诊断领域知识片段

## 核心知识（来自指南推荐意见第 1-3 条，页码 718-726）

### OSA 诊断标准（ICSD-3-TR）
满足以下 A+B 或 C+D 条件，且 AHI≥5 次/h，即可诊断 OSA：
- **A**：存在以下任一症状：嗜睡、疲劳、失眠或因憋气/窒息/喘气觉醒、习惯性打鼾、经他人观察到的呼吸暂停、生活质量（QOL）受损
- **B**：PSG 或睡眠中心外睡眠监测（OCST）证实 AHI≥5 次/h
- **C**：PSG 或 OCST 证实 AHI≥15 次/h
- **D**：无法用其他疾病来解释

### OSA 严重程度分级
| 分级 | AHI 范围 | 典型症状 |
|------|---------|---------|
| 轻度 | 5～15 次/h | 轻度日间嗜睡，打鼾 |
| 中度 | 15～30 次/h | 明显日间嗜睡，习惯性打鼾 |
| 重度 | ≥30 次/h | 严重日间嗜睡（ESS≥16），频繁憋醒 |

辅助指标：
- ODI（氧减指数）：简便，对中重度 OSA 准确性较高
- SASHB（睡眠呼吸暂停特异性缺氧负荷）：预测心血管风险的重要指标
- REM-AHI：OSA 在 REM 睡眠期可能加重（REM-OSA），与 CVD 风险密切相关

### 筛查工具
- **STOP-Bang 问卷（SBQ）**：最推荐筛查工具（证据等级 B，弱推荐）
  - SBQ≥3 分即为筛查阳性
  - 对轻/中/重度 OSA 的灵敏度：92%/95%/96%，特异度：35%/27%/28%
  - 涵盖：打鼾（S）、疲劳（T）、观察到的呼吸暂停（O）、高血压（P）、BMI（B）、年龄（A）、颈围（N）、性别（G）
- **ESS**（Epworth 嗜睡量表）：弱推荐，单独使用灵敏度低（49%）
- **NoSAS 评分**：可识别潜在 OSA（灵敏度 79.80%，特异度 58.20%）
- **III 型便携式监测（PM）**：推荐用于房颤（AF）患者筛查（B，强推荐）；可用于慢性心力衰竭（CHF）筛查（C，弱推荐）

### 临床症状与体征（推荐关注要点）
**症状**：
- 打鼾（70%～95% 患者有）；打鼾时间间隔指数（STII）与 AHI 相关性最强（r=0.89）
- 睡眠憋气、呼吸暂停（夜间窒息感）
- 日间嗜睡（15%～50% 患者有）；注意：部分 OSA 患者无日间嗜睡，不能以此排除 OSA
- 其他：夜尿、遗尿、夜间频繁觉醒、出汗、阳痿等（CPAP 治疗后可改善）

**体征**：
- 颈围粗大（颈围与 OSA 严重程度相关）
- BMI 偏高（OSA 患者 BMI 显著高于非 OSA 患者）
- 腰围/腰臀比（WHR）增大：腹部脂肪堆积减少肺容积，增加咽壁侧向张力
- 颌面畸形（如下颌后缩）、咽腔狭窄、扁桃体肥大

## Few-shot 示例 1

**问**：医生说要我去做多导睡眠监测，这个检查是什么？非做不可吗？

**【适用人群】**
适用于疑似 OSA 的成人患者，尤其是有打鼾、夜间憋醒、日间嗜睡等症状者。

**【方案要点】**
- 多导睡眠监测（PSG）是诊断 OSA 的金标准，在整晚睡眠中同步记录脑电、眼电、心电、口鼻气流、血氧、体位等多项生理信号，以计算 AHI（每小时呼吸暂停次数）
- 没有 PSG 或等效的睡眠中心外监测（OCST/III 型便携式监测），无法确诊 OSA 或判断严重程度
- 对于房颤或慢性心力衰竭患者，III 型便携式监测可作为筛查替代，但诊断仍需医生综合判读
- 检查结果决定后续治疗方案（CPAP 压力、是否手术等），不做检查就无法安全制定治疗方案

**【频率 / 时长 / 强度】**
- PSG：通常在医院睡眠中心进行一晚（7～8 小时）的整夜监测；检查前一天避免午睡、饮酒、咖啡因
- 便携式监测（PM）：可居家完成，将监测设备随身携带睡一晚，次日归还仪器
- 无需重复检查，一次完整的 PSG 结果通常够诊断使用

**【注意事项与红旗信号】**
注意事项：
- 做 PSG 前一天正常作息，不要刻意改变睡眠时间或使用助眠药物（除非医生要求）
- 便携式监测无法分辨非阻塞性睡眠呼吸暂停（CSA），若高度怀疑或结果不明确，需到医院做 PSG
- AHI 数值只是诊断依据之一，症状、血氧变化等综合指标才决定治疗方案

红旗信号：
- 监测当晚出现严重憋醒后呼吸无法恢复、口唇发绀 → 立即就医或拨打 120
- 日间嗜睡严重到驾车或操作机器时无法控制入睡 → 立即就医

**【依据】**
推荐意见第 3 条《成人阻塞性睡眠呼吸暂停诊断和外科治疗指南（2024）》/ 证据质量：B / 推荐强度：强推荐 / 来源页码：724

## Few-shot 示例 2

**问**：AHI 是什么意思？我老公 AHI 28，严重吗？

**【适用人群】**
适用于已完成 PSG 或睡眠监测、需要理解检查结果含义的患者及家属。

**【方案要点】**
- AHI（呼吸暂停低通气指数）指每小时睡眠中发生呼吸暂停和低通气事件的总次数，是诊断和分级 OSA 的主要指标
- AHI 28 次/h 属于**中度 OSA**（指南标准：15～30 次/h 为中度）
- 中度 OSA 意味着每小时大约有 28 次呼吸暂停或低通气事件，长期持续会增加高血压、心血管疾病和认知障碍等风险
- 中度 OSA 的首选治疗是 CPAP（持续气道正压通气）；如存在解剖异常（如扁桃体肥大、下颌后缩），也可结合医生评估考虑手术

**【频率 / 时长 / 强度】**
- 这里无需具体频率参数，关键是理解分级含义
- AHI 严重程度分级：5-15 次/h（轻度）/ 15-30 次/h（中度）/ ≥30 次/h（重度）
- 辅助参考：LSaO2（最低血氧饱和度），治疗有效标准是 AHI<20 次/h 且降低幅度≥50%

**【注意事项与红旗信号】**
注意事项：
- AHI 28 是中度，需要积极治疗，不能因为"不算最严重"而忽视
- AHI 数值不能单独判断，还需结合日间嗜睡程度、血氧变化、合并症等综合评估；具体治疗方案需由医生决定
- 不同医院、不同监测设备的 AHI 计算方法可能略有差异，建议以主诊医生判读为准

红旗信号：
- 中度 OSA 伴夜间胸闷、心悸、心律不齐 → 立即就医排查心血管问题
- 日间嗜睡导致驾车或操作机器时无法控制入睡 → 立即就医，治疗前暂停驾车

**【依据】**
