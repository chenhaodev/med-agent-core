# 越界 / 危机拒答模板（患者 / 家属模式）
# 供 ask.sh 在 oob_check 触发后直接输出，不进入 API 管道
# 格式：每个模板用 ## TYPE 分隔

## crisis

我很担心你现在的安全。你现在的感受很重要，也并不孤单——请立刻寻求专业、即时的帮助：

🆘 立即求助方式：
- **全国统一心理援助热线：拨打 12356**（24 小时免费，专业人员会倾听并帮助你）
- 如果你或他人有立即的危险（已采取行动、手边有危险物品），请**立即拨打 120 急救**或前往最近医院的**精神科急诊 / 急诊科**
- 联系你信任的家人或朋友，告诉他们你现在的状态，不要独自一人

请记住：强烈的痛苦感受是可以缓解的，专业帮助真的有用。本系统只能提供疾病科普，无法替代此刻你需要的即时人工支持——请优先联系上面的热线或急诊。

## medical_emergency

⚠️ 你描述的情况（服用精神类药物后出现高热、肌肉僵硬、意识改变 / 大量出汗等）**可能是危及生命的严重药物不良反应**（如神经阻滞剂恶性综合征），这是医疗急症，需要立即处理：

🚑 立即行动：
- **马上拨打 120 或立即前往最近医院急诊科**，告诉医生正在服用的精神类药物
- 在专业指导前不要让患者继续服用可疑药物，但**也不要自行处理或拖延**——尽快就医是关键
- 携带正在服用的所有药物 / 药盒给急诊医生看

这种情况不能等待，请优先就医。本系统只能提示风险，无法替代急诊救治。

## diagnosis

很抱歉，我无法通过文字判断你是否患有某种精神障碍。是否符合诊断需要精神科医生通过面诊、结构化访谈和综合评估才能确定，本系统依据 DSM-5 只能提供疾病科普，不能替代就诊。

建议你：
- 到精神科 / 心理科门诊，请专业医生进行评估
- 如果你想先了解某种精神障碍"是什么、有哪些表现、何时该求助"，欢迎继续提问

## prescription

很抱歉，我不能推荐任何精神类药物或给出用药方案。精神科药物的选择需要医生根据诊断、病史、躯体状况和既往用药反应综合决定，用错或自行用药可能有风险。

建议你：
- 到精神科门诊，请医生评估后决定是否用药及用哪种药
- 如果你想了解某种精神障碍本身的科普知识，欢迎继续提问

## dosing_change

精神类药物的调整（加量、减量、换药、停药）必须由你的精神科医生根据病情变化来决定。**自行停药或改量可能引起症状反复或戒断反应，风险较大。**

建议你：
- 联系你的主治医生或复诊，请医生根据你的情况调整方案
- 在医生指导前，请按原方案服药，不要自行改动

## unrelated

我是依据 DSM-5 专门为公众和家属提供常见精神障碍科普的顾问，你的问题超出了我能帮助的范围。

如果你有精神健康相关的问题，欢迎随时提问，例如：
- "抑郁症一般有哪些表现？"
- "惊恐发作是怎么回事？"
- "家人得了双相障碍，我该了解些什么？"
