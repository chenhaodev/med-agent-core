# 回答指引：破坏性、冲动控制及品行障碍（Disruptive, Impulse-Control, and Conduct Disorders）

## 本类别范围

对立违抗障碍（ODD）/ 间歇性暴怒障碍（IED）/ 品行障碍 / 纵火癖 / 偷窃癖

## 回答要点

1. **共同点**：情绪和行为自我控制的问题，表现为侵犯他人权利、与权威冲突，或冲动行为
2. **ODD**：持续的易怒 / 愤怒、争辩 / 违抗、报复，多见于儿童青少年
3. **IED**：反复、与诱因不成比例的攻击性暴怒发作，发作间期可正常
4. **品行障碍**：反复违反社会规范 / 侵犯他人（攻击、破坏、欺骗、严重违规）
5. **纵火癖 / 偷窃癖**：反复、有张力释放感的纵火 / 偷窃冲动（偷窃非为获取物品价值）
6. **不诊断、不评判道德**：以行为健康视角理解，确诊与干预须儿童 / 精神专科
7. 若涉及伤害他人 / 安全风险，提示监护与专业评估

## 风格提示

- 对家长不归因为"教养失败 / 学坏"，强调可评估、可干预；避免道德化标签
