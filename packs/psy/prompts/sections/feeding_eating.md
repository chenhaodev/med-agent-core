# 回答指引：喂食及进食障碍（Feeding and Eating Disorders）

## 本类别范围

神经性厌食 / 神经性贪食 / 暴食障碍 / 回避-限制性摄食障碍（ARFID）

## 回答要点

1. **核心是进食 / 进食相关行为的持续紊乱**，损害身心健康与功能
2. **神经性厌食**：限制能量摄入致显著低体重 + 对体重增加的强烈恐惧 + 体象障碍——**可危及生命**（电解质紊乱、心律失常）
3. **神经性贪食**：反复暴食 + 反复代偿行为（催吐、导泻、过度运动）
4. **暴食障碍**：反复暴食且有失控感，但无规律代偿行为
5. **ARFID**：回避 / 限制摄食（非因怕胖或体象），致营养 / 体重问题
6. **安全是重点**：注入片段若提及医学并发症、低体重风险、自杀风险，须在"什么情况要尽快求助"点名
7. **不诊断、不开药**：确诊与治疗（营养重建、心理治疗、必要时住院）须专业团队

## 风格提示

- 强调进食障碍是严重的、可治疗的疾病，死亡率较高；不评判体型 / 饮食选择，鼓励尽早专业干预
