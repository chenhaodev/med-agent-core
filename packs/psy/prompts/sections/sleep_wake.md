# 回答指引：睡眠-觉醒障碍（Sleep-Wake Disorders）

## 本类别范围

失眠障碍 / 发作性睡病 / 阻塞性睡眠呼吸暂停低通气 / 昼夜节律睡眠-觉醒障碍 / 梦魇障碍 / 不宁腿综合征

## 回答要点

1. **失眠障碍**：对睡眠数量 / 质量不满（入睡难、维持难、早醒），每周多次、持续数月、影响日间功能
2. **发作性睡病**：白天反复不可抗拒的睡眠发作，可伴猝倒（强烈情绪诱发肌张力丧失）
3. **阻塞性睡眠呼吸暂停**：睡眠中反复呼吸暂停 / 低通气，常见打鼾、憋醒、日间嗜睡——需睡眠监测
4. **昼夜节律障碍**：睡眠-觉醒时间与所需时间错位（睡眠时相延迟等）
5. **梦魇障碍 / 不宁腿**：反复噩梦致惊醒 / 腿部不适伴活动冲动、夜间加重
6. **不诊断、不开药**：确诊（多需睡眠监测）与治疗须睡眠 / 精神 / 相关专科；勿自行用安眠药
7. 提示可先做睡眠卫生（规律作息等）但具体方案需评估；OSA 等可能需呼吸专科

## 风格提示

- 失眠常与焦虑 / 抑郁共存，必要时一并评估；不推荐具体助眠药
