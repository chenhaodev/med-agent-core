# 回答指引：人格障碍（Personality Disorders）

## 本类别范围

A 组（偏执 / 分裂样 / 分裂型）、B 组（反社会 / 边缘 / 表演 / 自恋）、C 组（回避 / 依赖 / 强迫型）

## 回答要点

1. **核心定义**：持久、广泛、僵化的内在体验与行为模式，明显偏离文化预期，起于青少年 / 成年早期，引起痛苦或功能损害
2. **A 组**（古怪 / 孤僻）：偏执（普遍不信任猜疑）、分裂样（疏离、情感受限）、分裂型（古怪信念 / 知觉、社交不适）
3. **B 组**（戏剧 / 情绪 / 不稳）：反社会（无视 / 侵犯他人权利）、边缘（人际、自我形象、情绪不稳与冲动）、表演（过度情绪化与寻求关注）、自恋（夸大、需要赞美、缺乏共情）
4. **C 组**（焦虑 / 恐惧）：回避（社交抑制、自卑、对负评敏感）、依赖（过度需要被照顾）、强迫型（追求秩序、完美、控制）
5. **边缘型自杀 / 自伤风险**：注入片段若提及，须在求助段点名并给热线
6. **不诊断、不贴标签、不评判**：人格障碍诊断须精神科专业评估；避免将"性格不好"随意等同于人格障碍

## 风格提示

- 去污名化（尤其边缘型、反社会、自恋常被滥用为骂人词）；强调可评估、心理治疗可帮助
