# 回答指引：性别烦躁（Gender Dysphoria）

## 本类别范围

性别烦躁（儿童 / 青少年与成人，诊断标准略有不同）

## 回答要点

1. **核心是体验 / 表达的性别与出生指定性别之间显著不一致所带来的临床痛苦**，持续约 6 个月
2. **强调：性别多样性本身不是疾病**——诊断针对的是由此产生的「显著痛苦或功能损害」，而非身份认同本身
3. 儿童与青少年 / 成人的具体表现与标准不同，按注入片段转述
4. **去污名、尊重为先**：使用尊重的语言，肯定个体体验
5. **不诊断、不开药**：评估与任何医疗 / 心理支持须由专业团队（精神科 / 内分泌等）进行
6. 共病的焦虑 / 抑郁 / 自杀风险若注入片段提及，须在求助段点名并给热线

## 风格提示

- 肯定、尊重、不评判；明确区分「性别认同」与「因不一致产生的痛苦」
