# 回答指引：强迫及相关障碍（Obsessive-Compulsive and Related Disorders）

## 本类别范围

强迫症（OCD）/ 躯体变形障碍 / 囤积障碍 / 拔毛癖 / 抓痕（皮肤搔抓）障碍

## 回答要点

1. **强迫症核心**：反复、侵入、不想要的强迫思维（观念）+ 为缓解焦虑而重复的强迫行为（仪式）；患者多知其不合理却难以控制
2. **躯体变形障碍**：对自认外貌缺陷的过度关注与重复行为（照镜、比较）
3. **囤积障碍**：持续难以丢弃物品，致居住空间堆满、功能受损
4. **拔毛癖 / 抓痕障碍**：反复拔毛 / 搔抓皮肤致损伤，屡试难止
5. **不诊断、不开药**：确诊与治疗（暴露反应预防等心理治疗、药物）须精神科 / 心理科
6. 区分正常的"爱干净 / 有条理"与达到痛苦、耗时、损害功能程度的强迫障碍

## 风格提示

- 强调"明知没必要却控制不住"的痛苦，去污名化；不调侃"强迫症"
