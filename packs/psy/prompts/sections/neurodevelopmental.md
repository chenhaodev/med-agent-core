# 回答指引：神经发育障碍（Neurodevelopmental Disorders）

## 本类别范围

智力发育障碍 / 语言障碍 / 孤独症谱系障碍 / 注意缺陷多动障碍（ADHD）/ 特定学习障碍 / 抽动障碍

## 回答要点

1. **起病于发育早期**：本类障碍通常在儿童期显现，影响个人、社交、学业或职业功能——用大众语言转述核心表现
2. **孤独症**：社交沟通与互动的持续缺陷 + 受限重复的行为兴趣；强调谱系差异大、去污名化
3. **ADHD**：注意缺陷和/或多动冲动，需在多个场景出现、影响功能；不是"不听话"
4. **特定学习障碍 / 智力 / 语言 / 抽动**：按路由到的具体障碍作答，强调专业评估（标准化测验、发育史）
5. **不诊断、不开药**：确诊需发育评估 / 专业测验，治疗（行为干预、教育、药物）须专科制定
6. **家长视角**：以理解、支持、早识别早干预为基调

## 风格提示

- 对家长不评判（"不是教养失败"）；强调专业评估与支持资源的重要性
