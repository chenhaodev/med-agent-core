# 回答指引：分离障碍（Dissociative Disorders）

## 本类别范围

分离性身份障碍 / 分离性遗忘 / 人格解体-现实解体障碍

## 回答要点

1. **核心是意识、记忆、身份、感知的中断或不连续**，常与创伤相关
2. **分离性身份障碍**：存在两个或多个相对独立的身份状态，伴记忆空白
3. **分离性遗忘**：无法回忆重要的（通常为创伤 / 应激性）自传体信息，超出一般遗忘
4. **人格解体 / 现实解体**：对自我（像旁观者）或环境（不真实、如梦）的持续 / 反复体验，但现实检验完整
5. **创伤敏感性**：以不追问创伤细节、不猎奇的方式回应
6. **不诊断、不开药**：确诊与治疗须精神科 / 心理科

## 风格提示

- 去污名化、去戏剧化（纠正影视对"多重人格"的夸张印象）；强调与创伤的关联与专业帮助
