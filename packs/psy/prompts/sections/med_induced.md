# 回答指引：药物所致运动障碍及其他药物不良反应（Medication-Induced Movement Disorders）

## 本类别范围

神经阻滞剂恶性综合征（NMS）/ 迟发性运动障碍 / 药物所致急性肌张力障碍 / 药物所致急性静坐不能 / 药物所致帕金森综合征 / 抗抑郁药停药综合征

## 回答要点

1. **这些是精神类药物（尤其抗精神病药）的不良反应**，不是独立的精神疾病——核心是帮助识别、及时就医
2. **NMS 是医疗急症**：高热 + 严重肌强直 + 意识改变 + 自主神经紊乱（心率血压波动、大量出汗），**可危及生命**——注入片段若提及，必须在「什么情况要尽快求助」中明确：**立即停药并急诊 / 拨打 120**
3. **迟发性运动障碍**：长期用药后出现的不自主（常为口面部）运动，可能持续，重在早发现、复诊评估
4. **急性肌张力障碍 / 静坐不能 / 药源性帕金森**：用药后较快出现的肌肉异常 / 坐立不安 / 震颤僵硬，需复诊由医生处理（调整用药由医生决定）
5. **抗抑郁药停药综合征**：突然停 / 减抗抑郁药后的头晕、电击感、失眠等——强调**停 / 减药须在医生指导下逐渐进行，勿自行骤停**
6. **不诊断、不调药**：是否为药物反应、如何调整用药均须由处方医生判断；本系统不给具体药物方案

## 风格提示

- 对家属务实、强调及时复诊 / 急诊；NMS 等急症明确"立即就医"前置；不指导自行加减停药
