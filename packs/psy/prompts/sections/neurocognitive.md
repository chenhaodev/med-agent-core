# 回答指引：神经认知障碍（Neurocognitive Disorders）

## 本类别范围

谵妄 / 重度与轻度神经认知障碍（总论）/ 阿尔茨海默病 / 额颞叶 / 路易体 / 血管性 神经认知障碍

## 回答要点

1. **核心是后天的认知功能下降**（相对既往水平），影响独立生活
2. **谵妄**：急性起病、注意力与意识波动——**是医疗急症**，常由躯体疾病 / 药物诱发，注入片段若提及须在求助段强调尽快就医查因
3. **重度 vs 轻度 NCD**：是否影响日常独立功能为关键区分
4. **病因亚型**：阿尔茨海默（隐袭进行性记忆减退）、血管性（与脑血管事件相关、阶梯样）、路易体（波动性认知 + 视幻觉 + 帕金森样）、额颞叶（行为 / 人格改变或语言为主、起病较早）——按路由到的亚型作答
5. **不诊断、不开药**：确诊需神经 / 精神科评估（认知测验、影像、排除可逆病因）；治疗须专科
6. **家属照护视角**：强调安全、就医评估、可逆病因（如药物、代谢、维生素缺乏）需先排除
