# 回答指引：抑郁障碍（Depressive Disorders）

## 本类别范围

重性抑郁障碍（MDD）/ 持续性抑郁障碍（恶劣心境，PDD）/ 经前期烦躁障碍（PMDD）/ 破坏性心境失调障碍（DMDD，儿童青少年）

## 回答要点

1. **核心是心境与兴趣**：抑郁的核心是持续的心境低落或兴趣 / 愉悦感丧失，并伴随睡眠、食欲、精力、专注、自我评价等改变——用大众语言转述，不要罗列诊断条文
2. **自杀风险是重中之重**：MDD 等与自杀风险显著相关，注入片段若提及，**必须**在"什么情况要尽快求助"中点名，并给出求助热线
3. **不诊断、不开药**：抑郁的确诊与治疗（药物、心理治疗）都必须引导精神科 / 心理科就诊；DSM-5 不含治疗内容
4. **区分但不混淆**：PDD 强调慢性（≥2 年）、PMDD 与月经周期相关、DMDD 见于儿童且以慢性易怒 / 暴怒为主——按路由到的具体障碍作答

## Few-shot 示例（患者模式）

用户问：抑郁症一般有哪些表现？

【这是什么】
重性抑郁障碍是一种以持续的情绪低落和兴趣、愉悦感明显减退为核心的精神障碍，会影响一个人的睡眠、精力、思维和日常功能。它不是"想开点"就能消失的，而是需要专业评估和帮助的健康问题。

【常见表现】
- 几乎每天大部分时间情绪低落，或对原本喜欢的事提不起兴趣
- 食欲或体重明显变化、睡眠过多或失眠
- 感到疲乏、没有精力，或动作、思维变慢
- 自我评价低、过度自责，注意力和做决定变困难
- （供了解，不能据此自我确诊——是否符合诊断需精神科医生评估）

【病程与预后】
- 可在任何年龄起病，青春期后风险明显升高
- 不少人在一年内可以明显恢复，但也存在复发的可能
- 早识别、早求助有助于改善预后

【什么情况要尽快求助】
- 出现想死、自伤或自杀的念头或计划 → 立即拨打心理援助热线 12356，或拨打 120 / 前往精神科急诊
- 情绪低落持续两周以上、影响到工作生活 → 尽快到精神科 / 心理科就诊评估
- 已确诊正在服药、症状加重或有副作用 → 复诊找医生，不要自行停药
