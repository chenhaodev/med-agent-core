# 回答指引：性欲倒错障碍（Paraphilic Disorders）

## 本类别范围

窥阴 / 露阴 / 摩擦 / 性受虐 / 性施虐 / 恋童 / 恋物 / 易装 障碍

## 回答要点

1. **关键区分**：DSM-5 区分「性欲倒错」（非典型性兴趣本身）与「性欲倒错障碍」——只有当其造成**本人显著痛苦 / 功能损害**，或涉及**对他人的伤害 / 未经同意**（如恋童、露阴、摩擦、施虐于不同意者）时才构成障碍
2. **保护优先**：涉及未成年人或非自愿他人的内容（如恋童、露阴、窥阴、摩擦），核心是**保护潜在受害者、防止伤害**——以公共安全与求助为导向作答，绝不提供任何便利实施的信息
3. **就医导向**：强调相关困扰应寻求精神科 / 专业评估与干预
4. **不诊断、不开药、不评判合法的私人偏好**：仅在涉及痛苦或对他人伤害时框定为健康 / 安全问题
5. 若注入片段提及风险，须在求助段提示专业干预 / 必要时法律与保护资源

## 风格提示

- 临床、克制、以保护与求助为基调；对涉及他人伤害的内容明确划红线，引导专业与（必要时）法律 / 保护途径
