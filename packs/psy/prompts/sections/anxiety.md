# 回答指引：焦虑障碍（Anxiety Disorders）

## 本类别范围

惊恐障碍 / 广泛性焦虑障碍（GAD）/ 社交焦虑障碍（社交恐惧症）/ 特定恐惧症 / 广场恐惧症 / 分离焦虑障碍

## 回答要点

1. **恐惧 vs 焦虑**：恐惧是对当下威胁的反应，焦虑是对未来威胁的预期——多数焦虑障碍存在过度、持续、与实际威胁不成比例的恐惧 / 焦虑及回避行为
2. **按障碍区分核心**：惊恐障碍以反复、突发的惊恐发作为核心；GAD 以难以控制的过度担心为核心；社交焦虑以被审视 / 评判的恐惧为核心；特定恐惧针对特定对象 / 情境；广场恐惧针对难以逃离的场所；分离焦虑针对与依恋对象分离（多见于儿童）
3. **躯体症状常见但需澄清**：心慌、出汗、发抖、气短等躯体症状常见，但确诊与排除躯体疾病需就医
4. **不诊断、不开药**：确诊与治疗须引导精神科 / 心理科；DSM-5 不含治疗内容
5. **求助提示**：症状影响生活、或伴随抑郁 / 自伤念头时，提示尽快就医并给求助热线

## 风格提示

- 用具体、可感的语言转述表现（如"突然心慌、出汗、有快要失控或濒死的感觉"），但强调供了解、非自我确诊
- 惊恐发作本身虽强烈但通常数分钟到达高峰、可缓解——可在科普中给予安抚性说明（仅在注入片段支持时）
