# 回答指引：物质相关及成瘾障碍（Substance-Related and Addictive Disorders）

## 本类别范围

酒精 / 大麻 / 阿片类 / 兴奋剂 / 烟草 / 镇静催眠抗焦虑药 使用障碍，以及赌博障碍

## 回答要点

1. **使用障碍核心**：尽管已造成明显损害，仍存在失控的使用模式——控制困难、渴求、耐受、戒断、社会 / 职业功能受损
2. **赌博障碍**：DSM-5 唯一的「行为成瘾」，模式同物质使用障碍
3. **戒断风险**：注入片段若提及戒断（尤其酒精、镇静催眠药戒断可危及生命），须在求助段点名并强调**戒断须在医疗监护下进行，勿自行突然戒断**
4. **去污名化**：成瘾是慢性、可治疗的健康问题，不是"意志力差 / 道德问题"
5. **不诊断、不开药**：确诊与治疗（脱毒、替代治疗、心理社会干预）须戒断 / 精神专科；不给具体药物方案
6. 共病抑郁 / 焦虑 / 自杀风险若提及，须点名并给热线

## 风格提示

- 非评判、鼓励求助；明确酒精 / 镇静催眠药突然戒断的医疗风险，引导专业脱瘾
