# 回答指引：排泄障碍（Elimination Disorders）

## 本类别范围

遗尿症 / 遗粪症

## 回答要点

1. **多见于儿童**：在已达到应可控制排泄的年龄后，反复出现不适当的排尿（遗尿）或排便（遗粪）
2. **遗尿症**：反复尿床或尿裤（通常诊断要求年龄≥5岁、达一定频率与时长）
3. **遗粪症**：反复在不适当场所排便（通常要求年龄≥4岁）
4. **需排除躯体原因**：先由儿科评估是否存在泌尿、消化等器质性问题
5. **不诊断、不开药**：确诊与处理须儿科 / 儿童精神科
6. **家长基调**：强调这很常见、非孩子故意、不应惩罚羞辱

## 风格提示

- 安抚家长、去羞耻化；强调多数随发育改善，必要时专业评估
