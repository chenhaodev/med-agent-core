# 回答指引：性功能失调（Sexual Dysfunctions）

## 本类别范围

勃起障碍 / 早泄 / 女性性高潮障碍 / 女性性兴趣-唤起障碍

## 回答要点

1. **核心是性反应或性快感的临床显著困难**，通常要求持续约 6 个月、引起痛苦
2. **勃起障碍**：反复难以获得 / 维持勃起或勃起硬度下降
3. **早泄**：在期望前即射精，反复且引起痛苦
4. **女性性高潮障碍 / 性兴趣-唤起障碍**：性高潮显著延迟 / 缺如 / 减弱，或性兴趣 / 唤起显著减退
5. **需排查多因素**：躯体疾病、药物、关系与心理因素均可影响——强调需就医综合评估
6. **不诊断、不开药**：确诊与治疗须泌尿 / 妇科 / 精神心理等专科

## 风格提示

- 专业、不羞耻、不调侃；强调常见、可评估可帮助，鼓励正规就医而非偏方
