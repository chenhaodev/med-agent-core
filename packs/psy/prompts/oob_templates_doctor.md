# 越界 / 危机拒答模板（医生模式）
# 供 ask.sh 在 oob_check 触发后、--mode doctor 时直接输出
# 格式：每个模板用 ## TYPE 分隔

## crisis

该问题涉及急性自杀 / 自伤 / 危害风险信号，属危机情境，超出本系统（DSM-5 诊断简报）的异步科普范围。

临床处置方向（供专业人员参考）：
- 立即进行自杀风险评估（意念 / 计划 / 手段 / 意图，既往尝试史，保护与风险因素）
- 制定安全计划，必要时启动危机干预、收住院或转精神科急诊
- 来访者自助渠道：全国统一心理援助热线 12356（24 小时）；有立即危险时拨打 120 / 至急诊

如需该障碍的自杀风险因素、风险评估相关诊断要素，欢迎在常规科普范围内继续提问。

## medical_emergency

该描述（精神类药物使用后高热 + 肌强直 + 意识 / 自主神经紊乱）高度提示**神经阻滞剂恶性综合征（NMS）或 5-羟色胺综合征等危及生命的药物不良反应**，属医疗急症，超出异步科普范围。

临床处置方向（供专业人员参考）：
- 立即停用致病药物，启动急诊评估与支持治疗（监测生命体征、体温、肌酸激酶、肾功能）
- 按 NMS / 5-羟色胺综合征规范处理，必要时转 ICU；拨打 120 / 急诊
- 鉴别 NMS、5-羟色胺综合征、恶性高热、抗胆碱能中毒等

如需该类药物不良反应的诊断要点 / 鉴别，欢迎在常规科普范围内继续提问。

## diagnosis

本系统提供 DSM-5 诊断标准与鉴别框架的资料，但不能基于个案描述对特定患者作出确诊判断——诊断须结合结构化临床访谈、病史与多源信息综合形成。

如需了解相关障碍的 DSM-5 诊断标准、鉴别诊断框架或亚型 / 标注，欢迎继续提问。

## prescription

本系统依据 DSM-5（诊断手册，不含治疗内容）提供诊断简报，精神药物的选择与剂量方案超出覆盖范围，须按精神科治疗规范及个体情况制定。

如需了解相关障碍的诊断要点、病程或风险评估要素，欢迎继续提问。

## dosing_change

精神药物的剂量调整须由主管医生结合疗效、不良反应与病情决定，本系统不提供个体化剂量调整建议。

如需了解相关障碍的诊断 / 鉴别 / 风险评估要素，欢迎继续提问。

## unrelated

本系统是依据 DSM-5 的精神障碍诊断简报助手，该问题超出可协助范围。

如有精神障碍诊断 / 鉴别 / 流行病学相关问题，欢迎继续提问。
